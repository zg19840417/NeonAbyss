# 废土元年 — 世界设定与10章剧情总览

> **版本**：v1.0
> **版本日期**：2026-04-08
> **用途**：定义10章主线剧情的整体框架、世界观、章节串联

---

## 一、核心设定总览

### 1.1 游戏背景

| 项目 | 内容 |
|------|------|
| **世界观** | 核战后的废土末世，人类在庇护所中苟延残喘 |
| **核心机制** | 融合技术 - 将人类与变异生物融合创造"融合姬" |
| **玩家身份** | 继承者 - 继承前任管理者遗志，收集星舰零件重建文明 |
| **最终目标** | 收集10个星舰零件，重建星舰，带人类离开地球 |

### 1.2 三系元素

| 元素 | 特点 | 代表色 |
|------|------|-------|
| **水系** | 治愈、净化、守护 | 蓝色 |
| **火系** | 破坏、热情、冲动 | 红色 |
| **风系** | 自由、速度、变化 | 绿色 |

### 1.3 融合姬品质

| 品质 | 解锁条件 | 能力 |
|------|---------|------|
| **N** | 剧情加入 | 2主动技能，3能力槽 |
| **R** | 完成1套立绘 | 5能力槽，随机1-2能力 |
| **SR** | 完成2套立绘 | 3主动技能 |
| **SSR** | 完成3套立绘 | 7能力槽 |
| **UR** | 完成4套立绘 | 宠物系统 |
| **LE** | 完成5套立绘 | 9能力槽 |
