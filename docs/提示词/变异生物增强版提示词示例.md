# 变异生物增强版提示词示例

## 说明
本文件展示增强版提示词扩写效果，包含：
- **外貌细节**：元素特征、皮肤/外壳纹理
- **形态描述**：职业形态、职业特征
- **环境描述**：对应地图的野外环境
- **元素统一色调**：主体色、辅助色、元素光芒
