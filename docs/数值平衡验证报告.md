## 2.1 战斗数值验证

### 战斗公式

```
damage = max(1, floor((ATK - DEF * 0.5) * random(0.9, 1.1)))
DEF 默认值 = 5
暴击倍率 = 2倍
```

### 2.1a 第1层验证：玩家(3 common随从) vs 第1层敌人

**玩家队伍（3个common随从）:**

- 机械猎犬 (HP: 350, ATK: 15)

**第1层敌人:**

- 巡逻机甲 (HP: 120, ATK: 12, CRIT: 0.1, DODGE: 0.05)
- 机械猎犬 (HP: 80, ATK: 15, CRIT: 0.15, DODGE: 0.1)

**战斗模拟:**

- 玩家单体平均伤害: 12.6 (3人总DPS: 37.8)
- 敌人单体平均伤害: 54.0 (3人总DPS: 162.0)
- 击杀机械猎犬(HP:80)所需回合: 2.1回合
- 击杀巡逻机甲(HP:120)所需回合: 3.2回合
- 玩家单体存活回合(被机械猎犬攻击): 6.5回合
- **结论: 偏简单** - 平均2.6回合击杀，低于3回合目标
