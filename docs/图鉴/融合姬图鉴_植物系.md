## 植物系概述

植物系融合姬是与变异植物完成融合的女性幸存者。她们的融合对象包括树木、藤蔓、花类、菌类、仙人掌和水草等，融合后获得了操控植物生长、释放毒素孢子、编织防御屏障等能力。植物系融合姬通常性格温和，擅长辅助和治疗，但也有以藤蔓和荆棘为武器的强力输出型。她们的皮肤上常有淡绿色脉络纹路，头发中常带有植物元素。在废土上，植物系融合姬是最受聚落欢迎的存在——她们能让荒芜的土地重新变绿。

**背景故事：**

秦若兰是核战前最顶尖的植物学家，也是融合技术最早的志愿者之一。她在核战爆发的当天正在深山研究站中进行植物辐射实验，核弹的冲击波摧毁了研究站，大量变异植物孢子涌入她的身体。奇迹般地，她没有成为失心者——融合因子与她的植物学知识产生了奇妙的共鸣，她能感知方圆百里内所有植物的生命脉动。

核战后，她独自在深山中生活了数十年，用融合能力培育出大片可食用的变异植物，拯救了数百名饥饿的幸存者。幸存者们称她为"万根之母"，因为她培育的植物根系延伸到了每一个聚落。她从不战斗，但她的藤蔓可以编织成坚不可摧的防御墙，她培育的花粉可以治愈融合姬的伤痛。

她有一个秘密：她的身体已经与深山中一棵巨大的变异古树完全融合，她不能再离开那棵树。她通过根系网络感知世界，通过藤蔓传递信息。庇护所的前任管理者曾拜访过她，她给了他一颗特殊的种子——后来成为了庇护所花园中那棵永不凋零的树。

方晴现在独居在污染森林深处，她不再制作救人的药剂——因为她害怕自己的毒素会污染一切。但她偶尔会在森林边缘留下解毒药，帮助误入森林的幸存者。

**背景故事：**

叶青是一个沉默寡言的矿工女儿。她在废弃矿坑中长大，从小就帮父亲在黑暗的矿道中采集矿石。矿坑深处有一种特殊的发光苔藓，叶青每天都会采集一些带回家照明。

一次矿坑塌方事故中，叶青被埋在了数十米深的碎石下。在黑暗和窒息中，她身边的发光苔藓疯狂生长，覆盖了她的全身。苔藓释放的氧气让她活了下来，同时完成了融合。当她从碎石中爬出来时，她的皮肤上覆盖了一层厚厚的苔藓甲壳，坚硬如铁。

苔甲让叶青成为了天然的坦克——苔藓甲壳可以吸收冲击、缓慢再生，甚至能释放治愈孢子治疗同伴。她加入了庇护所的探索队，总是走在队伍最前面，用身体挡住一切危险。队友们叫她"苔甲"，因为她就像一面永远不会倒下的墙。

叶青很少说话，但她的行动比任何语言都有力。她有一个习惯——每次战斗结束后，她会在战场上种下一丛苔藓。她说："活着的东西，就应该继续活着。"

现在，柳依在庇护所中开设了一间"歌房"，用歌声帮助受伤的融合姬恢复。孩子们叫她"柳依姐姐"，这是她在废土上最珍视的称呼。

与秦若兰不同，林芷的融合对象是一棵已经"死去"的枯树。她在一次实验中尝试让枯木重新焕发生机，融合因子意外激活了枯树中残存的生命力。融合后，她获得了"回春"的能力——她可以让枯萎的植物重新生长，甚至可以让已经石化的植物组织恢复活性。

"枯木回春"这个称号既是赞美也是叹息——她能让一切枯木逢春，唯独无法让自己恢复年轻。秦若兰曾劝她不要再过度使用能力，但林芷说："如果我的生命能让一片土地重新变绿，那就值了。"

在一次冶炼事故中，熔化的铁水泼在了赵雪身上。剧痛中她跌入了铁莲丛中，铁莲的根系缠绕住了她的伤口，金属元素通过根系进入了她的身体。融合后，她的皮肤表面生长出了一层铁莲花瓣构成的天然铠甲，坚硬程度不亚于军用防弹衣。

赵雪性格直爽，说话做事都雷厉风行。她是庇护所探索队中最可靠的坦克之一，队友们说："有铁莲在前面，我们就什么都不怕。"

她的融合对象是沙漠中最顽强的植物——变异仙人掌。在一次沙暴中，她被埋在了沙下三天三夜，体内的水分几乎耗尽。一株巨大的变异仙人掌的根系找到了她，将自己的水分和生命力分享给了她。融合后，沙琳的身体可以在极端干旱的环境中存活数周不需要饮水，她的皮肤表面长出了细小的仙人掌刺，可以弹射出去攻击敌人。

沙琳很少说话，但她的行动总是最可靠的。她是少数能独自穿越沙漠绿洲的融合姬之一，经常为庇护所运送物资和情报。

### 3. 卡牌升级逻辑

```typescript
interface UpgradeCost {
  level: number;            // 目标等级
  myceliumCost: number;     // 菌丝消耗
  materialCost: Array<{     // 材料消耗
    itemId: string;
    count: number;
  }>;
}

// 品质系数映射
const QUALITY_COEFFICIENT: Record<string, number> = {
  'N': 1.0, 'R': 1.2, 'SR': 1.5, 'SSR': 2.0, 'UR': 2.5, 'LE': 3.0
};

// 升级消耗公式：cost = baseCost × level × qualityCoefficient
function getUpgradeCost(quality: string, level: number): number {
  const baseCost = 100; // 基础消耗（可配置）
  return Math.floor(baseCost * level * QUALITY_COEFFICIENT[quality]);
}

function upgradeCard(playerId: string, cardId: string): boolean {
  const card = getPlayerCard(playerId, cardId);
  const nextLevel = card.level + 1;

  // 等级上限 = 玩家声望等级
  const playerReputationLevel = getPlayerReputationLevel(playerId);
  if (nextLevel > playerReputationLevel) return false;

  const myceliumCost = getUpgradeCost(card.quality, nextLevel);

  // 检查菌丝是否足够
  if (!hasEnoughMycelium(playerId, myceliumCost)) return false;

  // 扣除菌丝
  deductMycelium(playerId, myceliumCost);

  // 提升属性
  const growthRates = getGrowthRates(card.quality, card.profession);
  card.hp += growthRates.hp;
  card.atk += growthRates.atk;
  card.spd += growthRates.spd;
  card.level = nextLevel;

  savePlayerCard(playerId, card);
  return true;
}
```

```typescript
function fuseCard(mainCardId: string, subCardId: string): FusionResult | null {
  const mainCard = getPlayerCard(mainCardId);
  const subCard = getPlayerCard(subCardId);

  if (!mainCard || !subCard) return null;

  // 1. 验证主副卡品质相同
  if (mainCard.quality !== subCard.quality) return null;

  // 2. 计算并检查菌丝消耗
  const fusionCost = getFusionCost(mainCard.quality, mainCard.level);
  if (!hasEnoughMycelium(mainCard.ownerId, fusionCost)) return null;

  // 3. 扣除菌丝
  deductMycelium(mainCard.ownerId, fusionCost);

  // 4. 调用 FusionSystem 执行融合
  //    FusionSystem 内部处理：属性继承、技能融合、品质提升概率等
  const result = FusionSystem.fuse(mainCard, subCard);

  // 5. 返回融合结果
  return result;
}
```

### 6. 队伍编组逻辑（3+1）

```typescript
interface Team {
  teamId: string;
  memberSlots: Array<{
    slotIndex: number;      // 0-2 为融合姬槽位，3 为强化芯片槽位
    cardId?: string;        // 融合姬卡牌ID
    chipId?: string;        // 强化芯片ID（仅槽位3）
  }>;
}

// 编组规则
function setTeamMember(team: Team, slotIndex: number, cardId: string): boolean {
  // 槽位0-2：融合姬
  if (slotIndex < 3) {
    // 检查是否已编入其他队伍
    if (isCardInOtherTeam(cardId, team.teamId)) return false;

    team.memberSlots[slotIndex].cardId = cardId;
    team.memberSlots[slotIndex].chipId = undefined;
    return true;
  }

  // 槽位3：强化芯片
  if (slotIndex === 3) {
    team.memberSlots[slotIndex].chipId = cardId;
    team.memberSlots[slotIndex].cardId = undefined;
    return true;
  }

  return false;
}

// 队伍验证
function validateTeam(team: Team): boolean {
  const members = team.memberSlots.slice(0, 3).filter(s => s.cardId);
  const chip = team.memberSlots[3];

  // 至少1个融合姬
  if (members.length === 0) return false;

  // 强化芯片槽可选
  if (chip.chipId) {
    // 检查芯片是否存在
    if (!getChipConfig(chip.chipId)) return false;
  }

  return true;
}
```
