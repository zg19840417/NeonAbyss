## 6. `drop_groups.xlsx`

### 字段说明

| 字段名 | 含义 | 示例 |
|------|------|------|
| `drop_group_id` | 掉落组唯一ID | `DG_CH01_ST01_F3` |
| `fixed_drop_json` | 固定掉落配置JSON数组 | `[{\"item_id\":\"ITM_FUSION_CARD_FG001\",\"count\":1}]` |
| `random_drop_json` | 随机掉落配置JSON数组 | `[{\"item_id\":\"ITM_PF_FG001_01_R\",\"count\":1,\"weight\":100}]` |
| `notes` | 备注 | `首通3星示例` |

### 固定掉落 JSON 结构

```json
[
  { "item_id": "ITM_MYCELIUM_SMALL", "count": 50 }
]
```

### 随机掉落 JSON 结构

```json
[
  { "item_id": "ITM_PF_FG001_01_R", "count": 1, "weight": 70 },
  { "item_id": "ITM_PF_FG001_01_SR", "count": 1, "weight": 30 }
]
```
