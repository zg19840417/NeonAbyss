# 废土元年总配表手册

> 版本：v1.0  
> 日期：2026-04-08  
> 用途：作为新 Cocos 项目的统一配表入口文档。按系统模块索引组织，映射到实际 Excel 文件名。

## 1. 配表原则

- Excel 是唯一源头
- JSON 是导出结果，不直接手写核心数值
- 程序读取 JSON，不反向修改 Excel
- 每张表必须有明确用途、主键、字段解释、被谁读取
