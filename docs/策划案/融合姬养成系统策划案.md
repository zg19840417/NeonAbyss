# 废土元年 — 融合姬养成系统策划案

> **版本**：v6.0
> **版本日期**：2026-04-08
> **用途**：定义融合姬从获取到成型的完整养成路径
> **核心机制**：立绘碎片抽取 + 品质解锁系统
> **养成方式**：升级 + 升品质（取消升星系统）

---

## 一、系统概述

### 1.1 核心机制

| 机制 | 说明 |
|------|------|
| **立绘碎片抽取** | 抽取立绘碎片，集齐4张提升品质1档 |
| **品质解锁** | 不同品质解锁不同能力（技能/能力槽/宠物） |
| **养成方式** | 仅包含升级和升品质（取消升星） |

### 1.2 融合姬获取方式

| 获取方式 | 说明 | 初始品质 |
|---------|------|---------|
| **剧情加入** | 水/火/风三系融合姬通过剧情逐步加入 | N品质 |

---

## 二、立绘碎片抽取系统

### 2.1 立绘卡结构

每个融合姬拥有 **6套立绘**：

| 立绘编号 | 品质档位 | 获取方式 | 说明 |
|---------|---------|---------|------|
| **立绘1** | N品质 | 剧情加入时自带 | 默认立绘 |
| **立绘2** | R品质 | 集齐4张R碎片 | 第一次品质提升 |
| **立绘3** | SR品质 | 集齐4张SR碎片 | 第二次品质提升 |
| **立绘4** | SSR品质 | 集齐4张SSR碎片 | 第三次品质提升 |
| **立绘5** | UR品质 | 集齐4张UR碎片 | 第四次品质提升 |
| **立绘6** | LE品质 | 集齐4张LE碎片 | 第五次品质提升 |

### 2.2 碎片抽取规则

| 项目 | 说明 |
|------|------|
| **抽取货币** | 源核 |
| **抽取单位** | 单抽200源核，十连2000源核 |
| **保底机制** | SR/SSR/UR保底（同现有抽卡系统） |

**抽取产出**：
- 每次抽取获得 **1张立绘碎片**
- 碎片按品质区分（R/SR/SSR/UR/LE碎片）
- 碎片绑定到指定融合姬

### 2.3 品质提升规则

| 当前品质 | 目标品质 | 所需碎片 | 提升效果 |
|---------|---------|---------|---------|
| N | R | 4张R碎片 | 解锁第2套立绘，增加2个能力槽（5个），随机生成1~2个能力 |
| R | SR | 4张SR碎片 | 解锁第3套立绘，解锁第3主动技能 |
| SR | SSR | 4张SSR碎片 | 解锁第4套立绘，增加2个能力槽（7个） |
| SSR | UR | 4张UR碎片 | 解锁第5套立绘，宠物栏位 |
| UR | LE | 4张LE碎片 | 解锁第6套立绘，增加2个能力槽（9个） |
| LE | - | - | 满品质，不可继续提升 |

### 2.4 重复碎片处理

| 项目 | 说明 |
|------|------|
| **溢出碎片** | 集齐当前档位碎片后，再抽到同档位碎片 |
| **溢出处理** | 溢出碎片自动转换为 **晶核** |
| **晶核用途** | 可在商店兑换指定角色的任意档位碎片 |

## 三、品质解锁系统

### 3.1 品质与能力对应

| 品质 | 主动技能 | 能力槽 | 能力说明 | 宠物栏 |
|------|---------|--------|---------|---------|
| **N** | 2个（普攻+主动） | **3个** | 3个通用能力（职业+元素+系别） | 0个 |
| **R** | 2个（普攻+主动） | **5个** | 3个通用 + 1~2个随机能力 | 0个 |
| **SR** | **3个** | 5个 | 3个通用 + 1~2个随机能力 | 0个 |
| **SSR** | 3个 | **7个** | 3个通用 + 1~2个随机能力 | 0个 |
| **UR** | 3个 | 7个 | 3个通用 + 1~2个随机能力 | **1个** |
| **LE** | 3个 | **9个** | 3个通用 + 1~2个随机能力 | 1个 |

### 3.2 各品质详细说明

#### N品质（初始）
- **可以升级**
- **主动技能**：2个
  - 技能1：普通攻击
  - 技能2：主动技能
- **能力槽**：3个
  - 能力1：**职业能力**（根据坦克/输出/辅助自动生成）
  - 能力2：**元素能力**（根据水/火/风自动生成）
  - 能力3：**系别能力**（根据系别自动生成）
- **宠物栏**：0个

#### R品质
- **可以升级**
- **主动技能**：2个（普通攻击+主动技能）
- **能力槽**：5个（增加2个）
  - 3个通用能力（职业+元素+系别）
  - **新增1~2个随机能力**（从能力池抽取）
- **宠物栏**：0个

#### SR品质
- **可以升级**
- **主动技能**：3个
  - 技能1：普通攻击
  - 技能2：主动技能
  - 技能3：**第3主动技能**（全新技能）
- **能力槽**：5个
  - 3个通用能力
  - 1~2个随机能力
- **宠物栏**：0个

#### SSR品质
- **可以升级**
- **主动技能**：3个
- **能力槽**：7个（增加2个）
  - 3个通用能力
  - 1~2个随机能力
  - **仅解锁新槽位，不随机生成能力**
- **宠物栏**：0个

#### UR品质
- **可以升级**
- **主动技能**：3个
- **能力槽**：7个
  - 3个通用能力
  - 1~2个随机能力
- **宠物栏**：**1个**（解锁）
  - 可驯服变异生物作为宠物
- **宠物特性**：随机属性加成，不可取下，只能替换

#### LE品质
- **可以升级**
- **主动技能**：3个
- **能力槽**：9个（增加2个）
  - 3个通用能力
  - 1~2个随机能力
  - **仅解锁新槽位，不随机生成能力**
- **宠物栏**：1个

## 四、能力槽系统详解

### 4.1 能力槽解锁进度

| 品质 | 能力槽总数 | 增加数量 | 随机生成能力 | 说明 |
|------|-----------|---------|-------------|------|
| **N** | 3个 | - | 0个 | 仅有3个通用能力 |
| **R** | 5个 | +2个 | **1~2个** | 新增槽位随机生成 |
| **SR** | 5个 | 0 | 1~2个 | 无变化 |
| **SSR** | 7个 | +2个 | 0个 | 仅解锁，不随机 |
| **UR** | 7个 | 0 | 0个 | 无变化 |
| **LE** | 9个 | +2个 | 0个 | 仅解锁，不随机 |

### 4.2 能力分类

| 能力类型 | 来源 | 说明 | 何时获得 |
|---------|------|------|---------|
| **通用能力** | 自动生成 | 职业/元素/系别带来 | N品质自带 |
| **随机能力** | 能力池抽取 | 从融合姬能力池抽取 | R品质时随机 |

### 4.3 能力槽填充规则

| 阶段 | 说明 |
|------|------|
| **N品质** | 3个槽位被3个通用能力填满 |
| **R品质** | 新增2个槽位，随机抽取1~2个能力填入（剩余槽位空置） |
| **SSR品质** | 新增2个槽位，但**不随机生成**，槽位空置 |
| **LE品质** | 新增2个槽位，但**不随机生成**，槽位空置 |

## 五、宠物系统（UR品质解锁）

### 5.1 宠物栏位

| 品质 | 宠物栏数量 |
|------|-----------|
| N/R/SR/SSR | 0个 |
| **UR** | **1个** |
| LE | 1个 |

### 5.2 宠物获取

| 获取方式 | 说明 |
|---------|------|
| **驯服** | UR品质解锁后，可在野外战斗中驯服变异生物作为宠物 |
| **随机属性** | 驯服时随机生成属性加成 |

### 5.3 宠物特性

| 特性 | 说明 |
|------|------|
| **不可取下** | 宠物绑定后不可解除 |
| **只能替换** | 可用新的宠物替换旧的宠物（替换后旧宠物消失） |
| **随机属性** | 每次驯服随机生成不同的属性加成 |
| **属性类型** | 攻击加成/防御加成/生命加成/速度加成 等 |

## 六、通用能力说明

### 6.1 三类通用能力

| 能力类型 | 来源 | 说明 |
|---------|------|------|
| **职业能力** | 坦克/输出/辅助 | 根据职业自动生成 |
| **元素能力** | 水/火/风 | 根据元素自动生成 |
| **系别能力** | 系别 | 根据系别自动生成 |

### 6.2 职业能力示例

| 职业 | 职业能力 |
|------|---------|
| **坦克** | 铁壁防御：受到伤害-15% |
| **输出** | 狂暴之力：造成伤害+20% |
| **辅助** | 治疗结界：回合结束恢复队友5%生命 |

### 6.3 元素能力示例

| 元素 | 元素能力 |
|------|---------|
| **水** | 水之亲和：水中战斗全属性+10% |
| **火** | 火之狂暴：火属性伤害+15% |
| **风** | 风之迅捷：行动速度+10% |

### 6.4 系别能力示例

| 系别 | 系别能力 |
|------|---------|
| **植物系** | 自然之力：生命恢复+5% |
| **动物系** | 野兽本能：暴击率+10% |
| **机械系** | 机械强化：防御+15% |
| **能量系** | 能量爆发：技能伤害+15% |
| **混合系** | 融合之力：随机属性+10% |

## 八、系统流程图

### 8.1 融合姬养成路径

```
剧情加入（获得N品质融合姬）
    ↓
抽取立绘碎片
    ↓
集齐4张碎片 → 品质提升1档 → 解锁新立绘
    ↓
N品质 → 3个能力槽（3个通用能力）+ 可升级
    ↓
R品质 → 5个能力槽（+2），随机生成1~2个能力
    ↓
SR品质 → 解锁第3个主动技能
    ↓
SSR品质 → 7个能力槽（+2），仅解锁不随机
    ↓
UR品质 → 宠物栏位
    ↓
LE品质 → 9个能力槽（+2），仅解锁不随机
    ↓
满品质 = 完全体
```

### 8.2 能力槽成长图

```
N品质 ─ 3个槽 ─┬─ 职业能力
               ├─ 元素能力
               └─ 系别能力
               （3/3 满）

R品质 ─ 5个槽 ─┬─ 职业能力
               ├─ 元素能力
               ├─ 系别能力
               ├─ 随机能力（1~2个）
               └─ 空置槽位
               （3~4/5 未满）

SSR品质 ─ 7个槽 ┬─ 职业能力
                ├─ 元素能力
                ├─ 系别能力
                ├─ 随机能力（1~2个）
                ├─ 空置
                └─ 空置
                （3~4/7 未满）

UR品质 ─ 7个槽 ─（同SSR）
               + 宠物栏位

LE品质 ─ 9个槽 ─┬─ 职业能力
                ├─ 元素能力
                ├─ 系别能力
                ├─ 随机能力（1~2个）
                ├─ 空置
                ├─ 空置
                ├─ 空置
                └─ 空置
                （3~4/9 未满）
```
