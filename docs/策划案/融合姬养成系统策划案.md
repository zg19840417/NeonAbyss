# 废土元年 — 融合姬养成系统策划案

> **版本**：v6.1
> **版本日期**：2026-04-08
> **用途**：定义融合姬从获取到成型的完整养成路径
> **核心机制**：立绘碎片收集 + 品质解锁系统
> **养成方式**：升级 + 升品质（取消升星系统）

---

## 一、系统概述

### 1.1 核心机制

| 机制 | 说明 |
|------|------|
| **立绘碎片收集** | 通过刷地图战斗获取碎片，集满一整套后获得1次升品质资格 |
| **品质解锁** | 不同品质解锁不同能力（技能/能力槽/宠物） |
| **养成方式** | 仅包含升级和升品质（取消升星） |

### 1.2 融合姬属性

| 属性 | 说明 |
|------|------|
| **职业** | 坦克/输出/辅助 |
| **元素** | 水/火/风 |
| **初始品质** | N品质（剧情加入） |

> **说明**：融合姬不再有"系别"设定，只有职业和元素两个属性

### 1.3 融合姬获取方式

| 获取方式 | 说明 | 初始品质 |
|---------|------|---------|
| **剧情加入** | 通过章节剧情逐步加入 | N品质 |

### 1.4 入队机制

#### 难度制融合姬（每章3难度×10关）

| 难度 | 融合姬 | 元素 |
|------|--------|------|
| **普通** | 凌霜 | 水 |
| **噩梦** | 凌霜 | 火 |
| **地域** | 凌霜 | 风 |

> **说明**：同一章节的融合姬名字相同，元素不同

#### 入队节点

| 节点 | 触发条件 | 效果 |
|------|---------|------|
| **首次相遇** | **难度第1关** | 触发剧情，了解融合姬存在 |
| **加入卡组** | **难度第3关** | 融合姬正式加入卡组，激活图鉴，可编辑上阵 |

> **说明**：
> - 每章3个难度，每个难度对应1个融合姬（名字相同，元素不同）
> - 融合姬加入卡组后，激活图鉴
> - 第10关为章节BOSS关，通关后获得星舰零件

## 二、立绘碎片获取系统

### 2.1 立绘套装结构

> **说明**：每个融合姬拥有 **3套立绘**

| 立绘套装 | 碎片组成 | 说明 |
|---------|---------|------|
| **R立绘套装** | 30个R碎片 | 集齐后解锁R品质立绘 |
| **SR立绘套装** | 6个SR碎片 | 集齐后解锁SR品质立绘 |
| **SSR立绘套装** | 1个SSR碎片 | 集齐后解锁SSR品质立绘 |

### 2.2 碎片获取方式

#### 方式一：刷地图战斗

| 来源 | 碎片类型 | 说明 |
|------|---------|------|
| **普通关卡** | R碎片 | 战斗胜利概率掉落 |
| **精英关卡** | R/SR碎片 | 战斗胜利高概率掉落 |
| **BOSS关卡** | SR碎片 | 稳定掉落 |

> **说明**：所有碎片都需要通过刷地图战斗获取

#### 方式二：3星通关奖励

| 通关评价 | 奖励内容 |
|---------|---------|
| **首次3星通关** | 该关卡通关奖励（一次性） |
| **重复3星通关** | 晶核/道具奖励（无碎片） |

> **3星通关条件**：战斗胜利 + 满血 + 回合数限制

### 2.3 碎片掉落机制

| 关卡类型 | R碎片掉落 | SR碎片掉落 | SSR碎片掉落 |
|---------|-----------|-------------|-------------|
| 普通关卡 | 10-20% | 0% | 0% |
| 精英关卡 | 30-50% | 5-10% | 0% |
| BOSS关卡 | 100% | 20-30% | 0% |

> **SSR碎片**：仅通过3星通关奖励一次性获得

### 2.4 品质提升规则

| 当前品质 | 目标品质 | 条件 | 提升效果 |
|---------|---------|------|---------|
| N | R | 任意完成1套立绘碎片 | 增加2个能力槽（4个），随机生成1个能力 |
| R | SR | 再完成任意1套立绘碎片 | 解锁第3主动技能 |
| SR | SSR | 再完成任意1套立绘碎片 | 增加2个能力槽（6个） |
| SSR | UR | 再完成任意1套立绘碎片 | 宠物栏位 |
| UR | LE | 再完成任意1套立绘碎片 | 增加2个能力槽（8个） |
| LE | - | 满品质 | 后续完成新立绘仅用于收藏，不再继续提升品质 |

**补充规则**：
- 集满一整套立绘碎片后，**不会自动升品质**
- 玩家获得一次“可升品质”资格，由玩家手动触发
- 升品质前必须展示 **品质预览确认页**
- 升品质不额外消耗资源
- 界面不展示累积次数；若连续满足多次升品质条件，则升完一次后再次提示“可升品质”

### 2.4 重复碎片处理

| 项目 | 说明 |
|------|------|
| **有效碎片** | 在该碎片类型未达到需求数量前，可继续累积并提供属性 |
| **多余碎片** | 超过该碎片需求数量后继续抽到的同类碎片 |
| **属性规则** | 多余碎片 **不再继续提供属性加成** |
| **转换规则** | 多余碎片自动转换为对应元素点数：水 / 火 / 风 |
| **兑换用途** | 元素点数可在对应商店兑换商品 |

### 2.5 碎片属性成长规则

每张碎片在 **未满需求数量前** 都会提供属性成长。

| 项目 | 规则 |
|------|------|
| **生效时机** | 抽到后立即永久累计到该融合姬 |
| **成长类型** | 仅可配置为：生命 / 攻击 / 速度 / 全属性 |
| **成长数值** | 每个融合姬、每套立绘、每种碎片单独配表 |
| **上限规则** | 达到该碎片需求数量后停止继续加属性 |

### 2.6 展示需求

- 需要展示每套立绘的 **4种碎片收集进度**
- 需要展示每种碎片 **当前已提供的属性总量**
- 需要展示整套立绘 **是否已完成**
- 需要展示角色当前是否处于 **可升品质** 状态

## 三、品质解锁系统

### 3.1 品质与能力对应

| 品质 | 主动技能 | 能力槽 | 能力说明 | 宠物栏 |
|------|---------|--------|---------|---------|
| **N** | 2个（普攻+主动） | **3个** | 3个通用能力（职业+元素+系别） | 0个 |
| **R** | 2个（普攻+主动） | **5个** | 3个通用 + 1~2个随机能力 | 0个 |
| **SR** | **3个** | 5个 | 3个通用 + 1~2个随机能力 | 0个 |
| **SSR** | 3个 | **7个** | 3个通用 + 1~2个随机能力 | 0个 |
| **UR** | 3个 | 7个 | 3个通用 + 1~2个随机能力 | **1个** |
| **LE** | 3个 | **9个** | 3个通用 + 1~2个随机能力 | 1个 |

### 3.2 各品质详细说明

#### N品质（初始）
- **可以升级**
- **主动技能**：2个
  - 技能1：普通攻击
  - 技能2：主动技能
- **能力槽**：2个
  - 能力1：**职业能力**（根据坦克/输出/辅助自动生成）
  - 能力2：**元素能力**（根据水/火/风自动生成）
- **宠物栏**：0个

#### R品质
- **可以升级**
- **主动技能**：2个（普通攻击+主动技能）
- **能力槽**：4个（增加2个）
  - 2个通用能力（职业+元素）
  - **新增1个随机能力**（从能力池抽取）
- **宠物栏**：0个

#### SR品质
- **可以升级**
- **主动技能**：3个
  - 技能1：普通攻击
  - 技能2：主动技能
  - 技能3：**第3主动技能**（全新技能）
- **能力槽**：4个
  - 2个通用能力
  - 1个随机能力
- **宠物栏**：0个

#### SSR品质
- **可以升级**
- **主动技能**：3个
- **能力槽**：6个（增加2个）
  - 2个通用能力
  - 1个随机能力
  - **仅解锁新槽位，不随机生成能力**
- **宠物栏**：0个

#### UR品质
- **可以升级**
- **主动技能**：3个
- **能力槽**：6个
  - 2个通用能力
  - 1个随机能力
- **宠物栏**：**1个**（解锁）
  - 可驯服变异生物作为宠物
- **宠物特性**：随机属性加成，不可取下，只能替换

#### LE品质
- **可以升级**
- **主动技能**：3个
- **能力槽**：8个（增加2个）
  - 2个通用能力
  - 1个随机能力
  - **仅解锁新槽位，不随机生成能力**
- **宠物栏**：1个

## 四、能力槽系统详解

### 4.1 能力槽解锁进度

| 品质 | 能力槽总数 | 增加数量 | 随机生成能力 | 说明 |
|------|-----------|---------|-------------|------|
| **N** | 2个 | - | 0个 | 仅有2个通用能力 |
| **R** | 4个 | +2个 | **1个** | 新增槽位随机生成 |
| **SR** | 4个 | 0 | 1个 | 无变化 |
| **SSR** | 6个 | +2个 | 0个 | 仅解锁，不随机 |
| **UR** | 6个 | 0 | 0个 | 无变化 |
| **LE** | 8个 | +2个 | 0个 | 仅解锁，不随机 |

### 4.2 能力分类

| 能力类型 | 来源 | 说明 | 何时获得 |
|---------|------|------|---------|
| **通用能力** | 自动生成 | 职业/元素各带来1个 | N品质自带 |
| **随机能力** | 能力池抽取 | 从融合姬能力池抽取 | R品质时随机 |

### 4.3 能力槽填充规则

| 阶段 | 说明 |
|------|------|
| **N品质** | 2个槽位被2个通用能力填满 |
| **R品质** | 新增2个槽位，随机抽取1个能力填入（剩余槽位空置） |
| **SSR品质** | 新增2个槽位，但**不随机生成**，槽位空置 |
| **LE品质** | 新增2个槽位，但**不随机生成**，槽位空置 |

## 五、宠物系统（UR品质解锁）

### 5.1 宠物栏位

| 品质 | 宠物栏数量 |
|------|-----------|
| N/R/SR/SSR | 0个 |
| **UR** | **1个** |
| LE | 1个 |

### 5.2 宠物获取

| 获取方式 | 说明 |
|---------|------|
| **驯服** | UR品质解锁后，可在野外战斗中驯服变异生物作为宠物 |
| **随机属性** | 驯服时随机生成属性加成 |

### 5.3 宠物特性

| 特性 | 说明 |
|------|------|
| **不可取下** | 宠物绑定后不可解除 |
| **只能替换** | 可用新的宠物替换旧的宠物（替换后旧宠物消失） |
| **随机属性** | 每次驯服随机生成不同的属性加成 |
| **属性类型** | 攻击加成/防御加成/生命加成/速度加成 等 |

## 六、通用能力说明

### 6.1 三类通用能力

| 能力类型 | 来源 | 说明 |
|---------|------|------|
| **职业能力** | 坦克/输出/辅助 | 根据职业自动生成 |
| **元素能力** | 水/火/风 | 根据元素自动生成 |
| **系别能力** | 系别 | 根据系别自动生成 |

### 6.2 职业能力示例

| 职业 | 职业能力 |

### 6.3 元素能力示例

| 元素 | 元素能力 |
|------|---------|
| **水** | 水之亲和：水中战斗全属性+10% |
| **火** | 火之狂暴：火属性伤害+15% |
| **风** | 风之迅捷：行动速度+10% |

### 6.4 系别能力示例

| 系别 | 系别能力 |
|------|---------|
| **植物系** | 自然之力：生命恢复+5% |
| **动物系** | 野兽本能：暴击率+10% |
| **机械系** | 机械强化：防御+15% |
| **能量系** | 能量爆发：技能伤害+15% |
| **混合系** | 融合之力：随机属性+10% |

## 八、系统流程图

### 8.1 融合姬养成路径

```
剧情加入（获得N品质融合姬）
    ↓
抽取指定角色 + 指定立绘套装 + 指定品质的碎片
    ↓
累计碎片属性成长 + 推进套装收集进度
    ↓
任意完成1整套立绘 → 获得1次升品质资格
    ↓
手动确认品质预览 → 升品质1档
    ↓
N品质 → 2个能力槽（2个通用能力）+ 可升级
    ↓
R品质 → 4个能力槽（+2），随机生成1个能力
    ↓
SR品质 → 解锁第3个主动技能
    ↓
SSR品质 → 6个能力槽（+2），仅解锁不随机
    ↓
UR品质 → 宠物栏位
    ↓
LE品质 → 8个能力槽（+2），仅解锁不随机
    ↓
满品质 = 完全体
```

### 8.2 能力槽成长图

```
N品质 ─ 3个槽 ─┬─ 职业能力
               ├─ 元素能力
               └─ 系别能力
               （3/3 满）

R品质 ─ 5个槽 ─┬─ 职业能力
               ├─ 元素能力
               ├─ 系别能力
               ├─ 随机能力（1~2个）
               └─ 空置槽位
               （3~4/5 未满）

SSR品质 ─ 7个槽 ┬─ 职业能力
                ├─ 元素能力
                ├─ 系别能力
                ├─ 随机能力（1~2个）
                ├─ 空置
                └─ 空置
                （3~4/7 未满）

UR品质 ─ 7个槽 ─（同SSR）
               + 宠物栏位

LE品质 ─ 9个槽 ─┬─ 职业能力
                ├─ 元素能力
                ├─ 系别能力
                ├─ 随机能力（1~2个）
                ├─ 空置
                ├─ 空置
                ├─ 空置
                └─ 空置
                （3~4/9 未满）
```
