## 动物系概述

动物系融合姬是与变异动物完成基因嵌合的女性幸存者。她们的融合对象包括猫科、犬科、鸟类、鱼类、爬行类和昆虫等，融合后获得了对应动物的感官能力、身体特征和部分本能。动物系融合姬是最早出现的种族之一，核战前军方"方舟计划"就已开始相关实验。她们通常拥有超凡的感知力、敏捷性和战斗本能，是废土上最全能的战士种族。

**背景故事：**

沈霜是核战前军方"方舟计划"的核心实验体。方舟计划旨在通过动物基因融合技术制造超级士兵，沈霜是唯一一个存活到核战爆发的实验体——她与一头变异雪豹的基因完成了深度嵌合。核弹落下时，军方基地被摧毁，她从废墟中爬出，成为了废土上第一个动物系融合姬。

她的融合程度远超后来的融合姬：她的瞳孔是竖直的雪豹金瞳，能在完全黑暗中视物；她的脊椎两侧长出了一排可伸缩的骨刺，奔跑时骨刺会自动展开形成锯齿状的防御结构；她的指甲是半透明的利爪，可以轻松撕裂防弹衣。但代价同样残酷——她能听到方圆十公里内所有动物的心跳声，这种持续的感官过载让她长期处于偏头痛和失眠之中。

核战后，沈霜独自在废土上游荡了十五年。她猎杀过军阀、撕碎过变异巨兽、也曾在深夜独自蜷缩在废墟中发抖。庇护所的前任管理者找到她时，她正站在一座被军阀屠杀殆尽的聚落中央，浑身是血，一言不发。她说："我能听到它们死前的恐惧，和动物一样。"

她是庇护所早期最强大的战力，也是主线剧情中揭开"方舟计划"真相的关键人物。她的存在证明了融合技术并非核战后的偶然产物，而是人类早已开始的预谋。

楚云是废土上唯一能够真正飞行的融合姬。她的融合对象是核战后出现在辐射云层中的一种变异游隼——这种游隼的翼展达到了六米，羽毛边缘锋利如刀片，飞行速度可以突破音障。

"裂空"这个称号来自她的攻击方式：她从高空俯冲时，翅膀边缘的锋利羽毛会形成一道空气刀刃，可以切割装甲车辆。她的飞行速度让她成为了废土上最好的侦察兵——她能在一天之内侦察整个辐射荒原。

但飞行是有代价的。每次高空飞行后，她的肺部都会因为辐射云的侵蚀而剧烈疼痛，她不得不定期服用抗辐射药剂。她的羽毛也在缓慢脱落，她知道总有一天她会飞不起来。在那之前，她要飞遍废土的每一个角落，把地面上看不到的景象记录下来。

**背景故事：**

许澜是废土上最出色的水下探索者。她的融合对象是一种生活在地下暗河深处的变异盲鱼——这种鱼没有眼睛，但能通过水压变化精确感知周围的一切。

许澜原本是一名水下考古学家，核战前专门研究沉船和海底遗迹。核战后地表水源大面积污染，她转入地下暗河系统寻找清洁水源。在一次深潜中，她的氧气设备故障，在溺水的恐惧中，一条变异盲鱼游到了她面前——鱼身的黏液接触到了她的皮肤，融合因子在水中完成了融合。

融合后的许澜颈部两侧长出了鱼鳃，可以在水下无限时呼吸。她的皮肤覆盖了一层半透明的黏液鳞片，在水中几乎完全隐形。她的手指间长出了蹼，游泳速度是普通人的十倍。最奇特的是她的感知能力——她能通过水压变化"看见"水中的所有物体，精度甚至超过了声纳。

"深渊鳃"这个称号来自她探索过的最深的地方——地下八百米的暗河深渊，那里没有任何光线，水压足以压碎普通人的骨骼。她为庇护所找到了三条清洁水源，拯救了数百人的生命。但她很少提起深渊中的经历，只是说："下面有些东西，不该被看到。"

### UR-025：蜕鳞·秦蜕

| 属性 | 值 |
|------|-----|
| 品质 | UR |
| 元素 | 火 |
| 职业 | 坦克 |
| 种族 | 动物系 |
| 子种族 | 爬行 |

**背景故事：**

秦蜕是废土上最令人不安的融合姬之一。她的融合对象是一种变异的沙漠响尾蛇——这种蛇的鳞片在高温下会变得坚硬如陶瓷，蛇毒可以腐蚀金属。

秦蜕原本是一个走私贩的女儿，从小跟着父亲在各个聚落之间运送物资。在一次穿越沙漠时，车队遭到了变异响尾蛇群的袭击，她被蛇群围困在一块岩石上。一条体型异常巨大的响尾蛇咬住了她的手臂，大量蛇毒和融合因子同时进入了她的血液。

融合的过程持续了三天三夜，高烧让她陷入了昏迷。当她醒来时，她的左臂已经完全被蛇鳞覆盖，右眼变成了竖瞳，舌头也变得细长分叉。她的体温可以自主调节到七十度以上，在这个温度下，她体表的蛇鳞会变得比陶瓷还硬。

"蜕鳞"这个称号来自她的特殊能力——她可以像蛇一样蜕皮。每次蜕皮后，新生的鳞片比旧鳞更坚硬。但蜕皮的过程极其痛苦，她会失去所有的防御力长达数小时。她总是在安全的隐蔽处蜕皮，身边放着一把上了膛的枪。

秦蜕性格冷酷，说话总是带着蛇一样的嘶嘶声。她不信任任何人，因为她父亲在她融合后第一个反应是试图把她卖给军阀做实验品。她亲手打碎了父亲的走私车，独自走进了沙漠。

韩烈是废土上冲锋速度最快的融合姬。她的融合对象是一头变异的赤狐——核辐射让这头狐狸的毛发变成了炽红色，奔跑时四爪会擦出火星。

韩烈是一个流浪商队的护卫，性格豪爽泼辣，说话声音大到隔着三条街都能听见。在一次护送任务中，商队遭到了军阀伏击。她独自断后，被逼到了一处悬崖边。就在她准备跳崖的时候，一头赤红色的变异狐狸从她脚下的灌木丛中窜出，撞进了她的怀里。狐狸的毛发像火一样灼热，融合因子通过接触完成了融合。

融合后的韩烈获得了超凡的爆发速度和火焰抗性。她的头发变成了赤红色，头顶长出了一对狐狸耳朵，身后是一条蓬松的赤色尾巴。她奔跑时四爪——不，四脚——会擦出火星，速度可以达到每小时一百二十公里。她穿着用防火织物和金属片拼接的轻型护甲，腰间别着两把短刀。

"赤鬃"这个称号来自她冲锋时的样子——赤红色的头发和尾巴在身后飘扬，像一团奔跑的火焰。她的战斗风格是"一击脱离"：以极限速度冲入敌阵，短刀连斩后立刻撤退。队友们说，看她冲锋就像看一道红色的闪电。

韩烈有一个软肋：她怕打雷。每次暴风雨来临时，她会缩在角落里用尾巴捂住耳朵，和一只受惊的狐狸没有任何区别。

**背景故事：**

苏锦是废土上最冷酷的战士之一。她的融合对象是一种生活在冻土层地下湖中的变异冰鱼——这种鱼的血液中含有天然防冻蛋白，能在零下六十度的环境中正常活动。

苏锦原本是一个冰库管理员，核战后她所在的地下冰库成为了附近幸存者的避难所。但冰库的制冷系统开始故障，温度不断上升，储存的食物面临腐烂。苏锦独自深入冻土层寻找冰源，在地下湖中遭遇了变异冰鱼群的攻击。一条体型最大的冰鱼撞碎了她的防护服，冰冷的血液和融合因子同时涌入了她的伤口。

融合后的苏锦体温常年维持在零下五度左右，她的皮肤表面覆盖了一层透明的冰晶鳞片，触摸她的皮肤会感到刺骨的寒冷。她可以从手掌释放极低温的水流，瞬间将目标冻结。她的攻击方式是将冰水喷射到敌人身上，然后一拳击碎被冻住的部位。

苏锦性格冷淡，很少与人交流。她总是独来独往，即使在庇护所中也保持着距离。但有人注意到，每到冬天，她会在庇护所的水管上凝结出精美的冰花——那是她唯一表达善意的方式。

周蜥原本是一名建筑工人，核战后她利用自己的建筑知识帮助幸存者修建避难所。在一次外出采集建材时，她被一头变异铠甲蜥蜴咬伤了腿。蜥蜴的牙齿折断在了她的伤口中，金属化的骨骼碎片成为了融合的媒介。

周蜥是个话痨，干活的时候嘴里一刻不停。她喜欢给每块鳞片起名字，说它们是她的"小工人们"。队友们觉得她吵，但没有人会拒绝她的保护。

### SSR-030：千丝·何蝶

| 属性 | 值 |
|------|-----|
| 品质 | SSR |
| 元素 | 风 |
| 职业 | 辅助 |
| 种族 | 动物系 |
| 子种族 | 昆虫 |

**背景故事：**

何蝶是废土上最奇特的融合姬之一。她的融合对象是一种变异的蚕蛾——核辐射让这种蚕蛾的体型膨胀到了半米长，吐出的丝线强度是凯夫拉纤维的五倍。

何蝶原本是一个纺织工的女儿，在废土上一个以纺织闻名的小型聚落长大。聚落被军阀攻陷后，她被关进了纺织工坊做苦力。在工坊的地下室里，她发现了大量的变异蚕茧——军阀让她用这些蚕茧纺织防弹丝线。在长期接触中，蚕蛾的融合因子通过她的呼吸道进入了体内。

### 3. 卡牌升级逻辑

```typescript
interface UpgradeCost {
  level: number;            // 目标等级
  myceliumCost: number;     // 菌丝消耗
  materialCost: Array<{     // 材料消耗
    itemId: string;
    count: number;
  }>;
}

// 品质系数映射
const QUALITY_COEFFICIENT: Record<string, number> = {
  'N': 1.0, 'R': 1.2, 'SR': 1.5, 'SSR': 2.0, 'UR': 2.5, 'LE': 3.0
};

// 升级消耗公式：cost = baseCost × level × qualityCoefficient
function getUpgradeCost(quality: string, level: number): number {
  const baseCost = 100; // 基础消耗（可配置）
  return Math.floor(baseCost * level * QUALITY_COEFFICIENT[quality]);
}

function upgradeCard(playerId: string, cardId: string): boolean {
  const card = getPlayerCard(playerId, cardId);
  const nextLevel = card.level + 1;

  // 等级上限 = 玩家声望等级
  const playerReputationLevel = getPlayerReputationLevel(playerId);
  if (nextLevel > playerReputationLevel) return false;

  const myceliumCost = getUpgradeCost(card.quality, nextLevel);

  // 检查菌丝是否足够
  if (!hasEnoughMycelium(playerId, myceliumCost)) return false;

  // 扣除菌丝
  deductMycelium(playerId, myceliumCost);

  // 提升属性
  const growthRates = getGrowthRates(card.quality, card.profession);
  card.hp += growthRates.hp;
  card.atk += growthRates.atk;
  card.spd += growthRates.spd;
  card.level = nextLevel;

  savePlayerCard(playerId, card);
  return true;
}
```

```typescript
function fuseCard(mainCardId: string, subCardId: string): FusionResult | null {
  const mainCard = getPlayerCard(mainCardId);
  const subCard = getPlayerCard(subCardId);

  if (!mainCard || !subCard) return null;

  // 1. 验证主副卡品质相同
  if (mainCard.quality !== subCard.quality) return null;

  // 2. 计算并检查菌丝消耗
  const fusionCost = getFusionCost(mainCard.quality, mainCard.level);
  if (!hasEnoughMycelium(mainCard.ownerId, fusionCost)) return null;

  deductMycelium(mainCard.ownerId, fusionCost);

  // 4. 调用 FusionSystem 执行融合
  //    FusionSystem 内部处理：属性继承、技能融合、品质提升概率等
  const result = FusionSystem.fuse(mainCard, subCard);

  return result;
}
```

```typescript
interface Team {
  teamId: string;
  memberSlots: Array<{
    slotIndex: number;      // 0-2 为融合姬槽位，3 为强化芯片槽位
    cardId?: string;        // 融合姬卡牌ID
    chipId?: string;        // 强化芯片ID（仅槽位3）
  }>;
}

// 编组规则
function setTeamMember(team: Team, slotIndex: number, cardId: string): boolean {
  // 槽位0-2：融合姬
  if (slotIndex < 3) {
    // 检查是否已编入其他队伍
    if (isCardInOtherTeam(cardId, team.teamId)) return false;

    team.memberSlots[slotIndex].cardId = cardId;
    team.memberSlots[slotIndex].chipId = undefined;
    return true;
  }

  // 槽位3：强化芯片
  if (slotIndex === 3) {
    team.memberSlots[slotIndex].chipId = cardId;
    team.memberSlots[slotIndex].cardId = undefined;
    return true;
  }

  return false;
}

// 队伍验证
function validateTeam(team: Team): boolean {
  const members = team.memberSlots.slice(0, 3).filter(s => s.cardId);
  const chip = team.memberSlots[3];

  // 至少1个融合姬
  if (members.length === 0) return false;

  // 强化芯片槽可选
  if (chip.chipId) {
    // 检查芯片是否存在
    if (!getChipConfig(chip.chipId)) return false;
  }

  return true;
}
```
