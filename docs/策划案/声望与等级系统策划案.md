## 一、系统概述

### 1.1 声望等级定义

声望等级（Reputation Level）是**账号维度的全局进度系统**，代表玩家在废土世界中的影响力与探索深度。声望等级与融合姬个体等级无关，是贯穿全部游戏内容的纵向成长线。

**核心特征：**

| 特征 | 说明 |
|------|------|
| 维度 | 账号绑定，不随角色切换而改变 |
| 范围 | Lv1 ~ Lv99，纯数字显示，无命名段位 |
| 作用 | 控制功能解锁、地图开放、奖励发放 |
| 重置 | 永不重置，单向递增 |
| 显示 | 主界面顶部状态栏、个人资料页、等级详情面板 |

**设计原则：**

1. **声望 = 进度标尺**：声望等级直观反映玩家在游戏中的整体进度，不设虚名段位，纯数字一目了然。
2. **解锁 = 动力**：每升5级/10级有明确奖励，关键等级解锁新地图或核心功能，形成持续升级动力。
3. **节奏 = 体验**：前期升级快（引导阶段），中期平稳（养成阶段），后期放缓（终局阶段），配合内容消耗节奏。
4. **奖励 = 即时正反馈**：升级时全屏特效 + 奖励弹窗，让每次升级都有仪式感。

### 1.2 等级范围与解锁机制

#### 1.2.1 等级分段

| 阶段 | 等级范围 | 定位 | 单级耗时（预估） | 累计耗时（预估） |
|------|---------|------|-----------------|-----------------|
| 新手期 | Lv1 ~ Lv10 | 引导+基础教学 | 5 ~ 15分钟 | 约1.5小时 |
| 成长期 | Lv11 ~ Lv30 | 核心玩法展开 | 30分钟 ~ 2小时 | 约2 ~ 3天 |
| 深入期 | Lv31 ~ Lv60 | 中期内容消耗 | 2 ~ 6小时 | 约2 ~ 3周 |
| 挑战期 | Lv61 ~ Lv80 | 高级内容解锁 | 6 ~ 12小时 | 约1 ~ 2个月 |
| 终局期 | Lv81 ~ Lv99 | 终局追求 | 12 ~ 24小时 | 约2 ~ 4个月 |

#### 1.2.2 解锁机制总则

- 声望等级达到对应数值后，自动解锁对应内容
- 解锁时弹出**解锁提示弹窗**，展示解锁内容图标与名称
- 已解锁内容在对应功能入口处显示**可用状态**
- 未解锁内容在对应功能入口处显示**锁定状态**（灰色 + 锁图标 + 解锁条件文字）
- 地图解锁后，该地图的全部关卡立即可见，但具体关卡仍需按序推进

#### 2.1.1 通关关卡经验

野外地图关卡首次通关时获得的声望经验，与关卡所在地图和关卡序号相关：

```
关卡声望经验 = 地图基础经验 × 关卡序号系数

地图基础经验：
  地图1（庇护所外围）：10
  地图2（废弃城区）：15
  地图3（荒野公路）：20
  地图4（污染森林）：25
  地图5（干涸河床）：30
  地图6（军事前哨）：35
  地图7（沙漠绿洲）：40
  地图8（军事基地）：45
  地图9（地下设施）：50
  地图10（辐射荒原）：55

关卡序号系数：
  第1关：1.0
  第2关：1.1
  第3关：1.2
  ...
  第10关：1.9
```

**示例**：地图3第5关首次通关 = 20 × 1.4 = 28经验

#### 2.1.2 禁区探索经验

禁区层数首次通关时获得的声望经验：

```
禁区声望经验 = 20 + (层数 - 1) × 8

即：
  第1层：20
  第5层：52
  第10层：92
  第15层：132
  第20层：172
```

### 2.2 经验公式

#### 2.2.1 每级所需经验公式

```
expRequired(level) = floor(baseExp × level × growthFactor)
```

**参数定义：**

| 参数 | 值 | 说明 |
|------|-----|------|
| baseExp | 8 | 基础经验系数 |
| growthFactor | 1.15 | 每级增长因子（复利） |
| level | 目标等级 | 当前要升到的等级 |

**分段调整规则：**

为平衡各阶段体验，在每10级的整数关卡处进行经验值压缩（向下取整后额外乘以压缩系数）：

| 等级段 | 压缩系数 | 说明 |
|--------|---------|------|
| Lv1 ~ Lv10 | 1.0 | 无压缩 |
| Lv11 ~ Lv20 | 1.0 | 无压缩 |
| Lv21 ~ Lv30 | 1.0 | 无压缩 |
| Lv31 ~ Lv40 | 1.0 | 无压缩 |
| Lv41 ~ Lv50 | 1.0 | 无压缩 |
| Lv51 ~ Lv60 | 1.0 | 无压缩 |
| Lv61 ~ Lv70 | 1.0 | 无压缩 |
| Lv71 ~ Lv80 | 1.0 | 无压缩 |
| Lv81 ~ Lv90 | 1.0 | 无压缩 |
| Lv91 ~ Lv99 | 1.0 | 无压缩 |

> 注：压缩系数默认为1.0，实际数值由 levelUp.json 配置表直接定义，公式仅用于生成初始配表。如需调整节奏，数值策划可直接修改配置表中的 expRequired 字段。

#### 2.2.2 经验计算示例

```
Lv2 所需经验 = floor(8 × 2 × 1.15) = floor(18.4) = 18
Lv5 所需经验 = floor(8 × 5 × 1.15) = floor(46.0) = 46
Lv10 所需经验 = floor(8 × 10 × 1.15) = floor(92.0) = 92
Lv20 所需经验 = floor(8 × 20 × 1.15) = floor(184.0) = 184
Lv30 所需经验 = floor(8 × 30 × 1.15) = floor(276.0) = 276
Lv50 所需经验 = floor(8 × 50 × 1.15) = floor(460.0) = 460
Lv70 所需经验 = floor(8 × 70 × 1.15) = floor(644.0) = 644
Lv99 所需经验 = floor(8 × 99 × 1.15) = floor(910.8) = 910
```

### 2.3 经验衰减机制（可选）

当前版本**不启用**经验衰减机制。所有经验来源均按固定值发放，不随等级增长而衰减。

**预留衰减规则（未来版本可选启用）：**

```
实际获得经验 = 基础经验 × max(0.3, 1 - (当前等级 - 来源推荐等级) × 0.05)

最低衰减下限：30%（即最多衰减70%）
衰减触发条件：玩家等级超过来源推荐等级
```

> 此规则仅作为预留设计，v1.0版本不实施。

```
玩家获得经验
    │
    ▼

    │
    ▼

    │
    ├── 否 → 更新经验条UI，结束
    │
    └── 是 → 触发升级流程
              │
              ▼
          扣除所需经验，等级+1
              │
              ▼
          判断：是否仍有溢出经验？
              │
              ├── 是 → 继续升级（支持连升）
              │
              └── 否 → 结束升级循环
                        │

                    播放升级特效
                        │
                        ▼
                    弹出奖励弹窗
                        │
                        ▼
                    检查功能解锁
                        │
                        ▼
                    如有解锁 → 弹出解锁提示
```

### 9.1 LevelUpSystem 类设计

```typescript
/**
 * 声望等级系统
 * 管理账号声望等级、经验计算、奖励发放、功能解锁
 */
class LevelUpSystem {
    // ========== 单例 ==========
    private static _instance: LevelUpSystem;
    public static get instance(): LevelUpSystem {
        if (!LevelUpSystem._instance) {
            LevelUpSystem._instance = new LevelUpSystem();
        }
        return LevelUpSystem._instance;
    }

    // ========== 数据 ==========
    private _currentLevel: number;       // 当前声望等级 (1-99)
    private _currentExp: number;         // 当前经验值
    private _levelUpData: LevelUpData[]; // 配置表数据
    private _claimedRewards: Set<number>; // 已领取奖励的等级集合

    // ========== 事件 ==========
    public onLevelUp: Phaser.Signal<{ oldLevel: number; newLevel: number; rewards: Reward[] }>;
    public onExpChanged: Phaser.Signal<{ currentExp: number; expToNext: number; progress: number }>;
    public onFeatureUnlocked: Phaser.Signal<{ featureId: string; featureName: string }>;

    // ========== 初始化 ==========
    /**
     * 初始化声望系统
     * @param saveData 存档中读取的声望数据
     */
    public init(saveData: SaveReputationData): void {
        this._levelUpData = this.loadConfig(); // 从 levelUp.json 加载
        this._currentLevel = saveData.level || 1;
        this._currentExp = saveData.exp || 0;
        this._claimedRewards = new Set(saveData.claimedLevels || []);
    }

    // ========== 经验计算 ==========

    /**
     * 获取升到指定等级所需经验
     * @param level 目标等级
     * @returns 所需经验值
     */
    public getExpRequired(level: number): number {
        if (level <= 1) return 0;
        if (level > 99) return Infinity;
        const data = this._levelUpData.find(d => d.level === level);
        return data ? data.expToNext : 0;
    }

    /**
     * 获取从Lv1升到指定等级的累计经验
     * @param level 目标等级
     * @returns 累计经验值
     */
    public getTotalExp(level: number): number {
        const data = this._levelUpData.find(d => d.level === level);
        return data ? data.totalExp : 0;
    }

    /**
     * 获取当前经验进度百分比
     * @returns 0.0 ~ 1.0
     */
    public getExpProgress(): number {
        const expToNext = this.getExpRequired(this._currentLevel + 1);
        if (this._currentLevel >= 99) return 1.0;
        if (expToNext <= 0) return 1.0;
        return Math.min(1.0, this._currentExp / expToNext);
    }

    // ========== 经验获取 ==========

    /**
     * 增加经验值（核心方法）
     * @param amount 经验量
     * @param source 经验来源标识
     */
    public addExp(amount: number, source: ExpSource): void {
        if (this._currentLevel >= 99) return; // 满级不再获得经验

        this._currentExp += amount;

        // 检查是否升级
        this.checkLevelUp();

        // 触发经验变化事件
        this.onExpChanged.dispatch({
            currentExp: this._currentExp,
            expToNext: this.getExpRequired(this._currentLevel + 1),
            progress: this.getExpProgress()
        });

        // 保存数据
        this.save();
    }

    /**
     * 检查并处理升级逻辑（支持连升）
     */
    private checkLevelUp(): void {
        const maxLevel = 99;
        let leveledUp = false;
        const oldLevel = this._currentLevel;

        while (this._currentLevel < maxLevel) {
            const expToNext = this.getExpRequired(this._currentLevel + 1);
            if (this._currentExp >= expToNext) {
                this._currentExp -= expToNext;
                this._currentLevel++;
                leveledUp = true;
            } else {
                break;
            }
        }

        if (leveledUp) {
            // 收集所有升级奖励
            const rewards = this.collectLevelUpRewards(oldLevel, this._currentLevel);

            // 触发升级事件
            this.onLevelUp.dispatch({
                oldLevel: oldLevel,
                newLevel: this._currentLevel,
                rewards: rewards
            });

            // 检查功能解锁
            this.checkFeatureUnlocks(oldLevel, this._currentLevel);
        }
    }

    // ========== 奖励系统 ==========

    /**
     * 收集从 oldLevel 到 newLevel 之间所有等级的奖励
     */
    private collectLevelUpRewards(oldLevel: number, newLevel: number): Reward[] {
        const rewards: Reward[] = [];

        for (let lv = oldLevel + 1; lv <= newLevel; lv++) {
            const data = this._levelUpData.find(d => d.level === lv);
            if (!data) continue;

            // 源核奖励（每5级）
            if (data.sourceCoreReward > 0) {
                rewards.push({
                    type: 'sourceCore',
                    amount: data.sourceCoreReward,
                    level: lv
                });
            }

            // 头像框奖励（每10级）
            if (data.avatarFrame) {
                rewards.push({
                    type: 'avatarFrame',
                    id: data.avatarFrame,
                    level: lv
                });
            }

            // 称号奖励（每10级）
            if (data.title) {
                rewards.push({
                    type: 'title',
                    text: data.title,
                    level: lv
                });
            }
        }

        return rewards;
    }

    /**
     * 领取等级奖励
     * @param level 要领取奖励的等级
     * @returns 领取的奖励列表，如果已领取或等级未达到则返回null
     */
    public claimReward(level: number): Reward[] | null {
        if (this._currentLevel < level) return null;
        if (this._claimedRewards.has(level)) return null;

        const data = this._levelUpData.find(d => d.level === level);
        if (!data) return null;

        const rewards: Reward[] = [];

        // 发放源核
        if (data.sourceCoreReward > 0) {
            CurrencySystem.instance.addCurrency('sourceCore', data.sourceCoreReward);
            rewards.push({ type: 'sourceCore', amount: data.sourceCoreReward, level });
        }

        // 发放头像框
        if (data.avatarFrame) {
            AvatarSystem.instance.unlockFrame(data.avatarFrame);
            rewards.push({ type: 'avatarFrame', id: data.avatarFrame, level });
        }

        // 发放称号
        if (data.title) {
            TitleSystem.instance.unlockTitle(data.title);
            rewards.push({ type: 'title', text: data.title, level });
        }

        this._claimedRewards.add(level);
        this.save();

        return rewards;
    }

    // ========== 功能解锁 ==========

    /**
     * 检查从 oldLevel 到 newLevel 之间是否有新功能解锁
     */
    private checkFeatureUnlocks(oldLevel: number, newLevel: number): void {
        for (let lv = oldLevel + 1; lv <= newLevel; lv++) {
            const data = this._levelUpData.find(d => d.level === lv);
            if (!data || !data.unlockFeature) continue;

            this.onFeatureUnlocked.dispatch({
                featureId: data.unlockFeature,
                featureName: data.unlockContent || data.unlockFeature
            });

            // 注册功能解锁状态
            FeatureUnlockSystem.instance.unlock(data.unlockFeature);
        }
    }

    /**
     * 检查指定功能是否已解锁
     * @param featureId 功能ID
     */
    public isFeatureUnlocked(featureId: string): boolean {
        const data = this._levelUpData.find(d => d.unlockFeature === featureId);
        if (!data) return true; // 不在配置表中的功能默认解锁
        return this._currentLevel >= data.level;
    }

    /**
     * 获取指定功能的解锁等级
     * @param featureId 功能ID
     */
    public getFeatureUnlockLevel(featureId: string): number {
        const data = this._levelUpData.find(d => d.unlockFeature === featureId);
        return data ? data.level : 1;
    }

    // ========== Getter ==========

    public get currentLevel(): number { return this._currentLevel; }
    public get currentExp(): number { return this._currentExp; }
    public get isMaxLevel(): boolean { return this._currentLevel >= 99; }

    // ========== 存档 ==========

    public save(): void {
        SaveManager.instance.set('reputation', {
            level: this._currentLevel,
            exp: this._currentExp,
            claimedLevels: Array.from(this._claimedRewards)
        });
    }

    // ========== 配置加载 ==========

    private loadConfig(): LevelUpData[] {
        return GameCache.instance.getJSON('levelUp');
    }
}
```

```typescript
    // 2. 判断是否为关键等级（每10级）
    const isKeyLevel = data.newLevel % 10 === 0;

    if (isKeyLevel) {
        // 关键等级：播放特殊特效
        UILayer.instance.playEffect('level_up_key');
    }

    // 3. 延迟弹出奖励弹窗
    this.time.delayedCall(isKeyLevel ? 2000 : 1000, () => {
        UILayer.instance.openWindow('LevelUpRewardWindow', {
            level: data.newLevel,
            rewards: data.rewards,
            isKeyLevel: isKeyLevel
        });
    });
});

// ========== 功能解锁事件监听 ==========
LevelUpSystem.instance.onFeatureUnlocked.add((data) => {
    // 延迟弹出解锁提示（在奖励弹窗之后）
    this.time.delayedCall(500, () => {
        UILayer.instance.openWindow('FeatureUnlockWindow', {
            featureId: data.featureId,
            featureName: data.featureName
        });
    });
});
```

### 9.4 功能解锁检查集成

各功能入口在打开前检查声望等级是否满足：

```typescript
// ========== 通用功能解锁检查 ==========
function openFeature(featureId: string): void {
    if (!LevelUpSystem.instance.isFeatureUnlocked(featureId)) {
        const requiredLevel = LevelUpSystem.instance.getFeatureUnlockLevel(featureId);
        UILayer.instance.showToast(`声望等级 Lv.${requiredLevel} 解锁`);
        return;
    }
    // 正常打开功能
    FeatureManager.instance.open(featureId);
}

// ========== 地图解锁检查 ==========
function openMap(zoneId: string): void {
    const requiredLevel = LevelUpSystem.instance.getMapUnlockLevel(zoneId);
    if (LevelUpSystem.instance.currentLevel < requiredLevel) {
        UILayer.instance.showToast(`声望等级 Lv.${requiredLevel} 解锁此地图`);
        return;
    }
    // 正常进入地图
    MapScene.instance.enter(zoneId);
}
```
