## 一、主界面整体布局

### 1.1 屏幕分区（竖屏 375×812）

```
┌──────────────────────────────┐
│  顶部状态栏                    │ ← 账号等级、经验条、货币
├──────────────────────────────┤
│                              │
│                              │
│     主展示区域                 │ ← 看板娘 + 活动入口
│     （看板娘立绘）              │
```

## 三、主展示区域（看板娘）

### 3.1 看板娘系统

**定义**：主界面中央展示的融合姬立绘，玩家可以选择已拥有的任意融合姬作为看板娘。

#### 功能说明

| 功能 | 说明 |
|------|------|
| 更换看板娘 | 长按看板娘或点击设置按钮，打开融合姬选择界面 |
| 互动 | 点击看板娘触发随机语音/动作（每融合姬有3-5条待机语音） |
| 着装显示 | 看板娘显示该融合姬的当前立绘（品质卡框效果） |
| 默认看板娘 | 未选择时，默认显示万根之母·秦若兰（LE-001） |

#### 看板娘选择界面

- 显示所有已拥有的融合姬列表
- 按品质/种族/元素筛选
- 选中后立即替换主界面看板娘
- 不影响战斗队伍配置

### 3.2 活动入口

**位置**：主展示区域右上角

**设计**：
- 常驻入口按钮，始终显示
- 有进行中的活动时，按钮高亮+红点提示+活动倒计时
- 无活动时，按钮置灰显示"暂无活动"
- 点击进入活动列表界面

#### 活动列表界面

| 内容 | 说明 |
|------|------|
| 当前活动 | 进行中的限时活动列表，显示名称、剩余时间、奖励预览 |
| 活动详情 | 点击进入活动专属页面 |
| 历史活动 | 已结束的活动（仅展示，不可参与） |

## 四、玩家信息栏

### 4.1 布局

```
┌──────────────────────────────────┐
│ [头像框]  称号·昵称               │
│  头像                              │
└──────────────────────────────────┘
```

### 4.2 元素说明

| 元素 | 说明 | 交互 |
|------|------|------|
| 主角头像 | 玩家的头像（默认头像或自定义） | 点击弹出个人信息面板 |
| 头像框 | 头像外框装饰，纯展示 | 点击打开头像框选择界面 |
| 称号 | 当前装备的称号文字，显示在昵称前方 | 点击打开称号选择界面 |
| 昵称 | 玩家的昵称 | 点击可修改（每日1次） |

### 4.3 头像框系统

**定位**：纯展示，无属性加成

#### 头像框获取方式

| 来源 | 示例 |
|------|------|
| 成就奖励 | "首次通关全部野外地图"解锁专属头像框 |
| 活动奖励 | 限时活动排名/积分奖励 |
| 等级奖励 | 达到特定等级解锁（如Lv.50、Lv.99） |
| 首充奖励 | 首次充值赠送 |
| 商城购买 | 星币商店限时出售 |

#### 头像框品质

| 品质 | 视觉效果 | 稀有度 |
|------|---------|--------|
| 普通 | 简单金属边框 | 常见 |
| 精良 | 带有元素色光效 | 少见 |
| 史诗 | 动态光效+粒子 | 稀有 |
| 传说 | 全息投影效果+专属动画 | 极稀有 |

### 4.4 称号系统

**定位**：纯展示，无属性加成

#### 称号格式

```
[称号] 昵称
```

#### 称号获取方式

| 来源 | 示例称号 |
|------|---------|
| 成就奖励 | "废土开拓者"、"星舰继承者" |
| 等级奖励 | "Lv.50 幸存者"、"Lv.99 废土传奇" |
| 活动奖励 | "第一届竞技场冠军" |
| 剧情解锁 | "觉醒之人"（通关第一章） |
| 特殊成就 | "全图鉴收集者"（收集全部融合姬） |

#### 称号装备

- 打开称号选择界面，显示所有已获得的称号
- 选择一个称号装备，显示在昵称前方
- 可以选择不装备任何称号

## 六、各功能界面交互流程

### 6.1 队伍编辑流程

```
队伍界面 → 点击"编辑队伍" → 显示已编组队伍（3融合姬+1芯片）
  → 点击空位/已有角色 → 打开融合姬/芯片选择列表
    → 选择角色 → 确认替换 → 返回队伍界面
  → 点击"保存" → 队伍配置更新
  → 点击"推荐编队" → AI根据关卡自动推荐最优队伍
```

### 6.2 探索流程

```
探索界面 → 选择地图 → 显示关卡列表（线性节点）
  → 点击关卡 → 显示关卡详情（敌人、奖励、准入条件）
    → 满足条件 → 点击"出发" → 进入战斗
    → 不满足条件 → 显示缺少条件提示
  → 点击禁区入口 → 显示禁区信息（层数、消耗抗辐射药剂）
    → 确认进入 → 消耗药剂 → 进入禁区第一层
```

### 6.3 抽卡流程

```
商店界面 → 点击"抽卡"标签 → 选择卡池（常驻/限定）
  → 显示卡池内容、概率、保底进度
  → 选择单抽(200源核)或十连(2000源核)
  → 确认消耗 → 播放抽卡动画 → 显示结果
  → 点击"再次抽取"或"返回"
```

## 七、新手引导流程

### 7.1 首次进入游戏

```
开场剧情（前任管理者的日志）→ 创建角色（输入昵称、选择头像）
→ 庇护所主界面（强制引导）
→ 引导1：点击"队伍" → 赠送初始融合姬（N品质×3）+ 初始芯片（N品质×1）
→ 引导2：编辑队伍 → 将3个融合姬和1个芯片编入队伍
→ 引导3：点击"探索" → 进入第一章废弃城区
→ 引导4：完成第1关战斗 → 引导结束
→ 自由探索
```

### 7.2 初始赠送内容

| 内容 | 数量 | 说明 |
|------|------|------|
| N品质融合姬 | 3张 | 随机N卡，确保坦克/输出/辅助各1张 |
| N品质强化芯片 | 1张 | 随机N芯片 |
| 菌丝 | 5000 | 初始基础货币 |
| 源核 | 1000 | 初始稀有代币 |
| 抗辐射药剂 | 3个 | 初始禁区入场道具 |

## 八、与配置表的关联

### 8.1 相关配置表

| 配置表 | 关键字段 | 说明 |
|--------|---------|------|
| items.xlsx | 新增头像框和称号道具 | itemType为"portrait_frame"和"title" |
| achievement.xlsx | 头像框/称号奖励关联 | achievementReward中可包含头像框/称号 |
| levelUp.xlsx | 等级奖励关联 | 特定等级解锁头像框/称号 |

### 8.2 数值策划配表要点

1. 头像框和称号在items表中配置，作为特殊道具类型
2. 称号和头像框的获取条件在对应系统中配置（成就/等级/活动）
3. 看板娘不需要额外配置表，直接引用已拥有的融合姬数据
4. 新手引导赠送的初始融合姬和芯片需要单独配置
