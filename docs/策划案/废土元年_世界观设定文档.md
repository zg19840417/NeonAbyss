# 废土元年 — 世界观设定文档 v2.0

> **游戏名**：废土元年（Wasteland Year One）
> **技术栈**：Phaser 3 + JavaScript + Vite
> **版本日期**：2026-04-06

---

## 一、核心世界观

### 1.1 时间线

```
核战争爆发 → 文明毁灭 → 生物变异 → 融合技术研发 → 融合现象出现 → 新秩序建立 → 玩家登场
```

### 1.2 核心设定

- 核战争摧毁了人类文明，世界变成废土
- 核辐射导致动植物和机械发生变异
- 战前军方秘密研发"生物融合技术"，试图创造能在核废土中生存的超级士兵
- 融合技术由前任管理者（首席工程师）主导研发
- 核战后，残存人类通过融合技术与变异生物/植物/机械进行"融合"
- 融合成功 → "融合姬"（保留人性，获得超凡力量）
- 融合失败 → "失心者"（失去理性，变成怪物）
- 融合姬是少数，庇护所是少数幸存者的避难所
- 野外是失心者和变异生物的地盘

### 1.3 庇护所的真相（核心剧透）

- 庇护所本身就是一艘可以太空飞行的星舰，由前任管理者秘密设计建造
- 主角（继承者）一开始并不知道这个真相
- 前任管理者在探索废土的过程中失踪，留下了一本设计图/日志
- 星舰缺少很多关键零件，这些零件被变异生物/失心者吞噬后嵌入了它们体内
- 每张地图的最终Boss体内都藏有一个星舰零件
- 玩家需要按照设计图的指引，逐一前往每张地图，击败Boss，取出零件
- 集齐所有零件后，启动星舰，带领幸存者和融合姬离开地球

### 1.4 阵营划分

**我方：融合姬（保留人性）**

- 通过与生物/植物/机械融合获得力量
- 全部为女性（因基因相容性差异，详见融合姬设定）
- 保留了人类的理性和情感
- 建立庇护所作为幸存者避难所
- 派遣队伍探索野外和禁区

**敌方：**

| 类型 | 说明 | 出现区域 |
|------|------|---------|
| 变异生物 | 辐射导致的动植物变异体，无融合能力 | 野外（地面层） |
| 失心者 | 融合失败体，拥有强大力量但失去理性 | 野外+禁区，精英/Boss级 |

**最终Boss**：终焉之核·奥米茄

- 融合技术的创造者，前任管理者的导师/合作者
- 世界上唯一一个"完美融合体"——既拥有失心者般强大的力量，又保持着完整的人类理性
- 他认为融合是人类的进化方向，要将所有幸存者都转化为完美融合体
- 主角的选择：留下接受进化 vs 离开保持人性——理念对决

## 二、融合姬设定

### 2.1 融合技术的起源

核战前，军方秘密研发"生物融合技术"，试图通过将人类与变异生物/植物/机械进行基因层面的融合，创造出能在核废土中生存的超级士兵。这项技术由前任管理者（首席工程师）主导研发。

### 2.2 为什么都是女性

融合技术有一个关键限制——**基因相容性**。核辐射对人类基因造成了普遍损伤，但女性的X染色体在辐射环境下表现出了更强的基因稳定性和修复能力。在融合实验中，女性受试者的融合成功率是男性的数十倍，且融合后能更好地保持理性。

军方最终将融合对象锁定为女性，并给她们取了一个代号——**"融合姬"**（取"融合"的"融"与代表女性的"姬"）。

### 2.3 融合原理

1. **提取**：从变异生物/植物/机械中提取"融合因子"（一种被辐射激活的特殊蛋白质）
2. **注入**：将融合因子注入女性受试者体内
3. **融合**：融合因子与受试者的基因结合，使其获得融合对象的能力（元素控制、物理强化等）
4. **结果**：
   - **成功** → 融合姬（保留人性+获得超凡力量），外观保留人形，融合对象特征以纹路/光环等形式体现
   - **失败** → 失心者（力量暴走+失去理性），外观严重扭曲

### 2.4 男性融合体的命运

- **普通男性**：核战中大量死亡，幸存者因辐射导致的基因损伤而逐渐丧失生育能力，数量锐减。到游戏时间线，男性已极为罕见，散落在各个幸存者聚落中
- **男性融合体**：核辐射对男性Y染色体造成不可逆损伤，几乎所有男性融合体都在融合后短时间内失去理智，变为失心者。由于男性天生更强的物理素质，男性失心者往往比女性失心者更加强大和暴虐

### 2.5 融合姬在故事中的作用

- 是玩家探索废土的核心战力
- 每个融合姬都有独特的融合背景故事
- 随着主线推进，玩家逐渐了解融合技术的全貌和伦理困境
- 部分融合姬的剧情与失心者有关联

## 三、失心者设定

### 3.1 核心悲剧

每一个失心者都曾经是一个有名字、有家人、有梦想的人。融合失败不是他们的错——是融合技术的不完善和核辐射的残酷造成的。

### 3.2 失心者的三个阶段

1. **挣扎期**：刚融合失败后，还残留部分人类意识，能感受到痛苦和恐惧，甚至能说出破碎的话语。这个阶段最悲剧——他们知道自己正在失去理智，但无法阻止
2. **暴走期**：人类意识基本消失，被本能和融合因子的力量驱动，变得极具攻击性。身体开始扭曲变异
3. **固化期**：完全失去人性，形态固化为某种扭曲的形态。这个阶段已无法被"拯救"，只能被击败

### 3.3 击败失心者 = "解放"

游戏中的"净化"概念不是杀死，而是"解放"——让失心者从痛苦中解脱。融合姬们知道失心者的真相，这让她们在战斗中承受着巨大的心理压力。

## 四、种族体系

### 4.1 概述

种族按融合对象分类，共 5 大种族，每个种族下辖 6 个子种族，共 30 个子种族。

子种族不显示在卡片上，以天赋技能的形式存在（可为主动或被动）。

### 4.2 种族列表

| 种族 | 融合对象 | 子种族（6个） |
|------|---------|-------------|
| 植物系 | 变异植物 | 菌类、花类、藤蔓、树木、仙人掌、水草 |
| 动物系 | 变异动物 | 猫科、犬科、鸟类、鱼类、爬行、昆虫 |
| 机械系 | 残存机械 | 战车、无人机、医疗、工业、军用、民用 |
| 能量系 | 辐射能量体 | 火焰、寒冰、雷电、光能、暗能、辐射 |
| 混合系 | 多种融合 | 动植混合、机植混合、机动混合、能量混合、三系混合、混沌 |

### 4.3 子种族天赋示例

| 子种族 | 天赋名称 | 类型 | 效果 |
|--------|---------|------|------|
| 猫科 | 血爪裂伤 | 被动 | 攻击时30%概率使敌人流血，每回合损失攻击力10%的血量，持续2回合 |
| 鱼类 | 水生适应 | 被动 | 在水属性战场中，行动速度增加20% |
| 无人机 | 远程协助 | 主动 | 每3回合可对任意敌人发射一次追踪导弹，造成攻击力50%的伤害 |

## 五、元素体系

### 5.1 五种元素

水、火、风、光、暗。每个融合姬/敌人固定1种元素属性。所有攻击都遵循卡片本身的元素属性，不区分物理/魔法分类。

### 5.2 克制关系

```
  水 ←克→ 火 ←克→ 风
  ↑                   ↓
  克                 克
  ↑                   ↓
  暗 ←互克→ 光
```

- 三角克制：水克火、火克风、风克水
- 互相克制：光克暗、暗克光

### 5.3 克制效果

- 攻击克制方：造成的最终伤害 +20%
- 被克制方攻击：受到的最终伤害 -20%
- 克制效果在所有伤害计算完成后应用（包括暴击、技能、芯片效果）

## 七、关卡架构

### 7.1 整体框架

- **10张野外地图**，每张10个战斗关卡，纯线性推进
- **20个禁区**（每张地图第5和第10节点各延伸1个），层数递增
- 野外不可重复挑战，禁区可重置后重新挑战（需消耗抗辐射药剂）

### 7.2 野外关卡规则

- 每关固定3个敌人，配置表驱动
- 通关条件：消灭全部敌人
- 失败无惩罚，可重试
- 奖励：经验+货币+首通奖励
- 准入条件：声望等级 + 通关前置关卡
- 视觉呈现：线性路径节点式

### 7.3 禁区规则

- 分支选路，每层2-3个节点卡片，完全显示信息
- 每层有且仅有一个事件（战斗/剧情/交易/随机）
- 事件可固定配置、事件池随机、或完全随机
- 有Boss配置的禁区，其最后一层固定为Boss战（失心者）
- 通关=打到底，失败=任一战斗输（已通过层数奖励保留）
- 奖励配置在每个事件中，含稀有奖励（融合姬卡、芯片、材料）
- 入场需消耗**抗辐射药剂**，重置后再进入需再次消耗
- 抗辐射药剂获取：商店用源核购买 + 每1小时自动免费获得1个

## 八、声望体系

- 声望 = 账号等级，范围1-99级
- 通过打怪、任务等方式获得声望经验值升级
- 每个关卡都有准入等级设定，需同时满足准入等级+通关前置关卡
- 主界面（庇护所）展示当前等级和经验条
- 等级越高，解锁越多内容（新地图、新融合姬、新芯片、新系统）

---

## 九、基地设定（庇护所）

- 庇护所 = 幸存者避难所 = 星舰指挥舱（主角初期不知情）
- 玩家是庇护所管理者（继承者），不参与战斗
- 主界面就是庇护所界面，展示等级、经验条等功能
- 设施：工作台、休眠舱、合成台、训练场、黑客终端（待详细设计）
- 核心循环：抽卡招募融合姬 → 组建队伍 → 探索野外/禁区 → 收集资源 → 强化队伍

## 十、职业体系

职业从原来的多个职业缩减为 **3种基础职业**：坦克、输出、辅助。

每种职业需要设计基础天赋技能（待详细设计）。

融合姬卡片同时拥有：元素属性 + 职业 + 种族 三个维度属性。
游戏计划有100+张融合姬卡。

## 十一、卡组系统

### 11.1 融合姬卡

- 每支队伍携带 3 张融合姬卡
- 所有融合姬均为女性
- 可升级（提升等级属性）
- 品质体系：N/R/SR/SSR/UR/LE（六级品质，品质=星级合二为一）
- 固定 1 种元素属性（水/火/风/光/暗）
- 固定 1 种职业（坦克/输出/辅助）
- 固定 1 种种族（植物系/动物系/机械系/能量系/混合系）
- 拥有子种族天赋技能（不显示在卡片上）
- 招募方式：源核抽卡（单抽/十连抽），需要配置卡池和抽取概率

### 11.2 强化芯片

- 每支队伍固定装备 1 张强化芯片
- 品质即星级，合二为一：N/R/SR/SSR/UR/LE
- 基础属性：生命% + 攻击%（每个芯片独立配置，Excel表驱动）
- 效果作用于全队 3 个融合姬，每人获得全额加成
- 全局技能数量：N=1, R=1, SR=1, SSR=2, UR=2, LE=3
- 技能类型：光环被动 / 触发型被动 / 改变现有技能
- 合成升级：3张同品质任意芯片 → 1张高一品质随机芯片
- 定向插件（可选）：职业插件 / 元素插件 / 种族插件

## 十二、主线剧情（10章）

### 第一章：废弃城区 — "继承者的觉醒"
- 开场：主角继承庇护所，发现前任管理者留下的日志和设计图碎片
- 探索：在废弃城区中学习战斗和生存，遇到第一批融合姬
- 小高潮：击败废弃矿坑Boss（锈蚀守卫·陈岚），获得**动力核心**
- 章末揭示：日志中提到"庇护所的真相远比你想象的伟大"，暗示星舰的存在
- 剧情事件：遇到一位老融合姬，成为向导角色

### 第二章：荒野公路 — "公路上的同行者"
- 主线：沿着荒野公路前进，寻找设计图中标注的下一个零件位置
- 支线：遇到其他幸存者聚落，了解废土现状和各方势力
- 小高潮：击败地下水脉Boss（深渊潮汐·赵琳），获得**导航模块**
- 章末揭示：导航模块启动后，显示了其他零件的大致方位
- 剧情事件：有幸存者加入庇护所，基地开始有其他NPC

### 第三章：污染森林 — "变异的源头"
- 主线：进入变异森林，发现植物变异异常剧烈
- 探索：深入了解融合现象的本质——为什么有些人成功有些人失败
- 小高潮：击败枯树之心Boss（枯树之心·苏薇），获得**生命维持装置**
- 章末揭示：在巨树体内发现前任管理者的随身物品，确认他还活着
- 剧情事件：揭示"失心者"的悲剧本质——他们曾经也是有人性的融合者

### 第四章：干涸河床 — "沉没的真相"
- 主线：前往干涸河床，设计图显示水下有战前军事设施
- 探索：发现战前政府隐瞒的真相——核战并非意外
- 小高潮：击败深层水道Boss，获得**护盾发生器**

### 14.3 文案风格要求

- **统一风格**：废土科幻 + 生物融合，禁止使用奇幻/神话风格词汇
- **禁止词汇**：神、神圣、魔法、永恒、世界树、战神、刺客公会、天网等
- **替换方向**：神圣→净化/源初，魔法→异能/能量，永恒→不灭/不朽，战神→铁血/战将
- **敌人type字段**：统一为"变异生物"(mutant)和"失心者"(lost)两类
- **技能描述**：不区分物理/魔法攻击，所有攻击遵循卡片元素属性
