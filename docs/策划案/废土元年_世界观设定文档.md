# 废土元年 — 世界观设定文档 v1.0

> **游戏名**：废土元年（Wasteland Year One）
> **技术栈**：Phaser 3 + JavaScript + Vite
> **版本日期**：2026-04-06

---

## 一、核心世界观

### 1.1 时间线

```
核战争爆发 → 文明毁灭 → 生物变异 → 融合现象出现 → 新秩序建立 → 玩家登场
```

### 1.2 核心设定

- 核战争摧毁了人类文明，世界变成废土
- 核辐射导致动植物和机械发生变异
- 残存人类发现可以与变异生物/植物/机械进行"融合"
- 融合成功 → "融合者"（保留人性，获得超凡力量）
- 融合失败 → "失心者"（失去理性，变成怪物）
- 融合者是少数，基地是少数幸存者的避难所
- 野外是失心者和变异生物的地盘

### 1.3 阵营划分

**我方：融合者（保留人性）**

- 通过与生物/植物/机械融合获得力量
- 保留了人类的理性和情感
- 建立基地作为幸存者避难所
- 派遣队伍探索野外和地下禁区

**敌方：**

| 类型 | 说明 | 出现区域 |
|------|------|---------|
| 变异生物 | 辐射导致的动植物变异体，无融合能力 | 野外（地面层） |
| 失心者 | 融合失败体，拥有强大力量但失去理性 | 野外+禁区，精英/Boss级 |

**最终Boss**：终焉之神·奥米茄

- 远古生物 + 幕后人类融合的终极产物
- 既保持人类理性，又拥有超强失心者能力

## 二、种族体系

### 2.1 概述

种族按融合对象分类，共 5 大种族，每个种族下辖 6 个子种族，共 30 个子种族。

子种族不显示在卡片上，以天赋技能的形式存在（可为主动或被动）。

### 2.2 种族列表

| 种族 | 融合对象 | 子种族（6个） |
|------|---------|-------------|
| 植物系 | 变异植物 | 菌类、花类、藤蔓、树木、仙人掌、水草 |
| 动物系 | 变异动物 | 猫科、犬科、鸟类、鱼类、爬行、昆虫 |
| 机械系 | 残存机械 | 战车、无人机、医疗、工业、军用、民用 |
| 能量系 | 辐射能量体 | 火焰、寒冰、雷电、光能、暗能、辐射 |
| 混合系 | 多种融合 | 动植混合、机植混合、机动混合、能量混合、三系混合、混沌 |

### 2.3 子种族天赋示例

| 子种族 | 天赋名称 | 类型 | 效果 |
|--------|---------|------|------|
| 猫科 | 血爪裂伤 | 被动 | 攻击时30%概率使敌人流血，每回合损失攻击力10%的血量，持续2回合 |
| 鱼类 | 水生适应 | 被动 | 在水属性战场中，行动速度增加20% |
| 无人机 | 远程协助 | 主动 | 每3回合可对任意敌人发射一次追踪导弹，造成攻击力50%的伤害 |

## 三、元素体系

### 3.1 五种元素

水、火、风、光、暗。每个随从/敌人固定1种元素属性。

### 3.2 克制关系

```
  水 ←克→ 火 ←克→ 风
  ↑                   ↓
  克                 克
  ↑                   ↓
  暗 ←互克→ 光
```

- 三角克制：水克火、火克风、风克水
- 互相克制：光克暗、暗克光

### 3.3 克制效果

- 攻击克制方：造成的最终伤害 +20%
- 被克制方攻击：受到的最终伤害 -20%
- 克制效果在所有伤害计算完成后应用（包括暴击、技能、芯片效果）

## 四、货币体系（三级）

| 货币 | 定位 | 说明 |
|------|------|------|
| 菌丝 | 基础货币 | 废土中疯长的变异真菌，可再生资源。大产大消，用于日常升级、强化、购买基础物资 |
| 源核 | 稀有代币 | 核战产生的能量结晶，稀有珍贵。用于招募随从、稀有培养、购买高级道具 |
| 星币 | 付费代币 | 人民币购买。用于商城礼包、可替换源核消费 |

## 五、探索体系

### 5.1 地面层（野外）

- 场景：城市废墟、荒野、污染区
- 敌人：低级变异生物 + 少量失心者
- 产出：菌丝、基础材料

### 5.2 地下层（禁区）

- 场景：地下实验室、军事基地、矿坑
- 敌人：大量失心者 + 精英怪 + Boss
- 产出：源核、强化芯片、关键剧情

---

## 六、声望体系

- 声望 = 账号等级，作为玩法养成进度的门槛
- 限制可解锁的区域和内容
- 每提升一定等级有一次性奖励
- 等级越高，解锁越多内容（新区域、新随从、新芯片、新系统）

---

## 七、基地设定

- 基地 = 幸存者避难所，玩家是基地管理者
- 设施：充电桩、休眠舱、合成台、训练场、黑客终端
- 核心循环：招募融合者 → 组建队伍 → 探索野外/禁区 → 收集资源 → 强化队伍

## 九、卡组系统

### 9.1 随从卡

- 每支队伍携带 3 张随从卡
- 可升级（提升等级属性）
- 可升星（提升品质倍率，品质即星级）
- 固定 1 种元素属性
- 拥有子种族天赋技能（不显示在卡片上）

### 9.2 强化芯片

- 每支队伍固定装备 1 张强化芯片
- 品质即星级，合二为一：N/R/SR/SSR/UR/LE
- 基础属性：生命% + 攻击%（每个芯片独立配置，Excel表驱动）
- 效果作用于全队 3 个随从，每人获得全额加成
- 全局技能数量：N=1, R=1, SR=1, SSR=2, UR=2, LE=3
- 技能类型：光环被动 / 触发型被动 / 改变现有技能
- 合成升级：3张同品质任意芯片 → 1张高一品质随机芯片
- 定向插件（可选）：职业插件 / 元素插件 / 种族插件
