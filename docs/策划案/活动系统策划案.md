# 废土元年 — 活动系统策划案

> **版本**：v1.0
> **版本日期**：2026-04-07
> **用途**：定义限时活动系统的完整规则

---

## 一、系统概述

### 1.1 活动系统定位

活动系统是《废土元年》的核心运营驱动力，承担以下功能：

| 功能 | 说明 |
|------|------|
| **拉新与回流** | 通过限时UP活动、节日主题活动吸引新玩家注册和老玩家回归 |
| **付费刺激** | 限定角色UP池、限定礼包促进付费转化 |
| **日活留存** | 双倍资源活动、累计登录活动维持日常活跃 |
| **内容消耗** | 限时挑战活动提供阶段性目标，消耗玩家积累的资源 |
| **社交竞争** | 排行榜活动激发玩家间的竞争欲望 |

### 1.2 活动类型总览

| 活动类型 | 开放频率 | 持续时间 | 核心奖励 | 主要货币消耗 |
|---------|---------|---------|---------|------------|
| 限时UP抽卡活动 | 每月1-2次 | 14天 | UP限定角色、活动积分 | 源核 |
| 双倍资源活动 | 每周固定 | 周六日各1天 | 菌丝/经验/掉落翻倍 | 无 |
| 限时关卡/挑战活动 | 每月1次 | 7天 | 活动积分、排行奖励 | 抗辐射药剂 |
| 累计登录活动 | 不定期 | 7/14/30天 | 阶梯奖励（源核、晶片等） | 无 |
| 节日/主题主题活动 | 按节日安排 | 7-14天 | 限定头像框、称号、角色碎片 | 星币/源核 |

## 二、活动类型详细设计

### 2.1 限时UP抽卡活动

#### 2.1.1 基本规则

| 配置项 | 数值 |
|--------|------|
| 活动周期 | 14天（含预告期2天） |
| UP角色品质 | SSR或SR |
| UP概率提升 | 从同品质总概率中分出（详见抽卡策划案） |
| 抽卡货币 | 源核（单抽200/十连2000） |
| 保底共享 | 与常驻池共享保底计数器 |

#### 2.1.2 活动专属机制

- **活动积分**：在UP池中每消耗1源核获得1点活动积分（十连2000源核=2000积分）
- **积分兑换**：活动积分可在活动商店中兑换限定道具
- **UP角色碎片**：活动商店中可兑换UP角色的碎片（每次活动限兑换50个碎片）
- **活动限定礼包**：星币商店中上架限定UP礼包（含源核+UP角色碎片）

#### 2.1.3 活动流程

```
预告期（2天）→ 正式开放（12天）→ 活动结算（1天）→ UP角色加入常驻池
```

| 阶段 | 时长 | 玩家可见内容 |
|------|------|------------|
| 预告期 | 2天 | 活动预告弹窗、UP角色立绘展示、活动规则说明 |
| 正式开放 | 12天 | UP池开放、活动商店开放、积分累计 |
| 活动结算 | 1天（活动结束后） | 活动商店保留24小时供兑换，积分不可再获取 |
| 归入常驻 | 结算后 | UP角色自动加入常驻池，概率恢复基础值 |

#### 2.1.4 双UP活动（特殊形式）

| 配置项 | 数值 |
|--------|------|
| 开放频率 | 每季度1次 |
| UP角色 | 2个同品质角色同时UP |
| UP概率 | 每个UP角色各分出同品质总概率的一半 |
| 示例 | SSR双UP：每个UP角色各1.0%（总计2.0%），非UP SSR 0.5% |

### 2.2 双倍资源活动

#### 2.2.1 基本规则

| 配置项 | 数值 |

#### 2.2.3 特殊双倍活动

### 2.3 限时关卡/挑战活动

#### 2.3.1 基本规则

| 配置项 | 数值 |

| 剩余血量比例 | 积分加成 |
|------------|---------|
| 100%-80% | x1.5 |
| 80%-50% | x1.2 |
| 50%-20% | x1.0 |
| 20%-0% | x0.8 |

### 3.1 活动积分系统

#### 3.1.1 积分获取途径

| 获取方式 | 积分产出 | 说明 |
|---------|---------|------|
| UP池抽卡 | 1积分/源核 | 每消耗1源核获得1积分 |
| 活动关卡（普通） | 100积分/次 | 基础积分 |
| 活动关卡（困难） | 200积分/次 | 基础积分 |
| 活动关卡（噩梦） | 400积分/次 | 基础积分 |
| 活动关卡（地狱） | 800积分/次 | 基础积分 |
| 主题活动任务 | 100-800积分/次 | 根据任务难度 |

#### 3.1.2 积分规则

- **积分独立**：每次活动的积分独立计算，不跨活动继承
- **积分有效期**：活动结束后24小时内可继续兑换，之后积分清零
- **积分无上限**：不设单日或总积分上限
- **积分显示**：活动主界面顶部显示当前积分和可兑换状态

### 4.1 活动轮换周期

| 活动类型 | 轮换周期 | 间隔要求 | 说明 |
|---------|---------|---------|------|

### 八、配表执行要点

1. **活动ID命名规范**：`ACT_{类型缩写}_{序号}`，如 `ACT_GACHA_001`、`ACT_CHALLENGE_001`
2. **活动积分货币**：每次活动独立一种积分，通过 `activityId` 关联区分
3. **时间字段**：统一使用 ISO 8601 格式（如 `2026-04-10T00:00:00+08:00`）
4. **限定道具**：头像框和称号在 items 表中注册，通过 `portrait_frame` 和 `title` 类型区分
5. **排行榜奖励**：称号和头像框通过 `titleId` 和 `frameId` 字段关联，为空表示不发
6. **活动商店**：`purchaseLimit` 为 0 表示无限购买，正整数表示限购次数
7. **活动关卡**：`enemyConfig` 和 `bossId` 引用现有的 enemies 和 bosses 配置表

### 二、ActivityManager 主系统

```typescript
class ActivityManager {
  private static instance: ActivityManager;
  private activities: Map<string, ActivityConfig> = new Map();
  private playerActivityData: Map<string, PlayerActivityData> = new Map();

  static getInstance(): ActivityManager {
    if (!ActivityManager.instance) {
      ActivityManager.instance = new ActivityManager();
    }
    return ActivityManager.instance;
  }

  /** 初始化：加载所有活动配置 */
  init(): void {
    const configs = ConfigManager.getAll<ActivityConfig>('activities');
    for (const config of configs) {
      this.activities.set(config.activityId, config);
    }
    // 加载玩家活动数据
    this.loadPlayerData();
  }

  /** 每帧/每分钟更新活动状态 */
  update(serverTime: Date): void {
    for (const [id, config] of this.activities) {
      const newStatus = this.calculateStatus(config, serverTime);
      if (newStatus !== config.status) {
        config.status = newStatus;
        EventBus.emit('activity:status_changed', {
          activityId: id,
          oldStatus: config.status,
          newStatus,
        });
      }
    }
  }

  /** 根据当前时间计算活动状态 */
  private calculateStatus(config: ActivityConfig, now: Date): ActivityStatus {
    const preview = new Date(config.previewTime);
    const start = new Date(config.startTime);
    const end = new Date(config.endTime);
    const shopEnd = new Date(config.shopEndTime);

    if (now < preview) return ActivityStatus.PREVIEW;
    if (now < start) return ActivityStatus.PREVIEW;
    if (now <= end) return ActivityStatus.ACTIVE;
    if (now <= shopEnd) return ActivityStatus.SETTLED;
    return ActivityStatus.ENDED;
  }

  /** 获取当前所有活跃活动 */
  getActiveActivities(): ActivityConfig[] {
    return Array.from(this.activities.values())
      .filter(a => a.status === ActivityStatus.ACTIVE || a.status === ActivityStatus.SETTLED);
  }

  /** 获取指定活动配置 */
  getActivity(activityId: string): ActivityConfig | undefined {
```

```typescript
    const config = ConfigManager.get<ActivityShopItem>('activityShop', shopItemId);

      UIManager.showToast('活动积分不足');
      return false;
    }

    // 检查购买次数限制
    const purchased = playerData.shopPurchases[shopItemId] || 0;
    if (config.purchaseLimit > 0 && purchased >= config.purchaseLimit) {
      UIManager.showToast('已达到购买上限');
      return false;
    }

    // 扣除积分
    manager.addActivityPoints(activityId, -config.cost);

    // 发放奖励
    RewardManager.grantItem(config.itemId, 1);

    // 更新购买记录
    playerData.shopPurchases[shopItemId] = purchased + 1;

    EventBus.emit('activity_shop:purchase', {
      activityId,
      shopItemId,
      remaining: config.purchaseLimit > 0 ? config.purchaseLimit - purchased - 1 : -1,
    });

    return true;
  }
```

### 四、活动挑战关卡系统

```typescript
class ActivityChallengeSystem {
  private static readonly MAX_DAILY_CHALLENGES = 5;
  private static readonly MAX_EXTRA_PURCHASES = 3;
  private static readonly EXTRA_COST = 50; // 星币

  /** 进入活动关卡 */
  enterStage(activityId: string, stageId: string): boolean {
    const manager = ActivityManager.getInstance();
    const stageConfig = ConfigManager.get<ActivityStageConfig>('activityStages', stageId);
    if (!stageConfig || stageConfig.activityId !== activityId) return false;

    const playerData = manager.getPlayerData(activityId);

    // 检查今日挑战次数
    const totalChallenges = playerData.challengeCount + playerData.extraChallengePurchased;
    if (totalChallenges >= this.MAX_DAILY_CHALLENGES + this.MAX_EXTRA_PURCHASES) {
      UIManager.showToast('今日挑战次数已用完');
      return false;
    }

    // 检查入场消耗
    for (const cost of stageConfig.entryCost) {
      if (!InventoryManager.hasItem(cost.id || cost.type, cost.amount || 0)) {
        UIManager.showToast('入场道具不足');
        return false;
      }
    }

    // 扣除入场消耗
    for (const cost of stageConfig.entryCost) {
      InventoryManager.consumeItem(cost.id || cost.type, cost.amount || 0);
    }

    // 增加挑战次数
    if (playerData.challengeCount < this.MAX_DAILY_CHALLENGES) {
      playerData.challengeCount++;
    } else {
      playerData.extraChallengePurchased++;
    }

    // 进入战斗
    BattleManager.startActivityBattle(stageConfig);
    return true;
  }

  /** 战斗结算 */
  onBattleResult(activityId: string, stageId: string, result: BattleResult): void {
    const manager = ActivityManager.getInstance();
    const stageConfig = ConfigManager.get<ActivityStageConfig>('activityStages', stageId);
    if (!stageConfig) return;

    const playerData = manager.getPlayerData(activityId);

    // 初始化关卡记录
    if (!playerData.stageClearRecords[stageId]) {
      playerData.stageClearRecords[stageId] = {
        cleared: false,
        bestScore: 0,
        firstCleared: false,
      };
    }

    const record = playerData.stageClearRecords[stageId];

    if (result.victory) {
      record.cleared = true;

      // 计算积分
      const hpRatio = result.remainingHpRatio; // 0.0 - 1.0
      const hpMultiplier = hpRatio >= 0.8 ? 1.5 : hpRatio >= 0.5 ? 1.2 : hpRatio >= 0.2 ? 1.0 : 0.8;
      const finalScore = Math.floor(
        stageConfig.baseScore * stageConfig.scoreMultiplier * hpMultiplier
      );

      // 更新最高分
      if (finalScore > record.bestScore) {
        record.bestScore = finalScore;
      }

      // 发放积分
      manager.addActivityPoints(activityId, finalScore);

      // 首次通关奖励
      if (!record.firstCleared) {
        record.firstCleared = true;
        for (const reward of stageConfig.firstClearReward) {
          RewardManager.grant(reward);
        }
      }

      EventBus.emit('activity_challenge:victory', {
        activityId,
        stageId,
        score: finalScore,
        bestScore: record.bestScore,
        isFirstClear: record.firstCleared,
      });
    } else {
      EventBus.emit('activity_challenge:defeat', {
        activityId,
        stageId,
      });
    }

    manager.savePlayerData(activityId);
  }

  /** 购买额外挑战次数 */
  purchaseExtraChallenge(activityId: string): boolean {
    const manager = ActivityManager.getInstance();
    const playerData = manager.getPlayerData(activityId);

    if (playerData.extraChallengePurchased >= this.MAX_EXTRA_PURCHASES) {
      UIManager.showToast('今日额外次数已买完');
      return false;
    }

    if (!CurrencyManager.hasEnough('starCoin', this.EXTRA_COST)) {
      UIManager.showToast('星币不足');
      return false;
    }

    CurrencyManager.deduct('starCoin', this.EXTRA_COST);
    playerData.extraChallengePurchased++;

    EventBus.emit('activity_challenge:extra_purchased', {
      activityId,
      remaining: this.MAX_EXTRA_PURCHASES - playerData.extraChallengePurchased,
    });

    return true;
  }

  /** 每日重置挑战次数（05:00刷新） */
  resetDailyChallenges(): void {
    for (const [activityId] of ActivityManager.getInstance().playerActivityData) {
      const data = ActivityManager.getInstance().getPlayerData(activityId);
      data.challengeCount = 0;
      data.extraChallengePurchased = 0;
    }
  }
}

interface BattleResult {
  victory: boolean;
  remainingHpRatio: number;  // 0.0 - 1.0
  turnsUsed: number;
}
```

```typescript
class ActivityLoginSystem {
  /** 检查并记录登录 */
  checkLogin(activityId: string): void {
    const manager = ActivityManager.getInstance();
    const activity = manager.getActivity(activityId);
    if (!activity || activity.activityType !== ActivityType.LOGIN) return;

    const playerData = manager.getPlayerData(activityId);
    const today = this.getTodayIndex(activity);

    // 检查今天是否已记录
    if (!playerData.loginDays.includes(today)) {
      playerData.loginDays.push(today);
      manager.savePlayerData(activityId);

      // 弹出登录奖励领取提示
      UIManager.showLoginRewardPopup(activityId, today);
    }
  }

  /** 领取登录奖励 */
  claimLoginReward(activityId: string, day: number): boolean {
    const manager = ActivityManager.getInstance();
    const playerData = manager.getPlayerData(activityId);

    // 检查是否已登录该天数
    if (!playerData.loginDays.includes(day)) {
      UIManager.showToast('尚未登录该天数');
      return false;
    }

    // 检查是否已领取
    if (playerData.claimedLoginDays.includes(day)) {
      UIManager.showToast('奖励已领取');
      return false;
    }

    // 获取奖励配置
    const loginConfig = ConfigManager.getFirstByFields<ActivityLoginConfig>(
      'activityLogin',
      { activityId, day }
    );
    if (!loginConfig) return false;

    // 发放奖励
    for (const reward of loginConfig.rewards) {
      RewardManager.grant(reward);
    }

    playerData.claimedLoginDays.push(day);
    manager.savePlayerData(activityId);

    EventBus.emit('activity_login:reward_claimed', {
      activityId,
      day,
      rewards: loginConfig.rewards,
    });

    return true;
  }

  /** 获取登录天数在活动中的索引（第几天） */
  private getTodayIndex(activity: ActivityConfig): number {
    const start = new Date(activity.startTime);
    const now = new Date();
    const diffDays = Math.floor(
      (now.getTime() - start.getTime()) / (1000 * 60 * 60 * 24)
    );
    return diffDays + 1; // 从第1天开始
  }

  /** 获取登录奖励列表（含领取状态） */
  getLoginRewards(activityId: string): ActivityLoginDisplay[] {
    const configs = ConfigManager.getAllByField<ActivityLoginConfig>(
      'activityLogin', 'activityId', activityId
    );
    const playerData = ActivityManager.getInstance().getPlayerData(activityId);

    return configs
      .sort((a, b) => a.day - b.day)
      .map(config => ({
        ...config,
        loggedIn: playerData.loginDays.includes(config.day),
        claimed: playerData.claimedLoginDays.includes(config.day),
        claimable: playerData.loginDays.includes(config.day) &&
          !playerData.claimedLoginDays.includes(config.day),
      }));
  }
```

```typescript
class ActivityQuestSystem {
  /** 初始化活动任务 */
  initActivityQuests(activityId: string): void {
    const manager = ActivityManager.getInstance();
    const quests = ConfigManager.getAllByField<ActivityQuestConfig>(
      'activityQuests', 'activityId', activityId
    );
    const playerData = manager.getPlayerData(activityId);

    for (const quest of quests) {
      if (!playerData.questProgress[quest.questId]) {
        playerData.questProgress[quest.questId] = {
          progress: 0,
          completed: false,
          rewarded: false,
        };
      }
    }
  }

  /** 处理游戏事件，更新活动任务进度 */
  onGameEvent(eventName: string, data: any): void {
    const activities = ActivityManager.getInstance().getActiveActivities();

    for (const activity of activities) {
      const quests = ConfigManager.getAllByField<ActivityQuestConfig>(
        'activityQuests', 'activityId', activity.activityId
      );
      const playerData = ActivityManager.getInstance().getPlayerData(activity.activityId);

      for (const quest of quests) {
        const progress = playerData.questProgress[quest.questId];
        if (!progress || progress.completed) continue;

        if (this.matchCondition(quest.condition, eventName)) {
          progress.progress += this.getProgressIncrement(quest.condition, data);

          if (progress.progress >= quest.targetCount) {
            progress.progress = quest.targetCount;
            progress.completed = true;

            // 发放任务奖励（活动积分）
            for (const reward of quest.rewards) {
              if (reward.type === 'activity_point') {
                ActivityManager.getInstance().addActivityPoints(
                  activity.activityId, reward.amount || 0
                );
              } else {
                RewardManager.grant(reward);
              }
            }

            EventBus.emit('activity_quest:completed', {
              activityId: activity.activityId,
              questId: quest.questId,
            });
          }

          EventBus.emit('activity_quest:progress', {
            activityId: activity.activityId,
            questId: quest.questId,
            progress: progress.progress,
            targetCount: quest.targetCount,
          });
        }
      }
    }
  }

  /** 条件匹配映射 */
  private matchCondition(condition: string, eventName: string): boolean {
    const mapping: Record<string, string[]> = {
      'complete_activity_stage': ['battle:activity_win'],
      'clear_activity_hard': ['battle:activity_hard_win'],
      'gacha_pull_limited': ['gacha:limited_pull'],
      'share_activity': ['activity:share'],
      'collect_activity_item': ['inventory:activity_item_add'],
      'kill_activity_boss': ['battle:activity_boss_kill'],
    };
    return mapping[condition]?.includes(eventName) || false;
  }

  /** 进度增量 */
  private getProgressIncrement(condition: string, data: any): number {
    switch (condition) {
      case 'gacha_pull_limited':
        return data.pullCount || 1;
      case 'collect_activity_item':
        return data.itemCount || 1;
      default:
        return 1;
    }
  }
}
```
