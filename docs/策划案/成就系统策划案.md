---

## 一、系统概述

### 1.1 定义

成就是永久性的里程碑目标，达成后可领取一次性奖励。成就分为多个类别，覆盖游戏各个系统的进度。

### 1.2 入口

主界面 → 底部导航"任务" → "成就"标签页

### 1.3 核心规则

- 成就**永久有效**，不会重置
- 达成后可领取**一次性奖励**
- 奖励未领取时，成就图标显示红点提示
- 成就进度实时更新，可在成就面板查看当前进度

## 四、成就奖励类型

### 4.1 奖励内容

| 奖励类型 | 说明 | 配置方式 |
|---------|------|---------|
| 菌丝 | 基础货币 | 直接发放 |
| 源核 | 稀有代币 | 直接发放 |
| 称号 | 纯展示称号 | 解锁称号，可在个人信息中装备 |
| 头像框 | 纯展示头像框 | 解锁头像框，可在个人信息中装备 |
| 专属头像框 | 稀有头像框 | 带有特殊动画效果的头像框 |
| LE称号 | 传说级称号 | 金色特殊称号 |

### 4.2 奖励配置结构

每个成就的奖励为JSON数组，可包含多个奖励：

```json
// 示例：单奖励
{"rewards": [{"type": "mycelium", "amount": 500}]}

// 示例：多奖励
{"rewards": [
  {"type": "sourceCore", "amount": 2000},
  {"type": "title", "id": "TITLE_EXP_012", "name": "废土开拓者"},
  {"type": "portrait_frame", "id": "FRAME_EXP_012", "name": "废土开拓者框"}
]}
```

## 五、与配置表的关联

### 5.1 achievement.xlsx 表结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| achievementId | string | 成就唯一ID |
| category | string | 分类（explore/combat/collection/growth/social/special） |
| name | string | 成就名称 |
| description | string | 成就描述（达成条件文字） |
| condition | string | 达成条件（程序判定用） |
| conditionValue | int | 条件数值（如"通关10关"中的10） |
| rewards | json | 奖励列表 |
| titleReward | string | 称号奖励ID（无则为空） |
| frameReward | string | 头像框奖励ID（无则为空） |
| isHidden | bool | 是否为隐藏成就 |
| sortOrder | int | 同分类内排序 |

### 5.2 数值策划配表要点

1. 每个成就一条记录
2. `condition`字段使用程序可判定的条件标识
3. `rewards`为JSON数组，支持多种奖励类型
4. 隐藏成就的`isHidden`为true，未达成前不显示条件
5. 成就排序按分类+sortOrder排列

#### 1.2 进度追踪系统

```javascript
class AchievementSystem {
  constructor() {
    this.achievements = {}; // achievementId -> { progress, completed, rewarded }
  }

  // 事件触发时检查所有相关成就
  onEvent(eventName, data) {
    const relatedAchievements = this.getAchievementsByCondition(eventName);
    for (const ach of relatedAchievements) {
      this.checkAndUpdateProgress(ach, data);
    }
  }

  checkAndUpdateProgress(achievementId, data) {
    const achConfig = AchievementConfig.get(achievementId);
    const state = this.achievements[achievementId] || { progress: 0, completed: false, rewarded: false };

    if (state.completed) return;

    // 更新进度
    state.progress = this.calculateProgress(achConfig.condition, data);

    // 检查是否达成
    if (state.progress >= achConfig.conditionValue) {
      state.completed = true;
      state.progress = achConfig.conditionValue;
      EventBus.emit('achievement:completed', { id: achievementId });
    }

    this.achievements[achievementId] = state;
    this.save();
  }

  // 领取奖励
  claimReward(achievementId) {
    const state = this.achievements[achievementId];
    const config = AchievementConfig.get(achievementId);

    if (!state || !state.completed || state.rewarded) return false;

    // 发放奖励
    for (const reward of config.rewards) {
      RewardManager.grant(reward);
    }
    if (config.titleReward) {
      unlockTitle(config.titleReward);
    }
    if (config.frameReward) {
      unlockPortraitFrame(config.frameReward);
    }

    state.rewarded = true;
    EventBus.emit('achievement:reward_claimed', { id: achievementId });
    this.save();
    return true;
  }
}
```
