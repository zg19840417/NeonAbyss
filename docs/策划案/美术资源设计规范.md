# 废土元年 — 美术资源设计规范

> **版本**：v2.0
> **版本日期**：2026-04-06
> **用途**：指导美术团队进行所有视觉资源的设计与制作

---

## 一、整体美术风格

### 1.1 风格定义

**超写实 + 韩漫粗线条**

- 角色立绘采用韩漫风格（参考《Solo Leveling》《Tower of God》），线条粗犷有力，轮廓清晰
- 角色渲染追求超写实质感，皮肤纹理、材质细节、光影效果要精细
- 整体氛围偏暗色调，废土末世感强烈，但角色本身要有视觉冲击力和吸引力
- 融合姬（己方角色）在废土环境中要有"希望感"和"生命力"，与灰暗环境形成对比

### 1.2 色彩基调

| 场景 | 色调 | 说明 |
|------|------|------|
| 废土环境 | 灰褐、锈红、暗黄 | 荒芜、破败、辐射感 |
| 庇护所内部 | 冷蓝、金属银 | 科技感、安全感、冰冷秩序 |
| 融合姬角色 | 各种族主题色 + 肤色偏白 | 与环境形成鲜明对比 |
| 失心者/Boss | 暗红、紫黑、病态绿 | 扭曲、危险、悲剧感 |
| UI界面 | 深灰底 + 霓虹绿/蓝点缀 | 废土科技风，不使用鲜艳色 |

### 1.3 线条规范

- **外轮廓线**：粗实线（3-5px），颜色为深褐色或黑色，确保角色在复杂背景下可辨识
- **内部结构线**：中等粗细（1-2px），用于表现服装褶皱、肌肉结构、机械接缝
- **细节装饰线**：细线（0.5-1px），用于纹身、电路纹路、植物脉络等装饰性元素
- **发光元素**：融合特征（纹路、眼睛、能量体）使用柔和发光效果，不画实线轮廓

### 1.4 通用禁止事项

- ❌ 禁止日系萌系/Q版风格
- ❌ 禁止奇幻/中世纪/神话风格元素（翅膀、光环、圣光、魔法阵）
- ❌ 禁止过度暴露的服装设计（废土生存环境需要防护性着装）
- ❌ 禁止纯写实照片风格（保留韩漫的线条和风格化特征）
- ❌ 禁止明亮欢快的色彩基调

## 二、融合姬美术规范

### 2.1 设计核心原则

融合姬 = 人类女性 + 变异生物/植物/机械/能量 的融合体

设计要点：
- **基底**：人类女性，面部保留美感，肤色偏白或带轻微变异色调
- **融合特征**：至少3处明显的融合痕迹（头发、眼睛、皮肤纹路、肢体改造等）
- **服饰**：废土生存风格，有防护功能感（皮革、金属、织物混搭），不是时装
- **武器/道具**：与融合对象相关的特色武器，体现种族特征
- **气质**：坚韧、危险但美丽，不是柔弱或可爱的感觉

### 2.2 立绘规格

| 项目 | 规格 |
|------|------|
| 画幅 | 竖屏 2:3（如 600×900px） |
| 构图 | 全身像，角色占画面70%以上 |
| 背景 | 简化废土场景，虚化处理，不抢角色视觉焦点 |
| 出拳 | 角色面部和上半身需在安全区内（适配不同UI裁切） |
| 姿势 | 每个角色姿势不同，体现职业和性格特征 |

### 2.3 五种族融合姬通用方向

#### 植物系

| 项目 | 说明 |
|------|------|
| 融合特征 | 头发带植物元素（藤蔓编发、花朵装饰、叶片发饰），皮肤有淡绿色脉络纹路，肩部/手臂有苔藓或藤蔓生长 |
| 配色基调 | 翠绿、嫩绿、枯黄、褐色 |
| 服饰方向 | 天然材料感（编织草、树皮纤维）+ 少量废土皮革，有"自然侵蚀文明"的感觉 |
| 武器特征 | 枯木法杖、花苞弓、藤蔓鞭、种子炸弹等植物衍生武器 |
| 气质关键词 | 生机、治愈、危险的美、缓慢侵蚀 |
| 代表元素 | 发光孢子、绽放花朵、缠绕藤蔓、腐朽与新生并存 |

#### 动物系

| 项目 | 说明 |
|------|------|
| 融合特征 | 动物耳朵/尾巴/翅膀等特征融入人体，瞳孔为动物型（竖瞳/横瞳），指甲变为爪，部分皮肤有鳞片/毛发 |
| 配色基调 | 按具体动物子种族（猫科-暖橙、犬科-灰白、鸟类-天蓝、爬行-墨绿、昆虫-暗紫、鱼类-银蓝） |
| 服饰方向 | 轻便灵活，皮革护甲为主，有动物皮毛装饰，行动方便的战斗服 |
| 武器特征 | 爪刃、骨制武器、兽牙项链（装饰+武器）、动物造型盾牌 |
| 气质关键词 | 野性、敏捷、直觉、领地意识 |
| 代表元素 | 兽耳/兽尾、利爪、兽瞳、动物纹路皮肤 |

#### 机械系

| 项目 | 说明 |
|------|------|
| 融合特征 | 肢体部分被机械替换（机械臂、机械腿），皮肤有电路纹路/接口，眼睛有机械光圈，头发中有金属丝 |
| 配色基调 | 金属银、钢铁灰、锈橙、霓虹蓝（能量线） |
| 服饰方向 | 工业风，金属板甲+废土布料拼接，有铆钉、齿轮、管道等机械元素装饰 |
| 武器特征 | 能量枪、机械臂炮、电磁刀、无人机辅助武器 |
| 气质关键词 | 精密、冰冷、力量、人机边界模糊 |
| 代表元素 | 机械关节、电路纹路、全息投影、蒸汽/能量排气 |

#### 能量系

| 项目 | 说明 |
|------|------|
| 融合特征 | 身体周围有可见的能量场/光晕，头发由能量构成或飘浮，皮肤有能量裂纹（像碎裂的瓷器中透出光），眼睛纯发光 |
| 配色基调 | 按具体能量子种族（火焰-赤红/橙、寒冰-冰蓝/白、雷电-金黄/紫、光能-金色/白、暗能-深紫/黑、辐射-病态绿） |
| 服饰方向 | 轻薄飘逸，有能量编织的布料感，部分衣物由能量构成（半透明发光），不是实体布料 |
| 武器特征 | 纯能量武器（光剑、能量球、闪电链），无实体武器 |
| 气质关键词 | 超凡、不稳定、危险、超越人类 |
| 代表元素 | 能量裂纹、光晕、飘浮物、能量粒子 |

#### 混合系

| 项目 | 说明 |
|------|------|
| 融合特征 | 同时具备2-3种种族的融合特征，设计上要有"拼接感"但不违和，体现融合的复杂性 |
| 配色基调 | 多色混搭，但要有主色调统一，避免花哨 |
| 服饰方向 | 混合多种风格元素，如机械外骨骼+植物藤蔓缠绕、动物皮毛+能量护盾 |
| 武器特征 | 复合型武器，如"藤蔓缠绕的机械臂炮""能量驱动的骨制巨剑" |
| 气质关键词 | 混沌、不可预测、进化的终极形态 |
| 代表元素 | 多种族特征的有机融合、不稳定的能量波动、拼接但协调的设计 |

### 2.4 品质视觉差异

同一角色不同品质的视觉差异主要体现在**特效和光效**上，基础立绘不变：

| 品质 | 立绘特效 | 卡框风格 |
|------|---------|---------|
| N | 无特效，基础立绘 | 灰色铁皮质感，无装饰 |
| R | 轻微光效（融合特征微发光） | 铜色边框，简单纹路 |
| SR | 融合特征明显发光，背景有轻微粒子 | 银色边框，种族图标浮雕 |
| SSR | 全身光效，背景有动态粒子流，融合特征强烈发光 | 金色边框，宝石镶嵌，种族图标发光 |
| UR | 华丽光效，背景有能量风暴，角色有能量羽翼/光环 | 铂金边框，复杂浮雕+动态光效 |
| LE | 顶级特效，背景有专属场景，角色有专属变身/觉醒形态展示 | 彩虹渐变边框，全息投影效果，专属动态卡面 |

### 2.5 职业视觉暗示

| 职业 | 体型特征 | 姿势倾向 | 装甲量 |
|------|---------|---------|--------|
| 坦克 | 偏壮实/高大，肩宽 | 稳重、防御姿态，武器常为盾或重型 | 重装甲，护具覆盖面积大 |
| 输出 | 偏修长/灵活，肌肉线条明显 | 攻击姿态，武器常为刀/枪/法杖 | 中等装甲，兼顾灵活 |
| 辅助 | 偏纤细/柔美，气质温和 | 支援姿态，常持有辅助道具或法器 | 轻装甲，注重机动性 |

## 三、10个Boss美术设计

### 3.1 Boss设计核心原则

所有Boss都是**失心者**——曾经是人类女性（除第10个Boss奥米茄为男性），融合失败后失去理智。

设计要点：
- **人形基底**：能看出曾经是人的痕迹（面部轮廓、体型），但已被严重扭曲
- **融合特征暴走**：融合特征过度生长/膨胀，覆盖大部分人体，体现"失控"感
- **悲剧美学**：不是纯粹的怪物，要有"曾经是美好的人"的暗示（残留的首饰、半边完好的脸、破烂的制服碎片）
- **压迫感**：体型比融合姬大1.5-2倍，姿态具有攻击性和威慑力
- **主题色**：每个Boss有专属主题色，与所在地图环境呼应

### 3.2 Boss立绘规格

| 项目 | 规格 |
|------|------|
| 画幅 | 横屏 16:9（如 1920×1080px），用于Boss战展示 |
| 构图 | 上半身至全身，Boss居中偏上，背景为战斗场景 |
| 背景 | Boss所在禁区的核心区域，体现环境特征 |
| 特效 | 专属能量特效，体现Boss的融合特征和力量 |

### 3.3 10个Boss详细设计

#### Boss 1：锈蚀守卫·陈岚（废弃矿坑Boss）

| 项目 | 描述 |
|------|------|
| 融合前身份 | 庇护所前哨守卫 |
| 融合对象 | 矿石/金属变异体 |
| 体型 | 高大壮硕（1.8倍人形），右臂极度膨胀为金属矿石臂 |
| 融合特征 | 全身覆盖铁锈色矿石甲壳，左半脸保留人类特征但布满裂纹，右眼为矿石结晶体发光。右臂完全矿石化，拳头巨大，表面有锈蚀和结晶 |
| 服饰残留 | 破碎的守卫制服碎片挂在身上，胸前的守卫徽章还在（锈迹斑斑） |
| 主题色 | 铁锈红、矿石灰、暗金（结晶发光） |
| 气质 | 沉默、固执、无意识的守护——她还在"守卫"着什么，但已经忘了是什么 |
| 关键视觉 | 锈蚀的守卫徽章、矿石结晶发光的右眼、沉重的矿石臂 |

#### Boss 2：暴走兽王·林瑶（地下水脉Boss）

| 项目 | 描述 |
|------|------|
| 融合前身份 | 商队护卫队长 |
| 融合对象 | 多种变异野兽（以大型猫科为主） |
| 体型 | 修长但充满爆发力（1.7倍人形），四肢带有野兽特征 |
| 融合特征 | 全身覆盖短而硬的兽毛，面部有猫科兽纹，双眼为金色兽瞳。四肢末端变为兽爪，背后有多条尾巴（融合了多种动物）。脊椎隆起，有骨刺外露 |
| 服饰残留 | 破烂的护卫队队长外套，背后的商队标志被撕裂 |
| 主题色 | 暗金、深棕、血红（暴走状态） |
| 气质 | 狂暴、悲伤、不可控——她的眼神中偶尔闪过清明，随即被兽性吞噬 |
| 关键视觉 | 多条尾巴、金色兽瞳中的泪痕、撕裂的商队标志 |

#### Boss 3：枯树之心·苏薇（枯树之心Boss）

| 项目 | 描述 |
|------|------|
| 融合前身份 | 植物学研究员 |
| 融合对象 | 变异巨树/菌丝网络 |
| 体型 | 下半身与巨树融为一体，上半身保持人形但被藤蔓缠绕（2倍人形，含树体） |
| 融合特征 | 下半身化为巨大的枯树根系，蔓延整个画面。上半身皮肤木质化，有树皮纹理。头发变为枯枝和藤蔓，手指变为细长的树枝。胸口有发光的"树心"——一颗脉动的绿色晶体 |
| 服饰残留 | 破碎的白大褂挂在树枝上，口袋里还有研究笔记 |
| 主题色 | 枯褐、苔绿、荧光绿（树心） |
| 气质 | 静谧、哀伤、缓慢——她不再移动，像一棵真正的树一样"生长"在那里 |
| 关键视觉 | 胸口发光的树心、挂在树枝上的白大褂和笔记、枯枝构成的手指 |

#### Boss 4：深渊潮汐·赵琳（深层水道Boss）

| 项目 | 描述 |
|------|------|
| 融合前身份 | 水下探测员 |
| 融合对象 | 深海水系变异生物群 |
| 体型 | 飘浮在水中，下半身化为流体水态（1.6倍人形，含水态延伸） |
| 融合特征 | 全身覆盖半透明的水膜，皮肤下有蓝色光点流动（模拟生物发光）。头发变为流动的水流，眼睛为深蓝色无瞳。双臂延伸为水母触手状，周围有小型水系生物环绕 |
| 服饰残留 | 撕碎的潜水服残片，头盔还挂在脖子上（面罩碎裂） |
| 主题色 | 深海蓝、生物荧光蓝、水白 |
| 气质 | 空灵、迷茫、群体意识——她的表情像在做梦，被水中的"声音"引导 |
| 关键视觉 | 流动的头发、皮肤下的蓝色光点、碎裂的潜水头盔、环绕的水母触手 |

#### Boss 5：铁血指挥官·周颖（前哨指挥部Boss）

| 项目 | 描述 |
|------|------|
| 融合前身份 | 军事前哨指挥官 |
| 融合对象 | 军事武器系统/战斗强化药剂 |
| 体型 | 精壮有力（1.8倍人形），全身被军用装甲覆盖 |
| 融合特征 | 全身植入军用级机械增强装置，脊椎外露为金属柱。面部半边为战术面甲，另半边为扭曲的人脸。双臂为重型武器臂（一臂炮管，一臂能量刃）。背后有多个武器挂架 |
| 服饰残留 | 军用指挥官制服被改造为战斗装甲的一部分，肩章和军衔还在 |
| 主题色 | 军绿、钢铁灰、火红（武器能量） |
| 气质 | 威严、偏执、军事化——她的每个动作都像在执行命令 |
| 关键视觉 | 半机械面甲、肩章军衔、脊椎金属柱、武器臂 |

#### Boss 6：幻境之主·白露（生命之泉Boss）

| 项目 | 描述 |
|------|------|
| 融合前身份 | 绿洲守护者 |
| 融合对象 | 绿洲核心能量体 |
| 体型 | 飘浮在空中，周围有花瓣和光粒子环绕（1.5倍人形） |
| 融合特征 | 全身半透明，像由水晶和光构成。皮肤有彩虹折射效果，头发为流动的光带。背后有巨大的能量翅膀（由花瓣和光组成），脚下有莲花状能量台 |
| 服饰残留 | 白色长裙残片在光中飘浮，像破碎的梦 |
| 主题色 | 珍珠白、彩虹折射、柔金（核心能量） |
| 气质 | 美丽但诡异、温柔但危险——她微笑着，但笑容是空洞的 |
| 关键视觉 | 半透明水晶身体、彩虹折射皮肤、破碎的白色长裙、空洞的微笑 |

#### Boss 7：机甲女皇·韩雪（核心军库Boss）

| 项目 | 描述 |
|------|------|
| 融合前身份 | 军方首席机甲工程师 |
| 融合对象 | 最先进军用机甲核心 |
| 体型 | 巨大的机甲体（2倍人形），人体嵌入机甲核心中 |
| 融合特征 | 身体与巨型机甲完全融合，只露出面部和部分躯干。机甲设计为女性形态，有流畅的金属曲线。机甲表面有无数管线与人体连接，人体的皮肤上有电路纹路。双眼为红色机械眼 |
| 服饰残留 | 无——她与机甲完全一体，但机甲的胸部面板内侧贴着一张全家福照片 |
| 主题色 | 金属银、深红（机甲能量）、暗蓝（电路） |
| 气质 | 冰冷、精密、无机质——她已经不认为自己是人类了 |
| 关键视觉 | 巨大的女性机甲体、嵌入核心的人体、红色机械眼、面板内的全家福照片 |

#### Boss 8：方舟守卫·沈梦（方舟核心Boss）

| 项目 | 描述 |
|------|------|

## 四、强化芯片美术规范

### 4.1 设计核心原则

强化芯片是装备在卡组中的科技道具，视觉上应体现"废土科技"风格。

设计要点：
- **形状**：六边形芯片为主体，边缘有电路接口
- **材质**：金属底板 + 发光能量核心 + 半透明防护罩
- **模块化**：芯片上有可识别的"插件槽"，暗示可扩展性
- **品质差异**：通过材质和光效区分品质等级
