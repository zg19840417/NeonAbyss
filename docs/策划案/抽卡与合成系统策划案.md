# 废土元年 — 抽卡与合成系统策划案

> **版本**：v5.1
> **版本日期**：2026-04-08
> **用途**：定义立绘碎片抽取系统的完整规则
> **核心变更**：从抽取融合姬改为抽取立绘碎片

---

## 一、系统概述

### 1.1 获取融合姬的方式

| 获取方式 | 说明 | 初始品质 |
|---------|------|---------|
| **剧情加入** | 水/火/风三系融合姬通过剧情逐步加入 | N品质 |

### 1.2 融合姬养成方式

| 养成方式 | 说明 |
|---------|------|
| **升级** | 消耗菌丝提升融合姬等级 |
| **升品质** | 任意完成1整套立绘碎片后，获得1次升品质资格 |
| **能力解锁** | 通过升品质解锁更多能力槽和技能 |

### 1.3 N品质融合姬获取方式

N品质融合姬**不通过抽卡直接获得**，通过剧情五节点解锁：

#### 难度制融合姬规则（每章3难度×10关）

| 难度 | 触发关卡 | 融合姬 |
|------|---------|--------|
| **普通** | 关卡1/3/5/7/9 | 融合姬A |
| **噩梦** | 关卡1/3/5/7/9 | 融合姬B |
| **地域** | 关卡1/3/5/7/9 | 融合姬C |

#### 五节点触发规则

| 节点 | 触发条件 | 效果 |
|------|---------|------|
| **首次相遇** | **难度第1关** | 触发对话/剧情，了解融合姬存在 |
| **加入卡组** | **难度第3关** | 融合姬正式加入卡组，激活图鉴，可编辑上阵 |
| **赠送R立绘碎片** | **难度第5关** | 赠送R立绘套装（30个R碎片） |
| **赠送SR立绘碎片** | **难度第7关** | 赠送SR立绘套装（6个SR碎片） |
| **赠送SSR立绘碎片** | **难度第9关** | 赠送SSR立绘套装（1个SSR碎片） |

> **说明**：
> - 每章3个难度，每个难度对应1个融合姬
> - 融合姬加入卡组后，购买立绘进入卡池
> - 赠送的立绘套装不进入卡池

## 二、立绘碎片抽取系统

### 2.1 抽取内容变更

| 变更项 | 旧方案 | 新方案 |
|--------|--------|--------|
| **抽取内容** | 融合姬本身 | **绑定到指定角色与指定立绘套装的立绘碎片** |
| **抽取产出** | 融合姬卡牌 | **1张立绘碎片** |
| **品质提升** | 直接获得高品质融合姬 | **完成任意1整套立绘碎片后获得1次升品质资格** |

### 2.2 卡池类型

#### 当前常驻卡池

- **名称**：立绘召唤
- **开放时间**：始终可用
- **内容**：**当前已登场/已解锁融合姬** 的立绘碎片
- **货币**：源核
- **价格**：单抽200源核，十连2000源核
- **掉落粒度**：角色 + 立绘套装 + 碎片品质

#### 限定UP卡池（预留）

- **名称**：按UP角色命名（如"凌霜·限定立绘召唤"）
- **开放时间**：限时开放（通常2周）
- **内容**：常驻池全部内容 + UP角色碎片概率提升
- **货币**：源核
- **价格**：单抽200源核，十连2000源核
- **UP规则**：
  - UP角色为SR碎片时：该SR碎片出现概率提升
  - UP角色为SSR碎片时：该SSR碎片出现概率提升
  - UP角色为UR碎片时：该UR碎片出现概率提升

### 2.3 碎片概率

| 碎片品质 | 单抽概率 |
|---------|---------|
| R碎片 | 60% |
| SR碎片 | 25% |
| SSR碎片 | 10% |
| UR碎片 | 5% |
| **合计** | **100%** |

### 2.4 保底机制

| 保底类型 | 触发条件 | 效果 |
|---------|---------|------|
| SR保底 | 每10抽未出SR，第10抽必出SR碎片 | 硬保底，计数器独立 |
| SSR保底 | 每40抽未出SSR，第40抽必出SSR碎片 | 硬保底，计数器独立 |
| UR保底 | 每200抽未出UR，第200抽必出UR碎片 | 硬保底，计数器独立 |

**保底规则说明：**
- 三种保底计数器**互相独立**，互不影响
- 出现对应品质或更高品质时，计数器**重置为0**
- 保底在**常驻池和限定池之间共享计数器**
- 新服/新玩家首次十连**必出1张SR碎片**

### 2.5 限定UP概率示例

以SSR角色UP为例：

| 碎片品质 | 常驻概率 | UP池概率 |
|---------|---------|---------|
| R碎片 | 60% | 60% |
| SR碎片 | 25% | 25% |
| SSR碎片（非UP） | 10% | 3% |
| SSR碎片（UP角色） | — | 7% |
| UR碎片 | 5% | 5% |

## 三、碎片处理规则

### 3.1 碎片绑定

| 规则 | 说明 |
|------|------|
| **绑定机制** | 碎片绑定到指定融合姬 + 指定立绘套装 |
| **使用限制** | 只能用于该融合姬对应立绘套装的收集与成长 |
| **属性成长** | 碎片在未满需求数量前会永久提供属性成长 |
| **溢出定义** | 达到该碎片需求数量后，再抽到同类碎片 = 溢出 |

### 3.2 品质提升规则

| 当前品质 | 目标品质 | 条件 |
|---------|---------|------|
| N | R | 任意完成1整套立绘碎片 |
| R | SR | 再完成任意1整套立绘碎片 |
| SR | SSR | 再完成任意1整套立绘碎片 |
| SSR | UR | 再完成任意1整套立绘碎片 |
| UR | LE | 再完成任意1整套立绘碎片 |

### 3.3 溢出碎片处理

| 项目 | 说明 |
|------|------|
| **溢出定义** | 达到该碎片需求数量后，再抽到同类碎片 |
| **溢出处理** | 自动转换为对应元素点数 |
| **点数类型** | 水点 / 火点 / 风点 |
| **兑换用途** | 在对应元素商店兑换商品 |

### 3.4 元素点商店

| 商店类型 | 使用点数 | 说明 |
|------|------|------|
| 水点商店 | 水点 | 兑换水系相关商品 |
| 火点商店 | 火点 | 兑换火系相关商品 |
| 风点商店 | 风点 | 兑换风系相关商品 |

### 3.5 碎片属性成长

| 项目 | 说明 |
|------|------|
| **成长时机** | 抽到碎片后立即生效 |
| **成长上限** | 达到该碎片需求数量后停止继续加属性 |
| **成长类型** | 生命 / 攻击 / 速度 / 全属性 |
| **数值来源** | 每个融合姬、每套立绘、每个碎片单独配表 |

## 五、品质解锁对应

### 5.1 碎片品质与解锁内容

| 完成次数 | 提升至该品质 | 解锁内容 |
|---------|-------------|---------|
| **第1次完成套装** | R品质 | 额外2个能力槽（5个），随机生成1~2个能力 |
| **第2次完成套装** | SR品质 | 第3个主动技能 |
| **第3次完成套装** | SSR品质 | 额外2个能力槽（7个），仅解锁 |
| **第4次完成套装** | UR品质 | 宠物栏位 |
| **第5次完成套装** | LE品质 | 额外2个能力槽（9个），仅解锁 |

### 5.2 能力槽详情

| 品质 | 能力槽总数 | 增加 | 随机生成 |
|------|-----------|------|---------|
| N | 3个 | - | 0个（3个通用能力） |
| R | 5个 | +2 | 1~2个 |
| SR | 5个 | 0 | 1~2个 |
| SSR | 7个 | +2 | 0个 |
| UR | 7个 | 0 | 0个 |

## 六、改造系统（模组）

### 6.1 模组简介

模组是用于填充能力槽的道具，每个模组对应一个能力。

**核心规则：**
- **一对一映射**：一个模组 = 一个能力
- **无品质限制**：模组可用于任意品质的融合姬
- **填充槽位**：将模组装备到空置的能力槽位

### 6.2 模组获取途径

| 模组类型 | 获取途径 |
|---------|---------|
| **基础模组** | 野外关卡掉落 |
| **高级模组** | 禁区Boss掉落 |
| **稀有模组** | 活动奖励 |
| **活动模组** | 限时活动商店 |

### 6.3 模组装备规则

| 项目 | 说明 |
|------|------|
| **装备数量** | 最多9个（LE品质） |
| **装备位置** | 空置的能力槽位 |
| **替换规则** | 可替换，替换后旧模组消失 |
| **升级** | 模组可升级（消耗同名模组） |

### 6.4 模组装备流程

```
玩家进入改造界面
    │
    ├─ 选择要改造的融合姬
    │     └─ 显示当前能力槽状态（已填/空置）
    │
    ├─ 选择要装备的模组
    │     └─ 显示模组对应的能力信息
    │
    ├─ 选择空置的槽位
    │     └─ 将模组装备到槽位
    │
    └─ 确认装备
          └─ 模组进入槽位，能力生效
```

## 七、系统流程图

### 7.1 抽取循环

```
玩家进行抽取
    │
    ├─ 获得立绘碎片
    │     │
    │     ├─ 未达到需求数量 → 积累进度 + 提供属性成长
    │     │
    │     └─ 达到需求数量后再获得 → 转换为元素点数
    │
    └─ 任意完成1整套立绘碎片
          │
          └─ 获得1次升品质资格
```

### 7.2 养成路径

```
剧情加入（N品质）
    ↓
抽取已登场融合姬的立绘碎片
    ↓
任意完成1整套立绘 → 获得1次升品质资格
    ↓
手动确认品质预览 → 升1档
    ↓
重复上述流程，直到LE
```
