# 废土元年 — 抽卡与合成系统策划案

> **版本**：v2.0
> **版本日期**：2026-04-06
> **用途**：定义抽卡招募和合成系统的完整规则

---

## 一、系统概述

### 1.1 获取融合姬的两种方式

| 方式 | 说明 | 产出品质 |
|------|------|---------|
| 源核抽卡 | 消耗源核在卡池中抽取 | R / SR / SSR / UR |
| 合成系统 | 消耗同品质卡牌+菌丝+可选改造晶片 | N→R→SR→SSR→UR→LE |

### 1.2 N卡获取方式

N卡**不在卡池中出现**，通过以下方式获取：
- 野外关卡首次通关奖励
- 禁区事件奖励
- 每日任务奖励
- 成就奖励
- 新手引导赠送

## 二、抽卡系统

### 2.1 卡池类型

#### 常驻卡池

- **名称**：融合召唤
- **开放时间**：始终可用
- **内容**：包含所有非LE品质的融合姬（R/SR/SSR/UR）
- **货币**：源核
- **价格**：单抽200源核，十连2000源核
- **限定角色**：限定UP结束后，该角色自动加入常驻池

#### 限定UP卡池

- **名称**：按UP角色命名（如"锈叶·苏蔓 限定召唤"）
- **开放时间**：限时开放（通常2周）
- **内容**：常驻池全部内容 + UP角色概率提升
- **货币**：源核
- **价格**：单抽200源核，十连2000源核
- **UP规则**：
  - UP角色为SR品质时：该SR角色出现概率从基础概率提升至UP概率
  - UP角色为SSR品质时：该SSR角色出现概率从基础概率提升至UP概率
  - UP角色为UR品质时：该UR角色出现概率从基础概率提升至UP概率
- **UP结束后**：UP角色自动加入常驻池，原概率恢复

### 2.2 基础概率（常驻池）

| 品质 | 单抽概率 | 说明 |
|------|---------|------|
| R | 85% | 基础品质 |
| SR | 12% | 精良品质 |
| SSR | 2.5% | 史诗品质 |
| UR | 0.5% | 传说品质 |
| **合计** | **100%** | |

### 2.3 保底机制

| 保底类型 | 触发条件 | 效果 |
|---------|---------|------|
| SR保底 | 每10抽未出SR，第10抽必出SR | 硬保底，计数器独立 |
| SSR保底 | 每40抽未出SSR，第40抽必出SSR | 硬保底，计数器独立 |
| UR保底 | 每200抽未出UR，第200抽必出UR | 硬保底，计数器独立 |

**保底规则说明：**
- 三种保底计数器**互相独立**，互不影响
- 出现对应品质或更高品质时，计数器**重置为0**
- 例如：第8抽出了SSR，则SR计数器和SSR计数器都重置，UR计数器继续累计
- 保底在**常驻池和限定池之间共享计数器**（切换卡池不重置）
- 新服/新玩家首次十连**必出1张SR**

### 2.4 限定UP概率（示例）

以SSR角色UP为例：

| 品质 | 常驻概率 | UP池概率 | 说明 |
|------|---------|---------|------|
| R | 85% | 85% | 不变 |
| SR | 12% | 12% | 不变 |
| SSR（非UP） | 2.5% | 0.5% | 分摊给UP角色 |
| SSR（UP角色） | — | 2.0% | 从SSR总概率中分出 |
| UR | 0.5% | 0.5% | 不变 |

**注意**：UP角色的额外概率从同品质总概率中分出，不增加总概率。

### 2.5 抽卡动画与演出

| 品质 | 抽卡演出 |
|------|---------|
| R | 白光闪烁，卡牌快速翻转 |
| SR | 蓝色光柱，卡牌缓慢翻转，有粒子特效 |
| SSR | 金色光柱+粒子风暴+屏幕震动，卡牌特写展示 |
| UR | 彩虹光柱+全屏特效+专属BGM片段，卡牌全屏展示 |

### 2.6 重复卡处理

抽到已拥有的融合姬时：
- **重复卡转化为"碎片"**，碎片按品质区分（R碎片/SR碎片/SSR碎片/UR碎片）
- 碎片用途：
  - 在"碎片商店"中兑换指定角色的碎片（集齐N个碎片可解锁对应角色）
  - 兑换其他资源（菌丝、源核等）

| 品质 | 重复卡→碎片数量 | 解锁所需碎片 |
|------|---------------|-------------|
| R | 1个R碎片 | 20个R碎片 |
| SR | 1个SR碎片 | 40个SR碎片 |
| SSR | 1个SSR碎片 | 80个SSR碎片 |
| UR | 1个UR碎片 | 150个UR碎片 |

## 三、合成系统

### 3.1 基础合成规则

**公式**：4张同品质卡 + 菌丝 → 1张高一品质的随机卡

注意：以上合成规则仅适用于融合姬卡。强化芯片的合成规则为：3张同品质任意芯片 → 1张高一品质随机芯片（不消耗菌丝和改造晶片）。

| 合成方向 | 材料卡 | 菌丝消耗 | 产出 |
|---------|--------|---------|------|
| N → R | 4张N卡 | 1000菌丝 | 1张随机R卡 |
| R → SR | 4张R卡 | 5000菌丝 | 1张随机SR卡 |
| SR → SSR | 4张SR卡 | 20000菌丝 | 1张随机SSR卡 |
| SSR → UR | 4张SSR卡 | 100000菌丝 | 1张随机UR卡 |
| UR → LE | 4张UR卡 | 500000菌丝 | 1张随机LE卡 |

**规则说明：**
- 4张材料卡**必须为同一品质**，但种族/元素/职业可以不同
- 产出为**随机**高一品质的融合姬（从该品质全部角色池中随机）
- 合成后材料卡**消失**，不可恢复
- 产出角色**可能重复**（已有角色转化为碎片）

### 3.2 定向合成（改造晶片系统）

在基础合成中加入**改造晶片**，可以定向影响合成结果。

#### 改造晶片分类

| 晶片类型 | 对应材料品质 | 效果 | 获取方式 |
|---------|-------------|------|---------|
| 职业改造晶片·R | R级合成 | 合成结果限定为**特定职业**（坦克/输出/辅助三选一） | 关卡掉落、任务奖励 |
| 种族改造晶片·SR | SR级合成 | 合成结果限定为**特定种族**（5种族选1） | 禁区Boss掉落、活动奖励 |
| 元素改造晶片·SR | SR级合成 | 合成结果限定为**特定元素**（5元素选1） | 禁区Boss掉落、活动奖励 |
| 异化改造晶片·SSR | SSR级合成 | 合成结果为**异化角色**（属性强化+可能额外技能） | 高级禁区掉落、限时活动 |

#### 定向合成公式

**R级定向合成：**
```
4张N卡 + 1000菌丝 + 职业改造晶片·R（坦克/输出/辅助）→ 1张指定职业的随机R卡
```

**SR级定向合成：**
```
4张R卡 + 5000菌丝 + 种族改造晶片·SR（指定种族）→ 1张指定种族的随机SR卡
4张R卡 + 5000菌丝 + 元素改造晶片·SR（指定元素）→ 1张指定元素的随机SR卡
```
（不可同时使用种族和元素晶片，二选一）

**SSR级异化合成：**
```
4张SR卡 + 20000菌丝 + 异化改造晶片·SSR → 1张随机异化SSR卡
```

#### 异化角色说明

异化角色是SSR品质的特殊变体，与普通SSR角色的区别：

| 属性 | 普通SSR | 异化SSR |
|------|---------|---------|
| 生命值 | 基础值 | 基础值 × 120% |
| 攻击力 | 基础值 | 基础值 × 120% |
| 技能数量 | 基础技能 | 基础技能 + 可能获得额外技能 |
| 外观 | 普通立绘 | 异化立绘（视觉上有明显变异特征） |
| 卡框 | SSR金色卡框 | 异化专属卡框（暗金+红色纹路） |

**异化额外技能概率：**
- 70%概率：无额外技能（仅属性提升）
- 25%概率：获得1个额外被动技能
- 5%概率：获得1个额外主动技能

### 3.3 改造晶片获取

| 晶片类型 | 获取方式 | 大致产出 |
|---------|---------|---------|
| 职业改造晶片·R | 野外关卡概率掉落 | 每关约5%概率 |
| 种族改造晶片·SR | 禁区事件奖励/Boss首次击杀 | 每个禁区Boss首次击杀必掉1个 |
| 元素改造晶片·SR | 禁区事件奖励/Boss首次击杀 | 每个禁区Boss首次击杀必掉1个 |
| 异化改造晶片·SSR | 高级禁区（地图6+）Boss概率掉落 | 约10%概率 |

## 四、碎片商店

### 4.1 商店内容

玩家可以用重复卡转化来的碎片，在碎片商店中兑换指定角色。

| 兑换项 | 价格 | 说明 |
|--------|------|------|
| 指定R角色 | 20个R碎片 | 可选择任意已有R角色 |
| 指定SR角色 | 40个SR碎片 | 可选择任意已有SR角色 |
| 指定SSR角色 | 80个SSR碎片 | 可选择任意已有SSR角色 |
| 指定UR角色 | 150个UR碎片 | 可选择任意已有UR角色 |
| 职业改造晶片·R | 10个R碎片 | 随机职业 |
| 菌丝 | 1个R碎片 → 500菌丝 | 基础兑换 |
| 源核 | 5个SR碎片 → 100源核 | 高级兑换 |

### 4.2 商店刷新

- 碎片商店**不刷新**，所有角色始终可兑换
- 新获得的融合姬会自动加入碎片商店的可兑换列表

## 五、与配置表的关联

### 5.1 相关配置表

| 配置表 | 关键字段 | 说明 |
|--------|---------|------|
| gachaPools.xlsx | poolId, poolName, poolType, currency, cost, upCharId, upRate, startTime, endTime | 卡池配置 |
| gachaItems.xlsx | poolId, charId, baseRate, upRate | 卡池内各角色概率 |
| items.xlsx | itemId, itemName, itemType, description, obtainMethod | 道具配置（含改造晶片） |
| shop.xlsx | shopId, shopType, itemId, cost, currency, refreshRule | 碎片商店配置 |

### 5.2 数值策划配表要点

1. **gachaPools表**：每个卡池一条记录，常驻池1条，每个限定UP池1条
2. **gachaItems表**：每个卡池中每个角色一条概率记录
3. **改造晶片**在items表中配置，itemType为"material"
4. **碎片商店**在shop表中配置，shopType为"fragment"
5. **合成消耗**的菌丝数量可在后续数值平衡中调整，当前为初始值
