# 废土元年 — 抽卡与合成系统策划案

> **版本**：v5.0
> **版本日期**：2026-04-08
> **用途**：定义立绘碎片抽取系统的完整规则
> **核心变更**：从抽取融合姬改为抽取立绘碎片

---

## 一、系统概述

### 1.1 获取融合姬的方式

| 获取方式 | 说明 | 初始品质 |
|---------|------|---------|
| **剧情加入** | 水/火/风三系融合姬通过剧情逐步加入 | N品质 |

### 1.2 融合姬养成方式

| 养成方式 | 说明 |
|---------|------|
| **升级** | 消耗菌丝提升融合姬等级 |
| **升品质** | 集齐4张同品质立绘碎片，提升品质1档 |
| **能力解锁** | 通过升品质解锁更多能力槽和技能 |

### 1.3 N卡获取方式

N卡**不在卡池中出现**，通过以下方式获取：
- 野外关卡首次通关奖励
- 禁区事件奖励
- 每日任务奖励
- 成就奖励
- 新手引导赠送

## 二、立绘碎片抽取系统

### 2.1 抽取内容变更

| 变更项 | 旧方案 | 新方案 |
|--------|--------|--------|
| **抽取内容** | 融合姬本身 | **立绘碎片** |
| **抽取产出** | 融合姬卡牌 | **1张立绘碎片** |
| **品质提升** | 直接获得高品质融合姬 | **集齐4张碎片提升1档品质** |

### 2.2 卡池类型

#### 常驻卡池

- **名称**：立绘召唤
- **开放时间**：始终可用
- **内容**：所有融合姬的立绘碎片（R/SR/SSR/UR/LE）
- **货币**：源核
- **价格**：单抽200源核，十连2000源核

#### 限定UP卡池

- **名称**：按UP角色命名（如"凌霜·限定立绘召唤"）
- **开放时间**：限时开放（通常2周）
- **内容**：常驻池全部内容 + UP角色碎片概率提升
- **货币**：源核
- **价格**：单抽200源核，十连2000源核
- **UP规则**：
  - UP角色为SR碎片时：该SR碎片出现概率提升
  - UP角色为SSR碎片时：该SSR碎片出现概率提升
  - UP角色为UR碎片时：该UR碎片出现概率提升

### 2.3 碎片概率

| 碎片品质 | 单抽概率 |
|---------|---------|
| R碎片 | 60% |
| SR碎片 | 25% |
| SSR碎片 | 10% |
| UR碎片 | 4% |
| LE碎片 | 1% |
| **合计** | **100%** |

### 2.4 保底机制

| 保底类型 | 触发条件 | 效果 |
|---------|---------|------|
| SR保底 | 每10抽未出SR，第10抽必出SR | 硬保底，计数器独立 |
| SSR保底 | 每40抽未出SSR，第40抽必出SSR | 硬保底，计数器独立 |
| UR保底 | 每200抽未出UR，第200抽必出UR | 硬保底，计数器独立 |

**保底规则说明：**
- 三种保底计数器**互相独立**，互不影响
- 出现对应品质或更高品质时，计数器**重置为0**
- 保底在**常驻池和限定池之间共享计数器**
- 新服/新玩家首次十连**必出1张SR碎片**

### 2.5 限定UP概率示例

以SSR角色UP为例：

| 碎片品质 | 常驻概率 | UP池概率 |
|---------|---------|---------|
| R碎片 | 60% | 60% |
| SR碎片 | 25% | 25% |
| SSR碎片（非UP） | 10% | 3% |
| SSR碎片（UP角色） | — | 7% |
| UR碎片 | 4% | 4% |
| LE碎片 | 1% | 1% |

## 三、碎片处理规则

### 3.1 碎片绑定

| 规则 | 说明 |
|------|------|
| **绑定机制** | 碎片绑定到指定融合姬 |
| **使用限制** | 只能用于该融合姬的品质提升 |
| **溢出处理** | 集齐当前档位碎片后，再抽到同档位碎片 = 溢出 |

### 3.2 品质提升规则

| 当前品质 | 目标品质 | 所需碎片 |
|---------|---------|---------|
| N | R | 4张R碎片 |
| R | SR | 4张SR碎片 |
| SR | SSR | 4张SSR碎片 |
| SSR | UR | 4张UR碎片 |
| UR | LE | 4张LE碎片 |

### 3.3 溢出碎片处理

| 项目 | 说明 |
|------|------|
| **溢出定义** | 集齐当前档位碎片后，再抽到同档位碎片 |
| **溢出处理** | 溢出碎片自动转换为 **晶核** |
| **晶核数量** | 每溢出1个碎片 = 1个晶核 |
| **晶核用途** | 可在商店兑换指定角色的任意档位碎片 |

### 3.4 晶核商店

| 商品 | 价格 | 说明 |
|------|------|------|
| R碎片（指定角色） | 10晶核 | 兑换后直接获得1张R碎片 |
| SR碎片（指定角色） | 20晶核 | 兑换后直接获得1张SR碎片 |
| SSR碎片（指定角色） | 50晶核 | 兑换后直接获得1张SSR碎片 |
| UR碎片（指定角色） | 100晶核 | 兑换后直接获得1张UR碎片 |
| LE碎片（指定角色） | 200晶核 | 兑换后直接获得1张LE碎片 |

## 五、品质解锁对应

### 5.1 碎片品质与解锁内容

| 碎片品质 | 提升至该品质 | 解锁内容 |
|---------|-------------|---------|
| **R碎片** | R品质 | 额外2个能力槽（5个），随机生成1~2个能力 |
| **SR碎片** | SR品质 | 第3个主动技能 |
| **SSR碎片** | SSR品质 | 额外2个能力槽（7个），仅解锁 |
| **UR碎片** | UR品质 | 宠物栏位 |
| **LE碎片** | LE品质 | 额外2个能力槽（9个），仅解锁 |

### 5.2 能力槽详情

| 品质 | 能力槽总数 | 增加 | 随机生成 |
|------|-----------|------|---------|
| N | 3个 | - | 0个（3个通用能力） |
| R | 5个 | +2 | 1~2个 |
| SR | 5个 | 0 | 1~2个 |
| SSR | 7个 | +2 | 0个 |
| UR | 7个 | 0 | 0个 |
| LE | 9个 | +2 | 0个 |

## 六、改造系统（能力芯片）

### 6.1 能力芯片简介

能力芯片是用于填充能力槽的道具，每个芯片对应一个能力。

**核心规则：**
- **一对一映射**：一个芯片 = 一个能力
- **无品质限制**：芯片可用于任意品质的融合姬
- **填充槽位**：将芯片装备到空置的能力槽位

### 6.2 芯片获取途径

| 芯片类型 | 获取途径 |
|---------|---------|
| **基础芯片** | 野外关卡掉落 |
| **高级芯片** | 禁区Boss掉落 |
| **稀有芯片** | 活动奖励 |
| **活动芯片** | 限时活动商店 |

### 6.3 芯片装备规则

| 项目 | 说明 |
|------|------|
| **装备数量** | 最多9个（LE品质） |
| **装备位置** | 空置的能力槽位 |
| **替换规则** | 可替换，替换后旧芯片消失 |
| **升级** | 芯片可升级（消耗同名芯片） |

### 6.4 芯片装备流程

```
玩家进入改造界面
    │
    ├─ 选择要改造的融合姬
    │     └─ 显示当前能力槽状态（已填/空置）
    │
    ├─ 选择要装备的芯片
    │     └─ 显示芯片对应的能力信息
    │
    ├─ 选择空置的槽位
    │     └─ 将芯片装备到槽位
    │
    └─ 确认装备
          └─ 芯片进入槽位，能力生效
```

## 七、系统流程图

### 7.1 抽取循环

```
玩家进行抽取
    │
    ├─ 获得立绘碎片
    │     │
    │     ├─ 碎片档位 > 当前品质 → 积累碎片
    │     │
    │     ├─ 碎片档位 = 当前品质 → 积累碎片
    │     │
    │     └─ 碎片档位 < 当前品质 → 溢出 → 转换为晶核
    │
    └─ 集齐4张同档位碎片
          │
          └─ 品质提升 + 解锁新内容
```

### 7.2 养成路径

```
剧情加入（N品质）
    ↓
抽取R碎片 → 集齐4张 → R品质
    ↓
抽取SR碎片 → 集齐4张 → SR品质
    ↓
抽取SSR碎片 → 集齐4张 → SSR品质
    ↓
抽取UR碎片 → 集齐4张 → UR品质
    ↓
抽取LE碎片 → 集齐4张 → LE品质
    ↓
满品质 = 完全体
```
