## 一、系统概述

### 1.1 设计目的

背包与物品系统是《废土元年》的核心资源管理模块。玩家通过探索、战斗、商店等途径获取各类道具，在背包中进行管理、使用、合成和出售。该系统直接影响玩家的养成进度和资源规划策略。

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **分类清晰** | 物品按类型/品质/功能严格分类，玩家可快速定位目标物品 |
| **容量可扩展** | 初始200格背包，通过扩展道具或星币解锁更多格子 |
| **操作便捷** | 支持批量操作、一键整理、快速使用等便捷功能 |
| **数据安全** | 所有物品变动实时同步服务端，防止数据丢失 |
| **视觉直观** | 通过图标颜色/边框/标签快速区分物品品质和类型 |

### 1.3 系统入口

| 入口位置 | 打开方式 | 说明 |
|---------|---------|------|
| 主界面 → 庇护所 → 背包 | 点击庇护所界面中的背包图标 | 主入口 |
| 战斗结算 → 获得物品 | 点击物品图标 | 查看物品详情 |
| 商店 → 购买成功 | 自动弹出 | 购买后查看 |
| 融合界面 → 选择材料 | 从背包选择 | 材料消耗入口 |

## 二、物品分类体系

### 2.1 物品大类

```
物品系统
├── 货币类 (Currency)          → 不占用背包格子，独立显示
│   ├── 菌丝 (Mycelium)        → 基础货币
│   ├── 源核 (SourceCore)      → 高级货币
│   └── 星币 (StarCoin)        → 充值货币
│
├── 消耗品类 (Consumable)      → 占用背包格子
│   ├── 抗辐射药剂             → 战斗中恢复HP
│   ├── 经验药水               → 融合姬获得经验
│   ├── 技能升级材料           → 升级融合姬技能
│   └── 体力药剂               → 恢复探索体力
│
├── 材料类 (Material)          → 占用背包格子
│   ├── 改造晶片               → 融合姬进阶材料（4种属性）
│   ├── 碎片                   → 融合姬召唤/升星材料（4品质）
│   └── 其他材料               → 活动限定/特殊用途材料
│
├── 卡牌类 (Card)              → 不占用背包格子，独立管理
│   ├── 融合姬卡 (FusionGirl)  → 核心战斗单位
│   └── 强化芯片 (Chip)        → 装备增强单位
│
└── 特殊类 (Special)           → 不占用背包格子
    ├── 头像框                 → 个人资料展示
    └── 称号                   → 角色名旁展示
```

### 2.2 品质等级

| 品质 | 颜色标识 | 边框样式 | 获取难度 | 占比(抽卡) |
|------|---------|---------|---------|-----------|
| N (普通) | 灰色 #9E9E9E | 细线边框 | 极易 | 50% |
| R (稀有) | 绿色 #4CAF50 | 单线边框 | 易 | 25% |
| SR (超稀有) | 蓝色 #2196F3 | 双线边框 | 中 | 15% |
| SSR (极稀有) | 紫色 #9C27B0 | 发光边框 | 难 | 7% |
| UR (传说) | 橙色 #FF9800 | 动态光效 | 极难 | 2.5% |
| LE (限定) | 金色 #FFD700 | 独特效框 | 限时 | 0.5% |

### 2.3 物品ID编码规则

```
格式: [类型前缀][品质码][序号]

类型前缀:
  CUR = 货币类
  CON = 消耗品
  MAT = 材料类
  CRD = 卡牌类
  SPC = 特殊类

品质码:
  0 = N, 1 = R, 2 = SR, 3 = SSR, 4 = UR, 5 = LE

示例:
  CON01001 = N品质消耗品第001号（抗辐射药剂）
  MAT02003 = R品质材料第003号（水属性改造晶片）
  CRD03015 = SR品质卡牌第015号
```

## 三、背包容量规则

### 3.1 容量体系

| 存储区域 | 初始容量 | 最大容量 | 扩展方式 | 单次扩展 |
|---------|---------|---------|---------|---------|
| 消耗品背包 | 200格 | 500格 | 背包扩展道具 / 星币 | +20格 |
| 材料背包 | 200格 | 500格 | 背包扩展道具 / 星币 | +20格 |
| 融合姬卡库 | 无限 | 无限 | - | - |
| 强化芯片库 | 无限 | 无限 | - | - |
| 特殊物品库 | 无限 | 无限 | - | - |

### 3.2 背包扩展方式

| 方式 | 消耗 | 说明 |
|------|------|------|
| 背包扩展道具 | 消耗品×1 | 通过活动/成就获取 |
| 星币扩展 | 100星币/次 | 付费扩展 |
| 等级解锁 | 达到指定等级 | 每10级自动+10格 |

### 3.3 背包满处理

- **获得新物品时背包已满**：弹出提示「背包已满，请清理后再获取」，物品暂存至邮件系统（保留7天）
- **战斗结算时背包已满**：正常结算，超出物品自动存入邮件
- **批量获取时部分满**：已满类型的物品存入邮件，其余正常入包

### 3.4 物品堆叠规则

| 物品类型 | 是否可堆叠 | 最大堆叠数 |
|---------|-----------|-----------|
| 货币类 | 不适用（独立计数） | - |
| 消耗品 | 可堆叠 | 999 |
| 材料类 | 可堆叠 | 999 |
| 卡牌类 | 不可堆叠 | 1 |
| 特殊类 | 不可堆叠 | 1 |

## 四、物品操作

### 4.1 使用操作

| 操作 | 触发方式 | 说明 |
|------|---------|------|
| 单个使用 | 点击物品 → 详情面板 → 【使用】按钮 | 需确认使用 |
| 快速使用 | 长按物品 | 直接使用（仅限消耗品） |
| 批量使用 | 多选 → 【批量使用】 | 支持选择数量 |

**使用限制：**
- 抗辐射药剂：仅可在战斗外使用（战斗内通过技能恢复）
- 经验药水：需选择目标融合姬
- 技能升级材料：需选择目标融合姬 + 目标技能
- 体力药剂：当前体力未满时可用

### 4.2 出售操作

| 操作 | 触发方式 | 说明 |
|------|---------|------|
| 单个出售 | 点击物品 → 详情面板 → 【出售】按钮 | 需确认 |
| 批量出售 | 多选 → 【批量出售】 | 显示总售价 |
| 按品质出售 | 筛选 → 【出售全部N品质】 | 二次确认 |
| 一键出售 | 背包界面 → 【快速出售】 | 按预设规则自动出售 |

**出售价格规则：**

| 物品类型 | 出售价格 | 说明 |
|---------|---------|------|
| 消耗品 | 购入价格的30% | 最低1菌丝 |
| 材料 | 固定价格（按品质） | N=10, R=50, SR=200, SSR=1000 |
| 碎片 | 固定价格（按品质） | N=5, R=25, SR=100, SSR=500 |
| 卡牌 | 不可出售 | - |
| 特殊物品 | 不可出售 | - |

### 4.3 合成操作

**合成入口**：背包界面 → 【合成】按钮

**合成规则：**

| 合成类型 | 材料需求 | 产出 | 成功率 |
|---------|---------|------|--------|
| 低级碎片→高级碎片 | 同品质碎片×10 | 高一品质碎片×1 | 100% |
| 改造晶片合成 | 同属性晶片×3 | 高一级晶片×1 | 100% |
| 抗辐射药剂合成 | 药剂碎片×5 | 抗辐射药剂×1 | 100% |
| 高级经验药水合成 | 低级经验药水×3 | 高级经验药水×1 | 100% |

**合成界面布局：**

```
┌─────────────────────────────────┐
│  ← 合成                    关闭 │
├─────────────────────────────────┤
│                                 │
│   [材料1] [材料2] [材料3]       │
│                                 │
│         ┌───────┐               │
│         │  合成  │               │
│         │  →    │               │
│         └───────┘               │
│                                 │
│   [产出物品预览]                 │
│                                 │
│   成功率: 100%                  │
│   拥有材料: 8/10                │
│                                 │
│   ┌──────────────────────┐      │
│   │      合成 ×1          │      │
│   └──────────────────────┘      │
│                                 │
└─────────────────────────────────┘
```

### 4.4 物品自动整理

- **默认排序**：品质（高→低）→ 类型 → 获取时间（新→旧）
- **手动排序**：支持按类型/品质/数量/获取时间切换排序方式
- **一键整理**：点击整理按钮，按默认规则重新排列所有物品
- **筛选功能**：支持按类型/品质/关键词筛选

### 9.2 核心类实现

```csharp
// InventoryManager.cs
public class InventoryManager : MonoBehaviour
{
    public static InventoryManager Instance { get; private set; }

    // 背包数据: bagType -> (itemId -> count)
    private Dictionary<int, Dictionary<string, int>> bags = new();

    // 背包容量: bagType -> currentSlots
    private Dictionary<int, int> bagCapacity = new();

    /// <summary>
    /// 初始化背包系统
    /// </summary>
    public void Initialize()
    {
        // 加载背包配置
        var config = InventoryConfigTable.GetAll();
        foreach (var bag in config)
        {
            bags[bag.bag_type] = new Dictionary<string, int>();
            bagCapacity[bag.bag_type] = bag.init_slots;
        }

        // 从服务端同步数据
        StartCoroutine(SyncFromServer());
    }

    /// <summary>
    /// 添加物品到背包
    /// </summary>
    /// <returns>实际添加数量（背包满时返回0）</returns>
    public int AddItem(string itemId, int count)
    {
        var itemConfig = ItemConfigTable.Get(itemId);
        if (itemConfig == null) return 0;

        // 卡牌和特殊物品不占格子
        if (itemConfig.item_type == 4 || itemConfig.item_type == 5)
        {
            return AddUnlimitedItem(itemId, count);
        }

        int bagType = itemConfig.item_type == 2 ? 1 : 2; // 消耗品=1, 材料=2
        int maxSlots = bagCapacity[bagType];
        int currentSlots = bags[bagType].Count;

        if (currentSlots >= maxSlots)
        {
            // 背包已满，发送至邮件
            MailSystem.SendItem(itemId, count, "背包已满，物品已转至邮件");
            return 0;
        }

        // 堆叠逻辑
        if (bags[bagType].ContainsKey(itemId))
        {
            int current = bags[bagType][itemId];
            int canAdd = Math.Min(count, itemConfig.max_stack - current);
            bags[bagType][itemId] += canAdd;
            return canAdd;
        }
        else
        {
            // 新物品占用一个格子
            int canAdd = Math.Min(count, itemConfig.max_stack);
            bags[bagType][itemId] = canAdd;
            return canAdd;
        }
    }

    /// <summary>
    /// 移除物品
    /// </summary>
    public bool RemoveItem(string itemId, int count)
    {
        foreach (var bag in bags.Values)
        {
            if (bag.ContainsKey(itemId) && bag[itemId] >= count)
            {
                bag[itemId] -= count;
                if (bag[itemId] <= 0) bag.Remove(itemId);
                return true;
            }
        }
        return false;
    }

    /// <summary>
    /// 使用物品
    /// </summary>
    public bool UseItem(string itemId, int count, string targetId = null)
    {
        var itemConfig = ItemConfigTable.Get(itemId);
        if (itemConfig == null || !itemConfig.is_usable) return false;

        if (!RemoveItem(itemId, count)) return false;

        // 执行物品效果
        var effect = EffectConfigTable.Get(itemConfig.effect_id);
        if (effect != null)
        {
            EffectExecutor.Execute(effect, targetId);
        }

        // 发送埋点
        AnalyticsManager.Track("item_use", new { itemId, count, targetId });

        return true;
    }

    /// <summary>
    /// 出售物品
    /// </summary>
    public bool SellItem(string itemId, int count)
    {
        var itemConfig = ItemConfigTable.Get(itemId);
        if (itemConfig == null || !itemConfig.is_sellable) return false;

        if (!RemoveItem(itemId, count)) return false;

        // 获得菌丝
        int totalGold = itemConfig.sell_price * count;
        CurrencyManager.AddCurrency("CUR00001", totalGold);

        // 发送埋点
        AnalyticsManager.Track("item_sell", new { itemId, count, totalGold });

        return true;
    }

    /// <summary>
    /// 扩展背包
    /// </summary>
    public bool ExpandBag(int bagType, ExpandType expandType)
    {
        var config = InventoryConfigTable.Get(bagType);
        if (bagCapacity[bagType] >= config.max_slots) return false;

        switch (expandType)
        {
            case ExpandType.Item:
                if (!RemoveItem(config.expand_item_id, 1)) return false;
                break;
            case ExpandType.Coin:
                if (!CurrencyManager.ConsumeCurrency("CUR00003", config.expand_coin_cost))
                    return false;
                break;
        }

        bagCapacity[bagType] += config.expand_slots;

        // 发送埋点
        AnalyticsManager.Track("bag_expand", new { bagType, expandType });

        return true;
    }

    /// <summary>
    /// 排序背包
    /// </summary>
    public void SortInventory(int bagType, SortType sortType)
    {
        var bag = bags[bagType];
        var sorted = bag.OrderBy(kvp =>
        {
            var config = ItemConfigTable.Get(kvp.Key);
            return sortType switch
            {
                SortType.Quality => -config.quality, // 品质高→低
                SortType.Type => config.item_sub_type,
                SortType.Count => -kvp.Value,
                SortType.ObtainTime => 0, // 由服务端排序
                _ => 0
            };
        }).ToList();

        bags[bagType] = new Dictionary<string, int>(sorted.ToDictionary(kvp => kvp.Key, kvp => kvp.Value));
    }
}
```

```csharp
    [SerializeField] private GameObject newItemBadge;
    [SerializeField] private GameObject selectedMark;

    private string itemId;
    private int count;
    private bool isSelected;

    /// <summary>
    /// 刷新格子显示
    /// </summary>
    public void Refresh(string itemId, int count, bool isNew = false)
    {
        this.itemId = itemId;
        this.count = count;

        var config = ItemConfigTable.Get(itemId);
        iconImage.sprite = Resources.Load<Sprite>(config.icon_path);
        qualityBorder.color = QualityColors.GetColor(config.quality);
        qualityBorder.material = QualityMaterials.GetMaterial(config.quality);
        countText.text = count > 1 ? count.ToString() : "";
        newItemBadge.SetActive(isNew);
    }

    public void OnPointerClick(PointerEventData eventData)
    {
        if (isSelected)
        {
            // 取消选中
            SetSelected(false);
            InventoryUI.OnCellDeselected(itemId);
        }
        else
        {
            SetSelected(true);
            InventoryUI.OnCellSelected(itemId);
        }
    }

    public void OnPointerDown(PointerEventData eventData)
    {
        // 长按快速使用（仅消耗品）
        if (eventData.duration > 0.5f)
        {
            var config = ItemConfigTable.Get(itemId);
            if (config.item_type == 2 && config.is_usable)
            {
                InventoryManager.Instance.UseItem(itemId, 1);
                Refresh(itemId, count - 1);
            }
        }
    }

    public void SetSelected(bool selected)
    {
        isSelected = selected;
        selectedMark.SetActive(selected);
    }
```

```csharp
// CraftSystem.cs
public class CraftSystem
{
    /// <summary>
    /// 执行合成
    /// </summary>
    public static CraftResult Craft(int craftId)
    {
        var config = CraftConfigTable.Get(craftId);
        if (config == null) return CraftResult.ConfigNotFound;

        // 检查材料是否充足
        foreach (var mat in config.material_list)
        {
            if (!InventoryManager.Instance.HasItem(mat.id, mat.count))
            {
                return CraftResult.MaterialNotEnough;
            }
        }

        // 扣除材料
        foreach (var mat in config.material_list)
        {
            InventoryManager.Instance.RemoveItem(mat.id, mat.count);
        }

        // 成功率判定
        if (Random.Range(0f, 1f) > config.success_rate)
        {
            // 合成失败
            AnalyticsManager.Track("craft_fail", new { craftId });
            return CraftResult.Fail;
        }

        // 发放产物
        InventoryManager.Instance.AddItem(config.product_id, config.product_count);

        // 扣除菌丝
        if (config.coin_cost > 0)
        {
            CurrencyManager.ConsumeCurrency("CUR00001", config.coin_cost);
        }

        AnalyticsManager.Track("craft_success", new { craftId, productId = config.product_id });
        return CraftResult.Success;
    }
}
```

### 9.5 数据同步实现

```csharp
// InventoryDataSync.cs
public static class InventoryDataSync
{
    private const float SYNC_INTERVAL = 30f; // 30秒同步一次
    private static float syncTimer = 0f;
    private static bool isDirty = false;

    /// <summary>
    /// 标记数据已变更
    /// </summary>
    public static void MarkDirty()
    {
        isDirty = true;
    }

    /// <summary>
    /// 每帧检查同步
    /// </summary>
    public static void Update(float deltaTime)
    {
        syncTimer += deltaTime;
        if (isDirty && syncTimer >= SYNC_INTERVAL)
        {
            syncTimer = 0f;
            isDirty = false;
            _ = SyncToServer();
        }
    }

    /// <summary>
    /// 同步到服务端
    /// </summary>
    private static async Task SyncToServer()
    {
        var data = InventoryManager.Instance.GetSnapshot();
        await NetworkManager.PostAsync("inventory/sync", data);
    }

    /// <summary>
    /// 从服务端拉取
    /// </summary>
    public static async Task PullFromServer()
    {
        var response = await NetworkManager.GetAsync("inventory/pull");
        InventoryManager.Instance.ApplySnapshot(response.data);
    }
}
```

### 9.6 关键注意事项

1. **并发安全**：物品添加/移除操作需加锁，防止快速点击导致数据异常
2. **服务端校验**：所有物品变动必须经服务端验证，客户端仅做预展示
3. **延迟处理**：网络延迟时使用乐观更新策略，失败后回滚
4. **内存优化**：物品图标使用SpriteAtlas打包，避免频繁加载
5. **滚动性能**：物品网格使用对象池(ObjectPool)复用ItemCell，保证滑动流畅
6. **离线缓存**：本地使用SQLite缓存背包数据，离线时可查看，联网后同步
7. **防刷检测**：服务端记录物品变动日志，异常变动触发安全审核
