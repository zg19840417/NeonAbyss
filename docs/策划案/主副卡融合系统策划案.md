# 废土元年 — 宠物系统策划案

> **版本**：v1.0
> **版本日期**：2026-04-08
> **用途**：定义宠物系统的完整规则
> **核心说明**：UR品质解锁宠物栏位，宠物替代原有的融合功能

---

## 一、系统概述

### 1.1 宠物系统说明

| 项目 | 说明 |
|------|------|
| **解锁条件** | 融合姬品质达到UR |
| **宠物栏位** | 1个 |
| **宠物来源** | 驯服野外变异生物 |
| **宠物效果** | 提供属性加成（随机生成） |
| **宠物特性** | 不可取下，只能替换 |

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **随机性** | 每次驯服随机生成属性加成 |
| **不可逆** | 宠物绑定后不可解除 |
| **可替换** | 可用新宠物替换旧宠物 |
| **唯一性** | 每个融合姬最多1只宠物 |

---

## 二、宠物获取

### 2.1 驯服条件

| 条件 | 说明 |
|------|------|
| **前置** | 融合姬品质达到UR |
| **战斗** | 在野外战斗中击败变异生物 |
| **概率** | 每次战斗有概率触发驯服选项 |

### 2.2 驯服触发

| 触发方式 | 概率 | 说明 |
|---------|------|------|
| **战斗胜利** | 10% | 击败变异生物后有概率触发 |
| **稀有变异生物** | 30% | 精英/首领级变异生物更容易触发 |
| **活动期间** | 提升 | 特定活动期间驯服概率提升 |

### 2.3 驯服界面

```
驯服成功！
    │
    ├─ 变异生物名称：XXX
    │
    ├─ 随机属性预览：
    │     ├─ 攻击加成：+XX%
    │     ├─ 防御加成：+XX%
    │     ├─ 生命加成：+XX%
    │     └─ 速度加成：+XX%
    │
    ├─ 当前宠物：（如有）
    │     └─ 显示当前宠物属性
    │
    └─ 操作选项：
          ├─ [确认驯服] → 替换当前宠物
          └─ [放弃驯服] → 保留当前宠物
```

## 三、宠物属性

### 3.1 属性类型

| 属性类型 | 效果 | 范围 |
|---------|------|------|
| **攻击加成** | 宠物为融合姬提供攻击力加成 | +5% ~ +20% |
| **防御加成** | 宠物为融合姬提供防御力加成 | +5% ~ +20% |
| **生命加成** | 宠物为融合姬提供生命值加成 | +5% ~ +30% |
| **速度加成** | 宠物为融合姬提供速度加成 | +3% ~ +15% |
| **暴击加成** | 宠物为融合姬提供暴击率加成 | +3% ~ +10% |
| **抵抗加成** | 宠物为融合姬提供异常抵抗加成 | +5% ~ +15% |

### 3.2 属性生成规则

| 规则 | 说明 |
|------|------|
| **随机生成** | 每次驯服随机生成1~3个属性 |
| **数值范围** | 每个属性在对应范围内随机 |
| **属性组合** | 同一宠物可能拥有多种属性 |
| **数值波动** | 存在小范围的数值波动 |

### 3.3 属性示例

| 宠物类型 | 常见属性组合 |
|---------|-------------|
| **攻击型** | 攻击+20%，暴击+8%，速度+5% |
| **防御型** | 防御+18%，生命+25%，抵抗+10% |
| **均衡型** | 攻击+10%，防御+10%，生命+15%，速度+5% |
| **速度型** | 速度+15%，攻击+12%，暴击+6% |

## 四、宠物特性

### 4.1 绑定特性

| 特性 | 说明 |
|------|------|
| **不可取下** | 宠物一旦绑定，不可解除 |
| **永久生效** | 宠物提供的属性加成持续有效 |
| **属性绑定** | 宠物的属性与融合姬绑定 |
| **跨属性共享** | 宠物的属性加成对所有战斗生效 |

### 4.2 替换规则

| 规则 | 说明 |
|------|------|
| **可替换** | 可以用新的宠物替换旧的宠物 |
| **替换消耗** | 替换后旧宠物消失，无需额外消耗 |
| **属性覆盖** | 替换后新宠物的属性完全覆盖旧属性 |
| **无保留** | 旧宠物的属性不会转移到新宠物 |

### 4.3 宠物羁绊

| 羁绊等级 | 条件 | 效果 |
|---------|------|------|
| **Lv.1** | 初始宠物 | 基础属性加成 |
| **Lv.2** | 同类型宠物替换3次 | 属性上限+5% |
| **Lv.3** | 同类型宠物替换5次 | 属性上限+10% |

## 六、宠物系统UI

### 6.1 宠物入口

宠物入口位于融合姬详情页面：

```
融合姬详情页面
    │
    ├─ 基础信息（名称、品质、等级、元素、职业）
    │
    ├─ 技能信息（主动技能列表）
    │
    ├─ 能力信息（能力槽状态）
    │
    ├─ 宠物信息
    │     ├─ [无宠物] → 显示"UR品质解锁宠物栏"
    │     └─ [有宠物] → 显示宠物图标+属性预览
    │
    └─ 装备信息（芯片槽状态）
```

### 6.2 宠物详情页

| 区域 | 内容 |
|------|------|
| **宠物头像** | 显示宠物立绘/图标 |
| **宠物名称** | 显示宠物名称+稀有度 |
| **属性展示** | 显示所有属性加成数值 |
| **羁绊等级** | 显示当前羁绊等级 |
| **操作按钮** | [替换宠物] / [查看图鉴] |

### 6.3 宠物图鉴页

| 功能 | 说明 |
|------|------|
| **已驯服** | 显示已获得的宠物列表 |
| **未驯服** | 显示未获得的宠物（灰色） |
| **属性预览** | 悬停显示可能的属性范围 |
| **驯服记录** | 记录每次驯服的时间和属性 |

## 八、系统流程图

### 8.1 宠物获取流程

```
野外战斗胜利
    │
    ├─ 判定是否触发驯服（10%概率）
    │     │
    │     ├─ 未触发 → 战斗结束
    │     │
    │     └─ 触发 → 显示驯服界面
    │
    └─ 驯服界面
          │
          ├─ 生成随机属性
          │
          ├─ 检查是否有现有宠物
          │     │
          │     ├─ 无宠物 → 直接驯服
          │     │
          │     └─ 有宠物 → 提示替换
          │
          └─ 确认后绑定宠物
                │
                └─ 宠物属性生效
```

### 8.2 宠物替换流程

```
遇到可驯服变异生物
    │
    ├─ 当前已有宠物？
    │     │
    │     ├─ 无 → 直接驯服
    │     │
    │     └─ 有 → 显示对比界面
    │
    └─ 对比界面
          │
          ├─ 显示当前宠物属性
          ├─ 显示新宠物属性
          │
          ├─ [确认替换] → 新宠物替换旧宠物
          └─ [放弃驯服] → 保留旧宠物
```
