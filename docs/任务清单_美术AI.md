### 建议分批绘制

**第一批（N品质12张）**：FM006~FM010, FM014, FM017~FM019, FM021~FM023
- N卡可使用简化流程（较低精度，快速产出）

- R卡中等精度

**第三批（SR品质22张）**：FM050~FM071（部分已有）
- SR卡需要较高精度，注意元素和种族特征

**第四批（SSR品质28张）**：FM072~FM099（部分已有）
- SSR卡需要高精度，注意品质光效

**第五批（UR+LE品质）**：FM100等
- UR/LE卡需要最高精度，特殊光效

### 角色数据来源
- `docs/策划案/融合姬图鉴.md` — 完整100张角色设定

- `assets/data/json/minionCards.json` — 100张卡牌数据（name, element, race, quality）

|------|---------|
| N | 简朴服装，无光效，灰暗色调 |
| R | 基础装备，微弱元素光效 |
| SR | 精致装备，明显元素光效，动态姿势 |
| SSR | 华丽装备，强烈元素光效，背景粒子特效 |
| UR | 传说级装备，全屏元素特效，独特背景 |
| LE | 超越传说，金色光效+专属背景+特殊形态 |

### 验收标准
