# 声誉系统策划案

## 1. 系统概述

### 1.1 系统定位
声誉系统是连接基地经营与禁区探索的桥梁，通过声誉等级限制高级内容的解锁，引导玩家逐步体验游戏内容。声誉越高，可招募的职业越强，可购买的能量饮料品质越高。

### 1.2 核心作用
- **解锁限制**：控制高级职业、能量饮料、设施的解锁
- **进度指引**：通过声誉等级让玩家了解游戏进度
- **成就感**：提升声誉带来荣誉感和成就感
- **经济平衡**：调节游戏内资源获取速度

## 2. 声誉等级设计

### 2.1 等级划分

| 等级 | 名称 | 所需声望值 | 解锁内容 |
|------|------|------------|----------|
| 1 | 新手拾荒者 | 0 | 基础设施、普通能量饮料、⭐级职业 |
| 2 | 见习驾驶员 | 100 | 活力苏打、中级设施升级 |
| 3 | 正式队员 | 300 | 强力能量、⭐⭐级职业 |
| 4 | 资深探险家 | 600 | 精准燃料、训练场解锁 |
| 5 | 队长 | 1000 | 超频能量、⭐⭐⭐级职业 |
| 6 | 英雄 | 1500 | 战意激化剂/守护合剂 |
| 7 | 传奇人物 | 2500 | 传说能量、⭐⭐⭐⭐级职业 |
| 8 | 废土领袖 | 4000 | 源初精华、所有设施满级 |
| 9 | 方舟使者 | 6000 | 禁忌药剂、隐藏角色概率+0.05% |
| 10 | 传奇永恒 | 10000 | 全部内容解锁、特殊称号 |

### 2.2 声望值获取方式

#### 2.2.1 主要获取途径

| 获取方式 | 基础值 | 上限 | 说明 |
|----------|--------|------|------|
| **通关楼层** | +10×楼层 | 无限 | 首次通关奖励 |
| **击败BOSS** | +50~500 | 无限 | 根据BOSS难度 |
| **完成每日任务** | +20~100 | +200/天 | 根据任务难度 |
| **完成成就** | +50~1000 | 无限 | 根据成就稀有度 |

#### 2.2.2 辅助获取途径

| 获取方式 | 基础值 | 上限 | 说明 |
|----------|--------|------|------|
| **每日登录** | +10 | +10/天 | 连续登录额外+5 |
| **招募新角色** | +5 | +50/天 | 每日首次招募额外+10 |
| **装备强化** | +2 | +20/天 | 每次强化成功获得 |
| **使用能量饮料** | +3 | +30/天 | 每日首次使用额外+5 |
| **分享战绩** | +15 | +15/天 | 社交功能（可选） |

### 2.3 声望值计算公式

```
当前总声望 = Σ(各途径获得的声望)
下一级所需 = 当前等级所需 - 已有声望
```

## 4. 声誉界面设计

### 4.1 声誉显示位置
- **基地界面**：右上角显示当前等级和名称
- **角色详情**：显示该角色的最低招募声誉要求
- **能量饮料商店**：未满足条件的饮料灰度显示并提示所需等级
- **帮助界面**：完整的声誉等级表和对应解锁内容

### 4.2 声誉等级图标设计
- 使用不同颜色区分等级段位：
  - 1-2级：铜色（#CD7F32）
  - 3-4级：银色（#C0C0C0）
  - 5-6级：金色（#FFD700）
  - 7-8级：铂金色（#E5E4E2）
  - 9-10级：钻石色（#B9F2FF）

## 6. 技术实现

### 6.1 系统架构
```javascript
class ReputationSystem {
    // 当前声望值
    currentReputation: number;
    
    // 当前等级
    currentLevel: number;
    
    // 各来源今日已获声望
    dailyGains: Map<string, number>;
    
    // 增加声望
    addReputation(source: string, amount: number): void;
    
    // 获取当前等级信息
    getCurrentLevel(): ReputationLevel;
    
    // 检查是否满足等级要求
    checkLevelRequirement(requiredLevel: boolean): boolean;
    
    // 重置每日上限

}
```

## 7. 平衡性考虑

### 7.1 声望获取节奏
- **第1周**：预计达到3-4级（正常游玩）
- **第2周**：预计达到5-6级（活跃玩家）
- **第1月**：预计达到7-8级（核心玩家）
- **长期**：9-10级需要持续投入

### 7.2 防刷机制
- 每日各类别有上限
- 重复行为递减收益（如连续招募同类型职业）
- 异常检测（短时间内大量获取声望）
