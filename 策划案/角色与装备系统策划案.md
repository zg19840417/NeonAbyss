# 角色与装备系统策划案

## 1. 系统策划

### 1.1 系统概述
角色与装备系统是游戏的核心养成系统，玩家通过招募不同职业的英雄，为他们装备武器、护甲和饰品，提升他们的属性和能力，组建强大的队伍。游戏中有12种职业，每种职业都有独特的主动技能和被动技能。

### 1.2 核心玩法
- **职业系统**：12种职业，每种职业有独特的主定位和辅助定位
- **技能系统**：每个职业有1个主动技能和1个被动技能
- **装备系统**：武器、护甲、饰品，提升角色属性
- **角色养成**：通过战斗获取经验值升级，提升属性
- **装备强化**：使用金币强化装备，提升属性

### 1.3 职业系统

#### 1.3.1 职业定位定义

| 定位 | 说明 |
|------|------|
| **坦克** | 提升自身生存能力（护盾、减伤），保护队友（伤害转移、减伤buff）。坦克职业攻击时会获得额外仇恨值。 |

| **治疗** | 治疗己方，清除debuff（治疗技能不能暴击） |

### 1.4 战斗属性系统

#### 1.4.1 面板展示属性

| 属性 | 说明 |
|------|------|
| **生命 (HP)** | 生命值，归零则死亡 |
| **攻击 (ATK)** | 攻击力，影响所有伤害 |
| **暴击率 (Crit%)** | 普通攻击暴击概率 |
| **闪避率 (Dodge%)** | 概率完全闪避攻击 |

#### 1.4.2 技能获取属性

| 属性 | 获取方式 |
|------|----------|
| **暴击伤害** | 被动技能提供（如：暴击时+50%伤害） |
| **减伤百分比** | 坦克主动技能提供（如：团队减伤15%） |
| **生命汲取** | 被动技能提供（如：攻击时+15%吸血） |
| **仇恨值加成** | 坦克定位提供（如：每伤害+3仇恨） |

#### 1.4.3 仇恨值系统

| 参数 | 说明 |
|------|------|
| **基础仇恨** | 每造成1点伤害，获得1点仇恨值 |
| **坦克主定位** | 主动技能额外获得**+3仇恨/damage** |
| **坦克辅助定位** | 被动技能额外获得**+3仇恨/damage** |
| **仇恨衰减** | 每回合结束时，仇恨值降低**10%** |
| **攻击目标计算** | 敌人选择攻击目标时，按**仇恨值占比**决定攻击概率 |

**仇恨值公式：**
```
目标被攻击概率 = 目标仇恨值 / 所有敌方目标仇恨值总和 × 100%
```

### 1.5 伤害公式

#### 1.5.1 普通攻击

```
伤害 = 攻击力 × 技能倍率 × (暴击? 暴击伤害 : 1) × (1 - 减伤%)
```

#### 1.5.2 治疗技能

```
治疗量 = 技能基础值  ← 不暴击，不受减伤影响
```

#### 1.5.3 伤害流程

1. 计算基础伤害（攻击力 × 技能倍率）
2. 判断是否暴击，计算暴击伤害
3. 应用减伤百分比
4. 计算实际伤害/治疗量

### 1.6 装备系统

#### 1.6.1 装备类型
- **武器（Weapon）**：增加攻击力
- **护甲（Armor）**：增加生命值、少量攻击
- **饰品（Accessory）**：均衡增加攻击和生命

#### 1.6.2 装备品质
- **普通（Common）**：基础属性
- **优秀（Uncommon）**：基础属性 × 1.15
- **稀有（Rare）**：基础属性 × 1.3
- **史诗（Epic）**：基础属性 × 1.5
- **传说（Legendary）**：基础属性 × 1.8

#### 1.6.3 装备强化
- **强化等级**：最高+15
- **强化成本**：基础成本 × (1.5)^(等级)
- **强化效果**：每级+8%属性

### 1.7 角色养成

#### 1.7.1 升级系统
- **经验值**：通过战斗获取
- **升级公式**：所需经验 = 50 × (1.15)^(等级-1)
- **属性成长**：每级+8%攻击、+5%生命

#### 1.7.2 转生系统
- **转生条件**：角色达到50级
- **转生效果**：
  - 等级重置为1
  - 保留100%属性加成
  - 获得额外**+10%全属性**

## 2. 数值策划

### 2.1 角色数值

#### 2.1.1 属性计算公式
- **生命值** = (基础生命 + 装备加成) × (1 + 等级 × 0.05)
- **攻击力** = (基础攻击 + 装备加成) × (1 + 等级 × 0.08)
- **暴击率** = 基础暴击率 + 装备加成（上限50%）
- **闪避率** = 基础闪避率 + 装备加成（上限40%）

## 3. 技术实现

### 3.1 角色系统架构
- **Character类**：基础角色属性和方法
- **Profession类**：职业配置（主/辅定位、技能）
- **Skill类**：主动/被动技能逻辑
- **HatredSystem类**：仇恨值计算和管理

### 3.2 数据配置
- **professions.xlsx**：职业配置（12职业、基础属性）
- **skills.xlsx**：技能配置（主动/被动效果）
- **equipment.xlsx**：装备配置（类型、品质、属性）
- **hatred.xlsx**：仇恨值配置（衰减率、加成系数）

### 3.3 存档系统
- **角色列表**：保存所有角色信息
- **装备列表**：保存所有装备信息
- **队伍配置**：保存当前队伍配置
