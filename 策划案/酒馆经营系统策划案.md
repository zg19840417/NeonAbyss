# 基地经营系统策划案

## 1. 系统策划

### 1.1 系统概述
基地经营是游戏的核心玩法之一，玩家通过升级基地设施、购买能量饮料来吸引不同职业的机甲驾驶员加入队伍，为禁区探索做准备。基地是雇佣兵的聚集地，玩家需要用心经营，才能吸引最强的驾驶员。

### 1.2 核心玩法
- **设施升级**：升级基地的充电桩、休眠舱、合成台等设施，提升基地功能和招募能力
- **能量饮料**：购买不同种类的能量饮料，吸引不同职业的驾驶员
- **驾驶员招募**：根据基地等级、能量饮料类型和声誉，招募不同品质的驾驶员
- **资源管理**：管理赛博币和能量晶，用于升级设施、购买能量饮料和招募驾驶员
- **伙伴刷新**：睡一觉（刷新时间）会有新伙伴出现

### 1.3 基地设施系统

#### 1.3.1 设施列表
- **充电桩（Chargers）**：
  - 功能：增加可同时招募的驾驶员数量
  - 每级效果：每级+1招募位
  - 基础升级成本：100赛博币
  - 成本倍数：1.5倍/级
  - 最大等级：10级

- **工作台（Workbench）**：
  - 功能：增加可同时招募的驾驶员数量
  - 每级效果：每级+1招募位
  - 基础升级成本：100赛博币
  - 成本倍数：1.5倍/级
  - 最大等级：10级

- **休眠舱（Dorms）**：
  - 功能：增加驾驶员的恢复速度，睡一觉刷新出新伙伴
  - 每级效果：恢复速度+10%
  - 基础升级成本：150赛博币
  - 成本倍数：1.5倍/级
  - 最大等级：10级

- **合成台（Synthesizer）**：
  - 功能：增加能量饮料的效果
  - 每级效果：能量饮料效果+5%
  - 基础升级成本：200赛博币
  - 成本倍数：1.5倍/级
  - 最大等级：10级

- **训练场（Training）**：
  - 功能：增加驾驶员的经验获取
  - 每级效果：经验获取+5%
  - 基础升级成本：250赛博币
  - 成本倍数：1.5倍/级
  - 最大等级：10级

- **黑客终端（Hacker）**：
  - 功能：解锁传送功能
  - 解锁效果：可在禁区中使用传送功能
  - 升级成本：500赛博币
  - 最大等级：1级（一次性解锁）

### 1.4 能量饮料系统

#### 1.4.1 饮料类型（完整10种）

**【普通级】基础饮料（游戏初期使用）**

- **普通能量（Common Energy）**：
  - 成本：50赛博币
  - 品质：⭐
  - 效果：吸引基础职业
  - 可招募职业：铁壁卫士、狂战士、元素法师
  - 使用建议：游戏初期使用

- **活力苏打（Vitality Soda）**：
  - 成本：60赛博币
  - 品质：⭐
  - 效果：吸引生存型职业
  - 可招募职业：生命守护者、自然治愈师
  - 特殊效果：招募角色生命+5%
  - 使用建议：需要坦克时使用

**【中级】进阶饮料（凑齐5人队后升级）**

- **强力能量（Strong Energy）**：
  - 成本：100赛博币
  - 品质：⭐⭐
  - 效果：吸引中级职业
  - 可招募职业：战术指挥官、暗影刺客、机械工程师
  - 使用建议：中期核心队伍组建

- **精准燃料（Precision Fuel）**：
  - 成本：120赛博币
  - 品质：⭐⭐
  - 效果：吸引敏捷型职业
  - 可招募职业：疾风剑客、时空行者
  - 特殊效果：招募角色闪避率+3%
  - 使用建议：需要高闪避角色时

**【高级】稀有饮料（优先招募治疗/关键角色）**

- **超频能量（Magic Energy）**：
  - 成本：150赛博币
  - 品质：⭐⭐⭐
  - 效果：吸引高级职业
  - 可招募职业：神圣牧师、元素领主
  - 使用建议：优先招募治疗，通关关键

- **战意激化剂（Battle Stimulant）**：
  - 成本：180赛博币
  - 品质：⭐⭐⭐
  - 效果：吸引输出型职业
  - 可招募职业：毁灭术士、机甲战神
  - 特殊效果：招募角色攻击力+8%
  - 使用建议：需要高DPS时使用

- **守护合剂（Guardian Elixir）**：
  - 成本：180赛博币
  - 品质：⭐⭐⭐
  - 效果：吸引坦克型职业
  - 可招募职业：钢铁壁垒、不屈意志
  - 特殊效果：招募角色减伤+5%
  - 使用建议：需要硬坦克时使用

**【传说级】顶级饮料（后期使用）**

- **传说能量（Legendary Energy）**：
  - 成本：250赛博币
  - 品质：⭐⭐⭐⭐
  - 效果：吸引传说职业
  - 可招募职业：铁血狂卫、幻影战将
  - 使用建议：后期追求最强角色

- **源初精华（Primal Essence）**：
  - 成本：300赛博币
  - 品质：⭐⭐⭐⭐
  - 效果：吸引全能型传说职业
  - 可招募职业：永恒守望者、命运编织者
  - 特殊效果：全属性+3%
  - 使用建议：终极目标，全属性加成

- **禁忌药剂（Forbidden Potion）**：
  - 成本：500赛博币
  - 品质：⭐⭐⭐⭐⭐
  - 效果：随机招募任意品质职业（有概率出隐藏角色）
  - 招募池：所有职业 + 隐藏角色（0.1%概率）
  - 特殊效果：可能获得独特外观或称号
  - 使用建议：赌徒选择，高风险高回报

#### 1.4.2 能量饮料品质与概率表

| 品质 | 出现概率 | 职业品质加成 |
|------|----------|--------------|
| ⭐ 普通 | 基础 | 无 |
| ⭐⭐ 中级 | +10%属性 | 小幅提升 |
| ⭐⭐⭐ 高级 | +20%属性 | 明显提升 |
| ⭐⭐⭐⭐ 传说 | +35%属性 | 显著提升 |
| ⭐⭐⭐⭐⭐ 禁忌 | +50%属性 | 极致提升 |

#### 1.4.3 能量饮料购买限制

| 限制条件 | 说明 |
|----------|------|
| 声誉等级 | 高级饮料需要对应声誉等级解锁 |
| 基地等级 | 部分饮料需要基地达到特定等级 |
| 每日上限 | 每类饮料每日最多购买10次 |
| 解锁顺序 | 必须先解锁低级才能购买高级 |

### 1.5 驾驶员招募系统

#### 1.5.1 招募条件
- **能量饮料**：必须购买对应类型的能量饮料
- **声誉等级**：必须达到要求的声誉等级
- **基地等级**：部分高级驾驶员需要特定的基地等级
- **赛博币成本**：不同职业和品质的驾驶员有不同的招募成本

#### 1.5.2 声誉系统
- **声誉获取**：
  - 通关更高楼层
  - 击败巨型机甲BOSS
  - 完成成就
  - 每日任务奖励
- **声誉等级**：
  - 初级声誉：可招募基础和中级职业
  - 中级声誉：可招募高级职业
  - 高级声誉：可招募传说职业
- **查看方式**：在基地界面点击右上角的"?"可查看所有伙伴的招募条件

### 1.6 界面设计

#### 1.6.1 基地主界面
- **布局**：竖屏布局，模拟基地场景
- **顶部区域**：
  - 赛博币数量显示
  - 能量晶数量显示
  - 基地等级显示
- **中间区域**：
  - 基地设施展示区（可点击升级）
  - 可招募驾驶员展示区（显示在基地内的驾驶员）
  - 能量饮料展示区（可点击购买）
- **底部区域**：
  - 禁区入口按钮
  - 装备商店/黑市按钮
  - 睡一觉按钮（刷新伙伴）
- **右上角区域**：
  - 帮助按钮（?）：查看所有伙伴的招募条件
  - 设置按钮
  - 成就按钮
  - 任务按钮

#### 1.6.2 设施升级界面
- **触发**：点击基地设施
- **显示内容**：
  - 设施名称和图标
  - 当前等级
  - 升级效果说明
  - 升级成本
  - 升级按钮
- **交互**：
  - 点击升级：消耗赛博币，提升设施等级
  - 点击关闭：关闭升级界面

#### 1.6.3 能量饮料购买界面
- **触发**：点击能量饮料
- **显示内容**：
  - 饮料名称和图标
  - 效果说明
  - 购买成本
  - 可招募的职业预览
  - 购买按钮
- **交互**：
  - 点击购买：消耗赛博币，购买能量饮料
  - 点击关闭：关闭购买界面

#### 1.6.4 驾驶员招募界面
- **触发**：点击可招募驾驶员
- **显示内容**：
  - 驾驶员头像和名称
  - 职业标识
  - 属性面板（生命值、攻击力、暴击率、闪避率）
  - 技能列表
  - 招募条件（能量饮料类型、声誉等级）
  - 招募成本
  - 招募按钮
- **交互**：
  - 点击招募：消耗赛博币，招募驾驶员加入队伍
  - 点击关闭：关闭招募界面

#### 1.6.5 帮助界面（招募条件表）
- **触发**：点击右上角"?"按钮
- **显示内容**：
  - 所有职业的招募条件列表
  - 当前已满足条件的职业高亮显示
  - 当前未满足条件的职业灰度显示
  - 显示所需能量饮料、声誉等级等信息
- **交互**：
  - 滚动查看所有职业
  - 点击关闭：关闭帮助界面

## 2. 数值策划

### 2.1 设施升级成本
- **升级公式**：升级成本 = 基础成本 × (1.5)^(等级-1)
- **充电桩（Chargers）**：
  - 1级：100赛博币
  - 2级：150赛博币
  - 3级：225赛博币
  - 以此类推...
- **工作台（Workbench）**：同上
- **休眠舱（Dorms）**：
  - 基础成本：150赛博币
  - 1级：150赛博币
  - 2级：225赛博币
- **合成台（Synthesizer）**：
  - 基础成本：200赛博币
- **训练场（Training）**：
  - 基础成本：250赛博币
- **黑客终端（Hacker）**：
  - 一次性成本：500赛博币

### 2.2 招募位计算
- **基础招募位**：2个
- **充电桩效果**：每级+1
- **工作台效果**：每级+1
- **总招募位** = 基础 + 充电桩等级 + 工作台等级

### 2.3 能量饮料效果
- **基础效果**：吸引对应职业类型
- **合成台加成**：每级+5%效果
- **效果计算**：基础效果 × (1 + 合成台等级 × 0.05)

## 3. 技术实现

### 3.1 系统架构
- **BaseSystem**：基地系统核心类
- **FacilityManager**：设施管理器
- **EnergyDrinkManager**：能量饮料管理器
- **RecruitmentManager**：招募管理器
- **ReputationSystem**：声誉系统

### 3.2 数据配置
- **baseFacilities.xlsx**：基地设施配置表
- **energies.xlsx**：能量饮料配置表
- **reputation.xlsx**：声誉等级配置表

### 3.3 存档系统
- **设施等级**：保存所有设施的当前等级
- **能量饮料库存**：保存已购买的能量饮料
- **声誉等级**：保存当前声誉等级
- **刷新时间**：记录上次刷新伙伴的时间
