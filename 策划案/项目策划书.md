# 苏打地牢 - 项目策划书

> 本文档为设计师备忘录，整合游戏基础设定、世界观、系统机制、玩法设计等核心内容。

---

## 一、项目概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 苏打地牢（Soda Dungeon）/ 机甲禁区 |
| 游戏类型 | 地牢探险 + 自动战斗 + 酒馆经营 |
| 目标平台 | 移动端（手机优先） |
| 游戏引擎 | Phaser 3 |
| 设计分辨率 | 375×812（移动端逻辑像素，iPhone X标准） |
| 运营模式 | 放置挂机 + 策略战斗 |

### 1.2 核心卖点

1. **放置挂机**：自动战斗，离线也能获取收益
2. **策略搭配**：16种职业，自由组建五人小队
3. **酒馆经营**：收集能量饮料，招募机甲驾驶员
4. **深度养成**：装备强化、套装搭配、技能组合

## 二、世界观设定

### 2.1 背景故事

**核战百年后**

一场毁灭性的核战争彻底改变了地球，曾经繁华的城市化为废墟。辐射无处不在，幸存的人类在废墟中艰难求生。

**融合技术**

人类掌握了禁忌的「融合技术」，通过神圣仪式与植物或动物融合，获得超凡能力。这项技术成为人类对抗末世的关键。

**避难方舟**

传说中由最强融合者建立的圣地，人类最后的希望之地。玩家被方舟接纳，成为「拾荒者队长」。

### 2.2 融合形态

| 融合类型 | 特征 | 能力倾向 |
|----------|------|----------|
| 植物融合 | 拥有藤蔓纹路、叶片装饰 | 再生、防护、治疗 |
| 动物融合 | 兽耳、爪痕、鳞片 | 速度、力量、敏捷 |
| 混合融合 | 两者兼具，形态多变 | 综合能力 |

### 2.3 势力划分

**融合者（清醒者）**
- 成功完成融合仪式的幸存者
- 保留完整人性和记忆
- 拥有神圣的融合纹路、发光粒子
- 画风：神圣、柔和、整体保持人形轮廓

**失心者（敌人）**
- 融合失败的堕落者
- 曾是平民、战士甚至英雄
- 狂暴混乱，攻击一切生命
- 画风：扭曲但不过于恐怖，保持可辨识度

### 2.4 玩家角色

- **身份**：唯一的「清醒者」——从失心状态中奇迹般恢复的人类
- **背景**：曾是失心者的你，在辐射风暴中堕落并迷失，最终奇迹般找回理智
- **使命**：被避难方舟接纳，成为拾荒者队长，带领队伍探索废土

## 四、核心系统

### 4.1 禁区探索系统

**玩法描述**：
玩家组建五人小队，在禁区中逐层推进，采用回合制战斗。

**核心机制**：
- 禁区关卡随机生成
- 每层包含多个房间和战斗
- 每10层出现强大BOSS
- 支持手动操作和自动战斗

**房间类型**：
| 房间类型 | 描述 | 触发条件 |
|----------|------|----------|
| 普通战斗 | 击败普通敌人 | 随机 |
| 精英战斗 | 击败精英敌人 | 随机 |
| BOSS战斗 | 击败巨型机甲 | 每10层 |
| 宝箱房 | 领取奖励 | 随机 |
| 事件房 | 触发随机事件 | 随机 |
| 商店 | 购买道具装备 | 随机 |

### 4.2 自动战斗系统

**战斗流程**：
1. 己方角色按速度排序
2. 每角色选择技能/普攻
3. 执行行动，播放动画
4. 结算伤害和效果
5. 敌方行动
6. 回合结束，检测胜负

**AI优先级**：
```
治疗 > 坦克技能 > 辅助buff > 辅助debuff > 群体攻击 > 单体攻击 > 普通攻击
```

**仇恨值机制**：
- 基础仇恨：每造成1点伤害获得1点仇恨
- 坦克定位：主动技能额外+3仇恨/damage
- 敌人根据仇恨占比选择攻击目标

### 4.3 酒馆经营系统

**设施列表**：
| 设施 | 功能 | 每级效果 |
|------|------|----------|
| 充电桩 | 增加可招募驾驶员数量 | +1 |
| 工作台 | 增加可同时招募数量 | +1 |
| 休眠舱 | 加速驾驶员刷新 | 更快刷新 |
| 合成台 | 增加能量饮料效果 | +5% |
| 训练场 | 增加经验获取 | +5% |
| 黑客终端 | 解锁传送功能 | 可跳转关卡 |

**能量饮料系统**：
| 品质 | 价格 | 效果 |
|------|------|------|
| ⭐ 普通 | 50-60币 | 基础属性 |
| ⭐⭐ 中级 | 100-120币 | 强化属性 |
| ⭐⭐⭐ 高级 | 150-180币 | 特殊效果 |
| ⭐⭐⭐⭐ 传说 | 250-300币 | 稀有角色 |
| ⭐⭐⭐⭐⭐ 禁忌 | 500币 | 隐藏角色概率 |

### 4.4 角色与装备系统

**职业体系**（16种）：

| 定位 | 主职业 | 辅助组合 | 数量 |
|------|--------|----------|------|
| 坦克 | 守护者 | 重装/守护/战术 | 4种 |
| 输出 | 毁灭者 | 狂暴/精密/战术 | 6种 |
| 辅助 | 暗影 | 诅咒/战术/守护 | 3种 |
| 治疗 | 圣职者 | 净化/守护/战术 | 3种 |

**装备系统**：
- 武器、护甲、饰品各9件
- 按品质分：普通→优秀→精良→史诗→传说
- 强化等级：+0到+10
- 套装效果：5套组合

**属性面板**（仅显示4项）：
- 生命值
- 攻击力
- 暴击率
- 闪避率

### 4.5 成就与任务系统

**成就分类**：
- 战斗成就：击杀敌人、通关层数
- 收集成就：获得装备、招募角色
- 经营成就：升级设施、消费货币
- 隐藏成就：特殊条件触发

**每日任务**：
- 固定任务：每日必做（6个）
- 随机任务：每日刷新（2个）
- 每周任务：周期性挑战（3个）

### 4.6 声誉系统

**声誉等级**：1-10级

**声誉获取**：
- 通关禁区
- 完成成就
- 日常任务

**声誉奖励**：
- 解锁新角色
- 解锁新设施
- 解锁新禁区

## 五、数值设计

### 5.1 伤害公式

```
伤害 = 攻击力 × 技能倍率 × (暴击 ? 暴击伤害 : 1) × (1 - 减伤%)
```

### 5.2 仇恨值公式

```
目标被攻击概率 = 目标仇恨值 ÷ 所有敌方目标仇恨值总和 × 100%
```

### 5.3 治疗公式

```
治疗量 = 技能基础值（不暴击，不受减伤影响）
```
