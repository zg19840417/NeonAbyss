# 美术资源设计规范

## 设计总则

### 世界观背景
- **游戏名**：废土元年（Wasteland Year One）
- **背景**：核战争摧毁了人类文明，核辐射导致动植物和机械发生变异
- **融合**：残存人类与变异生物/植物/机械进行"融合"
  - 融合成功 → "融合姬"（保留人性，获得超凡力量，全部为女性）
  - 融合失败 → "失心者"（失去理性，变成怪物）
- **庇护所**：少数幸存者的避难所（实际是星舰指挥舱），玩家是基地管理者（继承者）
- **探索**：地面层（野外）为低级区域，地下层（禁区）为高级区域
- **最终Boss**：终焉之核·奥米茄 — 融合技术的创造者，完美融合体，保持理性+强大力量
- **文案风格**：统一为废土科幻+生物融合风格，禁止奇幻/神话词汇

### 视觉风格定位
- **主题**：废土 + 生物融合（Wasteland Fusion）
- **核心色调**：待定（需根据世界观重新设计）
- **风格方向**：废土末日感 + 融合者的希望感，体现"从废墟中重生"的主题

### 角色设计原则
1. **融合姬特征（我方）**：
   - 全部为女性
   - 保留人形轮廓
   - 融合对象的特征自然融入人体（植物纹路/机械部件/能量光环等）
   - 融合纹路带有微弱发光效果
   - 无血腥、无恐怖元素
   
2. **变异生物（低级敌人）**：
   - 核辐射导致的动植物变异体
   - 外观扭曲但保持生物特征可辨识
   - 色彩偏暗淡/病态
   
3. **失心者（精英/Boss敌人）**：
   - 融合失败体，拥有强大力量但失去理性
   - 扭曲但可辨识的人形或半人形
   - 暗淡/浑浊的色彩，不稳定的能量波动
   - 保持一定的美感（不过度恐怖）

## 一、角色立绘设计（12种职业）

## 三、BOSS设计（10个禁区Boss — 均为失心者）

### 3.1 锈蚀守卫·陈岚（废弃矿坑 Boss）
- **名称**：锈蚀守卫·陈岚（Rust Guardian Chen Lan）
- **背景**：庇护所前哨守卫，被迫融合，守在庇护所入口附近无意识攻击接近者
- **体型**：中等
- **外观**：
  - 身体被锈蚀的金属装甲覆盖
  - 融合了机械变异体的特征
  - 眼睛发出不稳定的红色光芒
  - 手持残破的巡逻盾牌
- **配色**：锈红+暗金属+红色光芒
- **特殊效果**：
  - 攻击时有金属摩擦的火花
  - 血量低时装甲碎裂露出扭曲的人体
- **尺寸**：160×240px

### 3.2 暴走兽王·林瑶（地下水脉 Boss）

- **配色**：暗棕+血红+兽纹
- **特殊效果**：

## 六、技术规格

### 6.1 图片格式
- **格式**：PNG（支持透明）
- **色彩模式**：RGBA（32位）
- **分辨率**：基于375×812逻辑像素
- **适配**：@2x, @3x 高清版本

- **角色立绘**：140×240px
- **敌人小型**：100×160px
- **敌人中型**：120×180px
- **敌人大型**：140×200px
- **BOSS**：200×280px 起
- **图标**：24×24px, 32×32px, 48×48px
- **UI元素**：按实际需求

### 6.3 动画帧数
- **待机动画**：4-6帧循环
- **攻击动画**：8-12帧
- **受击动画**：4-6帧
- **死亡动画**：8-15帧
- **技能动画**：15-30帧

## 七、命名规范

### 7.1 文件名格式
```
[类型]_[名称]_[动作]_[帧号].png
```

**示例**：
- `char_tank_guardian_idle_01.png`
- `enemy_mech_hound_attack_03.png`
- `boss_destroyer_lord_skill_05.png`
- `ui_btn_normal.png`
- `ui_hp_bar_green.png`
- `bg_dungeon_ruins.png`

```
assets/images/
├── characters/
│   ├── tank/          # 坦克类
│   ├── dps/           # 输出类
│   ├── support/       # 辅助类

├── enemies/
│   ├── normal/        # 普通敌人

├── ui/
│   ├── buttons/       # 按钮
│   ├── icons/         # 图标
│   ├── panels/        # 面板
│   └── effects/       # 特效
└── backgrounds/
    ├── dungeon/       # 禁区背景
    ├── base/          # 基地背景
    └── battle/        # 战斗背景
```
